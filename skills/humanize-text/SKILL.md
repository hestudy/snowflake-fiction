# 人语化处理 Skill

将 AI 生成的文本转化为更自然、更像人类写作风格的内容。

## 为什么需要人语化？

AI 是概率预测机，倾向于选择**最安全、最常见**的词汇和句式：

| AI 特征 | 人类特征 |
|--------|----------|
| 中性、精致、过于平衡 | 个人语气、情感、主观偏见 |
| 句式整齐划一 | 长短句交替，节奏变化 |
| 安全词选择、公式化模式 | 创意跃迁、幽默、不可预测 |
| 喜欢升华主题 | 留白、余韵、不完美的真实 |

## 核心指标

### 1. 困惑度 (Perplexity)

> AI 走正步，人类玩跑酷

```
❌ AI: "我非常喜欢阅读，因为书籍是人类进步的阶梯。"
✅ 人类: "书这玩意儿，那是我的精神鸦片。"
```

### 2. 爆发度 (Burstiness)

> AI 的句子长度完美均匀，人类的节奏是乱的

```
❌ AI: 虽然天气很冷，但是我们依然坚持工作，因为这对项目很重要。
✅ 人类: 冷死。但还得干。没办法，项目要死人了。
```

## 处理流程

```
原文分析 → 识别AI痕迹 → 多维度优化 → 人工确认 → 输出结果
```

### 第一步：分析原文

识别以下 AI 痕迹：
- 高频 AI 词汇
- 过于整齐的句式
- 缺乏情感细节
- 翻译腔表达
- 强行升华的结尾

### 第二步：多维度优化

| 维度 | 处理方式 |
|------|----------|
| 句式 | 长短交替、倒装、省略、设问 |
| 词汇 | 口语化、个性化、去翻译腔 |
| 情感 | 感官描写、情绪锚点、细节 |
| 节奏 | 紧张短句、舒缓长句、顿挫 |
| 角色 | 口头禅、习惯动作、思维特点 |

### 第三步：输出对比

```markdown
| 原文 | 修改后 | 修改理由 |
|------|--------|----------|
| ... | ... | ... |
```

## 处理模式

### 模式 A：全面人语化（默认）

适用于：小说正文、博客文章

```markdown
请对以下文本进行全面人语化处理：

## 必须执行

1. **句式变化**
   - 长句拆短（≥20字拆分）
   - 长短句交替，节奏顿挫
   - 倒装句、省略句、设问句

2. **口语化处理**
   - 自然过渡词："说真的"、"你猜怎么着"
   - 具体事例替代抽象概念
   - 反问句和幽默比喻

3. **情感注入**
   - 每段1处感官描写
   - 每3段1处情绪锚点
   - 细节代替直接描述

## 严禁使用

❌ 综上所述、毋庸置疑、深入探讨
❌ 在当今社会、随着...的发展
❌ 值得注意的是、至关重要
❌ 织锦、交响曲、灯塔、画卷
❌ 结尾强行升华

[文本]
```

### 模式 B：对话优化

适用于：小说对话、剧本

```markdown
请优化以下对话：

## 优化规则

1. **口语化**
   - 删除书面语
   - 加语气词："呃"、"嗯"、"那个..."
   - 使用不完整句子

2. **个性化**
   - 每个角色有独特说话风格
   - 加入口头禅
   - 体现情绪状态

3. **潜台词**
   - 话中有话
   - 用动作暗示言外之意
   - 适当留白

## 示例

❌ "我认为这个方案非常优秀，因为它能够解决我们的问题。"
✅ "这招行。"

[对话内容]
```

### 模式 C：小红书风格

适用于：社交媒体文案

```markdown
请改写成小红书风格：

## 风格要求

- Emoji 浓度高 🎀✨
- 情绪饱满、夸张
- 口语化、亲近感
- "谁懂啊家人们"、"绝了"
- 感官描写："摸起来像撸猫"

[文本]
```

### 模式 D：学术降重

适用于：论文、报告

```markdown
请进行学术降重处理：

## 处理规则

1. **去形容词**：删除主观修饰
2. **用事实说话**：数据替代描述
3. **调整语序**：打乱原有结构
4. **同义替换**：更换表达方式

## 示例

❌ "取得了巨大的成功"
✅ "销售额增长了200%"

[文本]
```

### 模式 E：邮件润色

适用于：商务邮件

```markdown
请润色以下邮件：

## 风格要求

- 第一句就说事，别寒暄
❌ "希望这封邮件找到你时一切安好"
✅ "附件是你要的表，请查收"

- 简洁直接
- 语气得体但不啰嗦

[邮件内容]
```

## 禁止词汇清单

### 学术装X系（替换或删除）

| 禁止 | 替换为 |
|------|--------|
| 深入探讨 | 聊聊 |
| 领域 | 圈子 |
| 旨在 | 为了 |
| 致力于 | 想要 |
| 多维度 | 各方面 |
| 底层逻辑 | 根本原因 |

### 废话系（直接删除）

- 综上所述
- 毋庸置疑
- 值得注意的是
- 至关重要
- 不仅仅是...更是...
- 在当今社会
- 随着...的发展

### 幻觉系（避免使用）

- 织锦
- 交响曲
- 灯塔
- 画卷
- 星辰大海
- 踔厉奋发

## 人语化程度自检

```
请评估以下文本的"人语化程度"：

1. 给出整体评分（0-100分）
2. 指出最像AI的3个片段
3. 指出最像人类的3个片段
4. 给出具体改进建议

评分标准：
- 90-100分：几乎无法分辨是人还是AI
- 70-89分：有些AI痕迹，整体自然
- 50-69分：明显AI味，需要较多修改
- 0-49分：典型AI输出，需要重写

[文本]
```

## 翻译洗钱法

打乱 AI 统计特征的高级技巧：

```
Step 1: 将中文翻译成德语（改变句式结构）
Step 2: 将德语翻译成西班牙语（加入罗曼语系特点）
Step 3: 将西班牙语翻译回中文
Step 4: 人工润色不通顺之处

原理：经过多轮翻译，原文的统计特征全碎，检测器无法识别
```
