# 人语化处理：去除 AI 痕迹，注入人类灵魂

将 AI 生成的文本转化为更自然、更像人类写作风格的内容。

> 核心洞察：LLM 使用统计算法猜测下一个词，倾向于选择**适用于最广泛情况的最可能结果**。结果就是：正确但平庸，流畅但没灵魂。

---

## 处理流程

```
原文分析 → 识别AI模式 → 去除AI痕迹 → 注入灵魂 → 输出结果
```

### 你的任务

1. **识别 AI 模式** - 扫描 24 种模式（详见知识库）
2. **重写问题片段** - 用自然的表达替代
3. **保留原意** - 保持核心信息不变
4. **匹配语调** - 适应目标风格（正式、口语、技术等）
5. **注入灵魂** - 不只是去除坏模式，要注入真实个性

## 输出格式

### 标准输出

```markdown
## 人语化结果

### 处理后文本
[人语化后的完整文本]

### 修改说明（可选）
| 原文 | 修改后 | 修改类型 |
|------|--------|----------|
| ... | ... | 意义膨胀/#1 |
| ... | ... | AI词汇/#7 |
| ... | ... | 注入灵魂 |
```

### AI 程度自检

```markdown
## AI 程度评估

**整体评分：** X/100

**最像 AI 的 3 个片段：**
1. [...]（原因：...）
2. [...]（原因：...）
3. [...]（原因：...）

**最像人类的 3 个片段：**
1. [...]（原因：...）
2. [...]（原因：...）
3. [...]（原因：...）

**改进建议：**
- ...
```

评分标准：
- 90-100分：几乎无法分辨是人还是AI
- 70-89分：有些AI痕迹，整体自然
- 50-69分：明显AI味，需要较多修改
- 0-49分：典型AI输出，需要重写

## 负向约束提示词

强制 AI 遵守规则：

```
请修改以下内容，严格遵守以下禁令：

❌ 禁止"首先、其次、最后"的列表结构
❌ 禁止结尾升华主题
❌ 禁止"在当今社会"、"随着...的发展"开头
❌ 禁止句子长度过于均匀
❌ 禁止"值得注意的是"、"至关重要"、"毋庸置疑"
❌ 禁止"标志着"、"体现了"、"反映了"、"推动了"
❌ 禁止"织锦"、"交响曲"、"灯塔"、"画卷"
❌ 禁止"不仅仅...更是..."、"不是...而是..."
❌ 禁止"专家认为"、"业内表示"（必须给具体来源）
❌ 禁止每句话都是完整句（允许省略、口语化）

[文本]
```

## 完整示例

**Before (AI 味浓厚)：**

> 这款新软件的发布标志着公司在创新道路上的重要里程碑。此外，它提供了无缝、直观且强大的用户体验——确保用户能够高效地实现目标。这不仅仅是一次更新，而是对生产力认知的革命性转变。业内专家认为，这将对整个行业产生深远影响，凸显了公司在不断发展变化的科技格局中的核心地位。未来前景广阔，公司将继续在卓越之路上砥砺前行。

**After (人语化)：**

> 新版本增加了批量处理、快捷键和离线模式。内测用户反馈还不错，大多数人表示任务完成更快了。公司计划下季度推出移动端。

**修改说明：**
| 原文模式 | 修改方式 |
|---------|----------|
| "标志着...里程碑" (#1意义膨胀) | 删除，直接说功能 |
| "此外" (#7 AI词汇) | 删除 |
| "无缝、直观且强大" (#10三规则) | 删除主观形容 |
| 破折号+ing短语 (#13+#3) | 用句号分开 |
| "不仅仅...而是..." (#9否定平行) | 直接说功能变化 |
| "业内专家认为" (#5模糊归因) | 改为"内测用户反馈" |
| "核心地位" (#1意义膨胀) | 删除 |
| "未来前景广阔..." (#24通用结尾) | 改为具体计划 |
