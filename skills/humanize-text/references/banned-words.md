# 禁止词汇清单

## 意义膨胀系

| 禁止 | 可替换为 |
|------|----------|
| 标志着、见证了 | 时间+发生了什么 |
| 具有里程碑意义 | 是个大事 |
| 具有深远影响 | 影响... |
| 体现了、反映了、展示了 | 直接描述 |
| 推动了、促进了 | 帮助、让 |
| 塑造了、构建了 | 建立、形成 |
| 在...进程中 | 在...期间 |

## 学术装X系

| 禁止 | 可替换为 |
|------|----------|
| 深入探讨 | 聊聊、说说 |
| 领域 | 圈子、行业 |
| 旨在 | 为了 |
| 致力于 | 想要、在努力 |
| 多维度 | 各方面 |
| 底层逻辑 | 根本原因 |
| 赋能 | 帮助、支持 |
| 闭环 | 完整 |

## 废话系（直接删除）

- 综上所述
- 毋庸置疑
- 值得注意的是
- 至关重要
- 不仅仅是...更是...
- 在当今社会
- 随着...的发展
- 不难发现
- 由此可见
- 众所周知

## 模糊归因系（给出具体来源）

- 专家认为 → 谁？什么时候？在哪说的？
- 业内人士表示 → 具体是谁？
- 研究显示 → 哪个研究？谁做的？
- 普遍认为 → 谁这么认为？

## 幻觉系（避免使用）

- 织锦
- 交响曲
- 灯塔
- 画卷
- 星辰大海
- 踔厉奋发
- 笃行不怠
- 砥砺前行
