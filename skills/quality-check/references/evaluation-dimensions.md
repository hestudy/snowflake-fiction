# 五维度评估详解

## 维度一：冲突强度（25分）

### 评分标准

| 分数 | 描述 |
|------|------|
| 20-25 | 核心冲突明确且强烈，多层冲突交织，角色面临真实困境 |
| 15-19 | 有明确冲突但强度一般，或冲突层次单一 |
| 10-14 | 冲突模糊或流于表面，缺乏真实威胁感 |
| 5-9 | 冲突极弱，角色几乎没有阻碍 |
| 0-4 | 无冲突，纯叙述或流水账 |

### 检查清单

- 是否存在明确的核心冲突（人vs人/人vs环境/人vs自我）？
- 冲突是否有真实的后果和代价？
- 角色是否面临两难选择？
- 冲突是否随情节推进而升级？
- 是否有多层冲突（外部+内部）交织？

### 选择题 vs 判断题检查

冲突设计要避免让读者产生分歧（"选择题"），应设计成读者共识一致的"判断对错题"：

| 类型 | 特征 | 读者反应 | 弃书风险 |
|------|------|----------|----------|
| 判断题 | 反派罪无可赦，主角行为正义 | 读者一致认同 | 低 |
| 选择题 | 反派有苦衷，主角面临道德两难 | 读者分裂站队 | 高 |

**检查要点**：
- 主角的关键决定是否会引发读者分歧？
- 反派是否有可能让部分读者同情？
- 感情线是否存在"站队"风险？

### 常见问题

| 问题 | 表现 | 修复方向 |
|------|------|----------|
| 无冲突 | 角色顺利完成所有事情 | 设置障碍、对手、内心矛盾 |
| 假冲突 | 冲突轻易化解，无真实威胁 | 提高代价，让失败成为可能 |
| 冲突断裂 | 冲突提出后被遗忘 | 持续追踪冲突线，确保回收 |
| 选择题冲突 | 主角面临道德两难，读者分裂 | 简化为判断对错，删除反派同情点 |

## 维度二：情绪密度（25分）

### 评分标准

| 分数 | 描述 |
|------|------|
| 20-25 | 情绪饱满且层次丰富，读者能强烈共情，情绪转折自然 |
| 15-19 | 有情绪表达但深度不够，或情绪变化缺乏铺垫 |
| 10-14 | 情绪平淡，角色像在陈述事实而非经历事件 |
| 5-9 | 几乎无情绪波动，角色如木偶 |
| 0-4 | 完全无情绪，纯信息传递 |

### 检查清单

- 角色是否有明确的情绪反应？
- 情绪是否通过行为/对话/内心独白展现（而非直接告知）？
- 情绪变化是否有合理的触发点？
- 是否有复杂情绪（矛盾、纠结、五味杂陈）？
- 读者能否在关键时刻产生共情？
- 情绪是否在角色之间传递（A的情绪影响B，B的反应又影响C）？
- 情绪是否有递进（每次冲击比上一次更强烈）？

### 常见问题

| 问题 | 表现 | 修复方向 |
|------|------|----------|
| 情绪告知 | "他很伤心"而非展示伤心 | 用行为、对话、身体反应展示 |
| 情绪单一 | 全程只有一种情绪 | 加入情绪层次和转折 |
| 情绪断层 | 重大事件后角色无反应 | 补充角色的情绪处理过程 |
| 情绪孤立 | 角色的情绪不影响其他人 | 让情绪在角色间传递，形成连锁反应 |
| 情绪跳级 | 小剧情后突然大爆发，再回到小剧情 | 情绪递进要匹配事件升级，不能跳级 |

## 维度三：期待感（20分）

### 评分标准

| 分数 | 描述 |
|------|------|
| 16-20 | 读者强烈想知道后续发展，每段都有推进动力 |
| 12-15 | 有一定吸引力但不够紧迫 |
| 8-11 | 期待感薄弱，读者可能中途放弃 |
| 4-7 | 几乎无期待感，情节可预测 |
| 0-3 | 完全无悬念，读者无继续阅读的动力 |

### 检查清单

- 是否有未解答的问题驱动读者继续？
- 是否有信息不对称（读者知道角色不知道的，或反之）？
- 是否有即将到来的威胁或机会？
- 情节走向是否有不可预测性？
- 是否在关键处留下悬念？

### "知而未见"公式

期待感的本质是"知而未见"——读者预见了即将发生的事，想看它落地，但作者没让它落地，中间的线索却不断指向这件事。

**检查要点**：
- 是否有读者能预见但尚未发生的事件？
- 中间是否有线索不断指向这件事？
- "事"的本质是否是人物碰撞和角色变化（而非单纯事件）？

### 常见问题

| 问题 | 表现 | 修复方向 |
|------|------|----------|
| 无悬念 | 所有信息一次性交代完 | 分步揭示，保留关键信息 |
| 可预测 | 读者能猜到所有发展 | 加入意外转折或反转 |
| 无紧迫感 | 没有时间压力或后果 | 设置deadline或不可逆后果 |
| 期待断档 | 一个期待满足后无新期待续上 | 用锁链模式提前铺垫下一个期待 |

## 维度四：节奏控制（15分）

### 评分标准

| 分数 | 描述 |
|------|------|
| 12-15 | 张弛有度，紧张与舒缓交替，节奏服务于情节 |
| 9-11 | 节奏基本合理但有局部拖沓或过快 |
| 6-8 | 节奏问题明显，大段拖沓或全程紧绷 |
| 3-5 | 节奏严重失控，影响阅读体验 |
| 0-2 | 无节奏意识，平铺直叙 |

### 检查清单

- 紧张场景和舒缓场景是否交替出现？
- 重要场景是否给了足够篇幅？
- 过渡段落是否简洁不拖沓？
- 对话与叙述的比例是否合理？
- 是否有不必要的重复描写？

### 爽点节奏完整性检查

副本/剧情段的节奏应包含完整的三段式：

| 阶段 | 检查项 | 缺失后果 |
|------|--------|----------|
| 爽点预热（起因） | 是否有铺垫和期待拉升？ | 高潮来得突兀，读者无感 |
| 爽点爆发（经过） | 高潮是否集中爆发、不过度冗长？ | 紧张节奏拉长导致阅读疲惫 |
| 爽点回味（结果） | 是否有收获感和舒缓？ | 高潮后直接进入下一段，读者喘不过气 |

### 常见问题

| 问题 | 表现 | 修复方向 |
|------|------|----------|
| 全程平淡 | 无高低起伏，像流水账 | 制造节奏波动，高潮前铺垫 |
| 过度拖沓 | 大段描写无推进 | 删减冗余，保留有效信息 |
| 节奏过快 | 重要场景一笔带过 | 放慢关键场景，增加细节 |
| 高潮过长 | 紧张节奏持续太久 | 集中爆发，缩短高潮篇幅 |
| 无回味期 | 高潮后直接进入下一事件 | 补充收获/成长/日常舒缓段落 |

## 维度五：钩子设计（15分）

### 评分标准

| 分数 | 描述 |
|------|------|
| 12-15 | 开头立刻抓住注意力，结尾留下强烈悬念 |
| 9-11 | 开头或结尾有一个做得好 |
| 6-8 | 开头和结尾都平淡 |
| 3-5 | 开头无吸引力，结尾无悬念 |
| 0-2 | 完全无钩子意识 |

### 检查清单

- 开头第一句/第一段是否能抓住读者？
- 是否在前300字内建立了核心吸引力？
- 章节结尾是否有悬念或转折？
- 是否有"翻页动力"（让读者想看下一章）？
- 钩子是否与主线相关（而非噱头）？

### 常见问题

| 问题 | 表现 | 修复方向 |
|------|------|----------|
| 平淡开头 | 以背景描写或日常开场 | 从冲突/悬念/行动切入 |
| 无悬念结尾 | 章节在平静中结束 | 在转折点或悬念处断章 |
| 虚假钩子 | 钩子与后续无关 | 确保钩子承诺能兑现 |

### 钩子密度量化标准（崩文预防）

**核心指标**：每章至少保持 1.75 个活跃钩子

| 钩子密度 | 评分 | 读者反应 |
|---------|------|---------|
| ≥2个 | 12-15分 | 强烈追更欲望 |
| 1.5-2个 | 9-11分 | 基本维持期待 |
| 1-1.5个 | 6-8分 | 期待感薄弱 |
| <1个 | 0-5分 | 弃书风险高 |

**钩子计算公式**：
```
总钩子数 = 章节结尾悬念(1) + 中段转折(0.5×N) + 未解决问题线(0.25×N)
```

**悬念交替原则**：在第1个悬念揭晓之前就铺垫第2个悬念，不是快结束时才铺。多条悬念线交替推进，让每章的活跃钩子保持在1.75个以上，避免悬念断档导致崩文。
