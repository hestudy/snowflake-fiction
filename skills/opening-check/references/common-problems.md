# 常见开篇问题诊断

## 问题1：寡淡开场

**表现**：平平无奇的开场，没有吸引力

**示例**：
```
❌ "林风是个普通的上班族，每天朝九晚五。
   今天和往常一样，他坐地铁去上班..."
```

**诊断**：缺少爆点，读者3秒内就会划走

**修复**：
```
✅ "被公司开除的第三天，林风发现银行卡里只剩下382块钱。
   手机响了——是他欠了三个月房租的房东。"
```

---

## 问题2：信息轰炸

**表现**：开篇大量倾倒世界观设定

**示例**：
```
❌ "在玄天大陆东南方，有一个三千年历史的青云宗。
   青云宗是东南方最大的宗门，有外门弟子五千人，
   内门弟子三百人，核心弟子三十人..."
```

**诊断**：读者记不住，也不想记住这些信息

**修复**：
```
✅ "外门弟子？"林默捏着那块粗糙的木牌，"就这？"
   接待的师兄冷笑："你知道有多少人挤破头都进不来吗？"
```

---

## 问题3：拖沓入题

**表现**：迟迟不进入正题

**示例**：
```
❌ [三段天气描写]
   [两段起床洗漱]
   [一段吃早饭]
   "今天是个特殊的日子..."（终于开始入题了）
```

**诊断**：读者耐心有限，不会等到你"开始"

**修复**：
```
✅ 直接从"特殊的日子"开始写
```

---

## 问题4：无爽点

**表现**：第三章没有小高潮

**示例**：
```
❌ 第三章全程铺垫，主角还在准备阶段
```

**诊断**：读者等了三章还没尝到甜头

**修复**：
```
✅ 第三章必须让主角"小赢一次"
```

---

## 问题5：无钩子

**表现**：结尾没有悬念

**示例**：
```
❌ "这就是他的一天。第二天，他又重复了同样的生活。"
```

**诊断**：读者没有追更的理由

**修复**：
```
✅ "他不知道的是，真正的危险才刚刚开始。"
```
