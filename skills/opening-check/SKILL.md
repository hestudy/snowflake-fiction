# 开篇质量检查 Skill

专门检查小说开篇（前三章）的质量，基于"黄金三章"法则。

## 为什么开篇决定生死？

网文平台数据残酷：

| 数据 | 说明 |
|------|------|
| 读者平均 **3秒** 决定是否弃书 | 第一章前300字是生死线 |
| 前三章流失率 **>80%** | 大部分读者在这里放弃 |
| 第一章有金手指/危机 → 留存率高 **40%** | 开篇设计直接影响推荐 |

## 黄金三章法则

### 第一章：3秒抓眼球

**核心任务**：从"炸楼"开始写，让读者立刻紧张起来

**开篇公式**：
```
环境异变 + 主角反常行为 + 致命威胁
```

**两种爆款开篇模板**：

| 模板 | 特点 | 示例 |
|------|------|------|
| **危机式开篇** | 主角一出场就陷入绝境 | "穿越即断头台，监斩官喊'午时已到'，她突然想起自己是穿成了被灭门的废柴嫡女" |
| **金手指落地** | 直接跳过铺垫，开局获得能力 | "重生回高考前一天，她脑中响起：'考神系统绑定，押题率100%'" |

**第一章检查清单**：

- [ ] **前300字有爆点**：危机/金手指/反转
- [ ] **主角已登场**：第一章必须出现主角
- [ ] **避免寡淡开场**：不是"XX是个普通人..."
- [ ] **避免信息轰炸**：不大段倾倒世界观
- [ ] **避免拖沓入题**：不写三段天气描写

**三大开篇雷区**：

| 雷区 | 表现 | 示例 |
|------|------|------|
| 寡淡开场 | 平平无奇 | "林风是个普通上班族，每天朝九晚五..." |
| 信息轰炸 | 大量设定 | "在玄天大陆东南方，有个三千年历史的青云宗..." |
| 拖沓入题 | 迟迟不入正题 | 先写三段天气、两段起床洗漱... |

### 第二章：强化冲突

**核心任务**：给主角一个紧迫目标 + 设置具体阻碍

**结构公式**：
```
目标可视化 + 阻碍具体化 = 读者替主角着急
```

**示例**：

| 类型 | 目标 | 阻碍 |
|------|------|------|
| 重生文 | 三天内找到家族被陷害的证据 | 反派处处监视，连出门都被拦 |
| 系统文 | 1小时内赚到1000块 | 身无分文，还被债主堵门 |
| 修仙文 | 7天内突破练气期 | 资质差，没有功法，被师兄弟排挤 |

**第二章检查清单**：

- [ ] **有紧迫目标**：主角需要完成什么？（时间限制）
- [ ] **有具体阻碍**：什么在阻止他完成目标？
- [ ] **世界观通过剧情展开**：不是说明文
- [ ] **金手指/机遇足够吸引**：让读者想要

**目标设计原则**：
- 不能太遥远（"称霸世界"❌）
- 要紧迫具体（"24小时内救回妹妹"✅）

### 第三章：爽点反馈

**核心任务**：让读者尝到甜头，即使只是小胜利

**爽点设计公式**：
```
铺垫（被看不起） → 反转（展示实力） → 反应（围观群众震惊）
```

**常见爽点类型**：

| 类型 | 示例 |
|------|------|
| 打脸小反派 | 被嘲讽赚不到钱 → 用系统赚第一桶金 → 当场打脸 |
| 突破小困境 | 被锁柴房 → 用智慧逃脱 → 展现隐藏技能 |
| 身份小曝光 | 被质疑废物 → 考试满分 → 众人震惊 |

**第三章结尾钩子类型**：

| 钩子类型 | 示例 | 效果 |
|----------|------|------|
| 悬念钩 | "他不知道的是，真正的危险才刚刚开始……" | 好奇 |
| 转折钩 | "而那个杀手，竟然是他的亲生弟弟。" | 震惊 |
| 期待钩 | "明天，就是他突破筑基期的日子。" | 期待 |
| 危机钩 | "门外传来脚步声——有人来了。" | 紧张 |

**第三章检查清单**：

- [ ] **有小爽点**：打脸小反派/突破小困境/展示隐藏技能
- [ ] **爽点与前两章呼应**：不是凭空出现的爽
- [ ] **结尾有钩子**：引导读者"加入书架"
- [ ] **让读者想看下一章**：有期待感

## 评分标准

### 开篇评分等级

| 等级 | 分数 | 预计留存率 | 建议 |
|------|------|------------|------|
| 优秀 | 80-100 | 高 | 保持即可 |
| 良好 | 60-79 | 中 | 针对性优化 |
| 及格 | 40-59 | 中低 | 需要改写 |
| 不及格 | 0-39 | 低 | 建议重写 |

### 各章评分标准

**第一章（40分）**：
- 30-40分：开篇炸裂，3秒抓住读者
- 20-29分：开篇合格，但不够抓人
- 10-19分：开篇平淡，存在明显问题
- 0-9分：开篇失败，读者会直接弃书

**第二章（30分）**：
- 24-30分：目标紧迫，阻碍明确，读者紧张
- 16-23分：有目标有阻碍，但不够强烈
- 8-15分：目标模糊或阻碍太弱
- 0-7分：没有明确目标或阻碍

**第三章（30分）**：
- 24-30分：爽点到位，钩子强力，读者追更
- 16-23分：有爽点但不够爽，钩子较弱
- 8-15分：爽点缺失或钩子缺失
- 0-7分：无爽点无钩子

## 常见开篇问题诊断

### 问题1：寡淡开场

**表现**：平平无奇的开场，没有吸引力

**示例**：
```
❌ "林风是个普通的上班族，每天朝九晚五。
   今天和往常一样，他坐地铁去上班..."
```

**诊断**：缺少爆点，读者3秒内就会划走

**修复**：
```
✅ "被公司开除的第三天，林风发现银行卡里只剩下382块钱。
   手机响了——是他欠了三个月房租的房东。"
```

### 问题2：信息轰炸

**表现**：开篇大量倾倒世界观设定

**示例**：
```
❌ "在玄天大陆东南方，有一个三千年历史的青云宗。
   青云宗是东南方最大的宗门，有外门弟子五千人，
   内门弟子三百人，核心弟子三十人..."
```

**诊断**：读者记不住，也不想记住这些信息

**修复**：
```
✅ "外门弟子？"林默捏着那块粗糙的木牌，"就这？"
   接待的师兄冷笑："你知道有多少人挤破头都进不来吗？"
```

### 问题3：拖沓入题

**表现**：迟迟不进入正题

**示例**：
```
❌ [三段天气描写]
   [两段起床洗漱]
   [一段吃早饭]
   "今天是个特殊的日子..."（终于开始入题了）
```

**诊断**：读者耐心有限，不会等到你"开始"

**修复**：
```
✅ 直接从"特殊的日子"开始写
```

### 问题4：无爽点

**表现**：第三章没有小高潮

**示例**：
```
❌ 第三章全程铺垫，主角还在准备阶段
```

**诊断**：读者等了三章还没尝到甜头

**修复**：
```
✅ 第三章必须让主角"小赢一次"
```

### 问题5：无钩子

**表现**：结尾没有悬念

**示例**：
```
❌ "这就是他的一天。第二天，他又重复了同样的生活。"
```

**诊断**：读者没有追更的理由

**修复**：
```
✅ "他不知道的是，真正的危险才刚刚开始。"
```

## 注意事项

1. **黄金三章决定生死**：前三章是读者去留的关键，必须精雕细琢
2. **开篇300字是生死线**：读者平均3秒决定是否继续
3. **第三章必须有钩子**：这是读者决定是否"加入书架"的关键
4. **爽点不能太迟**：如果第四章才开始反击，80%读者已经流失
5. **每个字都要有存在意义**：开篇没有空间浪费
