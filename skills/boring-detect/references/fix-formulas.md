# 流水账修复公式

## 公式1：无冲突 → 加意外

```
原文：事件A → 事件B → 事件C
改写：事件A → 意外/阻碍 → 情绪反应 → 解决尝试
```

### 示范

**Before：**
> 他起床，刷牙，吃早饭，出门上班。地铁上人很多，他站了一路。到公司后，他打开电脑开始工作。

**After：**
> 他刚刷完牙，手机响了——房东的催租电话。
> 他看了一眼余额：382块。房租要2000。
> 地铁上他一直在算，怎么才能凑够这个月的房租。

**改写要点：**
1. 删除无意义的日常流水（起床刷牙吃早饭）
2. 加入意外事件（催租电话）打断日常
3. 用具体数字制造紧迫感（382 vs 2000）
4. 让日常动作（坐地铁）承载情绪（焦虑算账）

## 公式2：无情绪 → 加反应

```
原文：他看到了X。
改写：他看到了X——[身体反应]。[内心独白/感官细节]。
```

### 示范

**Before：**
> 他走进房间，看到桌上的信，打开看了一遍，然后放下。

**After：**
> 他走进房间，看到桌上的信——是他等的那个offer。
> 手抖了一下，信封差点掉地上。
> 撕开信封的时候，他甚至不敢呼吸。

**改写要点：**
1. 补充信件的具体含义（offer）
2. 加入身体反应（手抖、差点掉）
3. 加入感官细节（不敢呼吸）

---

## 公式3：无期待感 → 加悬念

```
原文：这就是他的一天。
改写：[暗示未来变化]。[制造信息差]。
```

### 示范

**Before：**
> 这就是他的一天。第二天，他又重复了同样的生活。

**After：**
> 他不知道的是，那封信里藏着改变他命运的机会。
> 而现在，他正准备把它扔进垃圾桶。

**改写要点：**
1. 用"他不知道的是"制造读者与角色的信息差
2. 暗示即将发生的变化
3. 用角色的错误行为（扔掉）制造紧张感

---

## 公式4：信息堆砌 → 加互动

```
原文：[大段说明文]
改写：[角色对话] + [角色反应] + [少量关键信息]
```

### 示范

**Before：**
> 青云宗是玄天大陆东南方最大的宗门，有三千年历史，创始人是化神期大能青云真人，宗门分为外门、内门、核心弟子，外门弟子有五千人...

**After：**
> "外门弟子？"林默捏着那块粗糙的木牌，"就这？"
> 接待的师兄冷笑："就这？你知道有多少人挤破头都进不来吗？"
> "三千年了，"老者叹道，"这宗门，我看着它起高楼，也看着它..."

**改写要点：**
1. 通过角色对话展示信息
2. 用角色态度（冷笑、叹息）传递情绪
3. 只保留关键信息，删除百科式罗列

---

## 修复优先级

| 优先级 | 问题类型 | 原因 |
|--------|----------|------|
| P0 | 无冲突 | 没有冲突就没有故事 |
| P1 | 无期待感 | 没有期待感读者就弃书 |
| P2 | 无情绪 | 没有情绪就无法共情 |
| P3 | 信息堆砌 | 堆砌导致跳读 |
