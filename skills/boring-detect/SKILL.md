# 流水账检测：定位无冲突、无情绪、无期待感的平淡段落

专门检测小说中的"流水账"问题——只有事件罗列，没有冲突、情绪、期待感。

> 核心洞察：流水账是网文**最常见**也**最致命**的问题。读者可能看不出AI痕迹，但一定看得出流水账。

---

## 检测流程

```
原文分析 → 四维度扫描 → 定位问题段落 → 评分分级 → 输出报告
```

### 你的任务

1. **四维度扫描** - 逐段检测四个维度（详见知识库）
2. **定位问题段落** - 标记具体位置和问题类型
3. **评分分级** - 按权重计算综合分数
4. **给出修复建议** - 针对每个问题段落给出具体改写方向
5. **提供改写示范** - 选取最严重的问题段落进行改写示范

## 输出格式

### 标准输出（纯文本检测）

按 `./references/report-template.md` 中的单章报告模板输出。

### 核心指标

- 综合评分：0-100分（四维度加权）
- 流水账程度：无/轻微/中等/严重
- 问题段落定位：具体位置 + 问题类型 + 改进建议

## 注意事项

1. **流水账比AI痕迹更致命**：读者可能看不出AI，但一定看得出来流水账
2. **每段都需要存在意义**：如果一个段落删掉不影响故事，它就是多余的
3. **冲突是核心**：没有冲突就没有故事
4. **情绪是连接**：没有情绪就没有代入感
5. **期待感是钩子**：没有期待感就没有追更
