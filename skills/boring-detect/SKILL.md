# 流水账检测 Skill

专门检测小说中的"流水账"问题——只有事件罗列，没有冲突、情绪、期待感。

## 为什么流水账是致命问题？

流水账是网文**最常见**也**最致命**的问题：

| 流水账特征 | 读者反应 | 数据影响 |
|------------|----------|----------|
| 无冲突 | 看不下去 | 追读率暴跌 |
| 无情绪 | 无法共情 | 完读率下降 |
| 无期待感 | 看完就忘 | 书架转化低 |
| 信息堆砌 | 直接跳过 | 跳出率飙升 |

## 检测维度

### 1. 无冲突检测

检查文本是否只有事件罗列，没有矛盾碰撞。

**流水账表现**：
```
❌ "他起床，刷牙，吃早饭，出门上班，坐地铁，到公司打卡..."
   → 纯时间线推进，没有意外，没有阻碍
```

**正确做法**：
```
✅ "他刚刷完牙，手机响了——房东的催租电话。
   他看了一眼余额：382块。房租要2000。"
   → 立刻出现需要解决的问题
```

### 2. 无情绪检测

检查文本是否只有动作描述，缺少角色的情感反应。

**流水账表现**：
```
❌ "他走进房间，看到桌上的信，打开看了一遍，然后放下。"
   → 纯动作，不知道他在想什么，感受如何
```

**正确做法**：
```
✅ "他走进房间，看到桌上的信——是他等的那个offer。
   手抖了一下，信封差点掉地上。
   撕开信封的时候，他甚至不敢呼吸。"
   → 情绪层层递进
```

### 3. 无期待感检测

检查文本是否让读者想知道"接下来会发生什么"。

**流水账表现**：
```
❌ "这就是他的一天。第二天，他又重复了同样的生活。"
   → 读者完全不想知道后面
```

**正确做法**：
```
✅ "他不知道的是，那封信里藏着改变他命运的机会。
   而现在，他正准备把它扔进垃圾桶。"
   → 读者：别扔！快捡回来！
```

### 4. 信息堆砌检测

检查文本是否大量灌输设定和背景。

**流水账表现**：
```
❌ "青云宗是玄天大陆东南方最大的宗门，有三千年历史，
   创始人是化神期大能青云真人，宗门分为外门、内门、
   核心弟子，外门弟子有五千人..."
   → 像读说明书一样枯燥
```

**正确做法**：
```
✅ ""外门弟子？"林默捏着那块粗糙的木牌，"就这？"
   接待的师兄冷笑："就这？你知道有多少人挤破头都进不来吗？"
   → 通过角色互动展示信息
```

## 流水账检测清单

```
请检测以下文本的流水账问题：

## 检测清单

### 冲突检测
- [ ] 本章是否有明确的冲突/矛盾？
- [ ] 主角是否面临需要解决的问题？
- [ ] 是否有人物之间的对立/碰撞？
- [ ] 冲突是否足够强烈？

### 情绪检测
- [ ] 是否有角色的情绪变化？
- [ ] 读者是否能感受到角色的感受？
- [ ] 每段是否有情绪锚点？
- [ ] 是否有感官描写（视觉/听觉/触觉）？

### 期待感检测
- [ ] 读者是否想知道接下来发生什么？
- [ ] 章节结尾是否有钩子/悬念？
- [ ] 是否有未解决的问题吸引读者？
- [ ] 是否有伏笔让读者好奇？

### 节奏检测
- [ ] 是否有张弛变化？
- [ ] 是否全程平淡或全程紧绷？
- [ ] 信息密度是否合理？
- [ ] 是否有过渡拖沓？
```

## 流水账程度分级

| 等级 | 分数 | 描述 | 表现 | 建议 |
|------|------|------|------|------|
| **无流水账** | 80-100 | 内容扎实 | 有冲突、有情绪、有期待 | 保持即可 |
| **轻微** | 60-79 | 偶有平淡 | 少数段落缺乏张力 | 局部优化 |
| **中等** | 40-59 | 多处问题 | 多段缺乏吸引力 | 重点改写 |
| **严重** | 0-39 | 通篇流水账 | 全程平铺直叙 | 需要重构 |

## 流水账修复公式

### 无冲突 → 加意外

```
原文：事件A → 事件B → 事件C
改写：事件A → 意外/阻碍 → 情绪反应 → 解决尝试
```

### 无情绪 → 加反应

```
原文：他看到了那封信。
改写：他看到了那封信——手抖了一下，差点拿不住。
```

### 无期待感 → 加悬念

```
原文：这就是他的一天。
改写：他不知道的是，真正的麻烦才刚刚开始。
```

### 信息堆砌 → 加互动

```
原文：青云宗有三千年历史...
改写："三千年了，"老者叹道，"这宗门，我看着它起高楼，也看着它..."
```

## 常见流水账场景

| 场景 | 流水账表现 | 修复方案 |
|------|------------|----------|
| **日常描写** | 起床刷牙吃早饭 | 加入意外打断 |
| **路途描写** | 坐地铁走路到公司 | 用内心戏/回忆填充 |
| **背景介绍** | 大段世界观说明 | 通过对话/冲突展示 |
| **时间过渡** | 三天过去了 | 用事件锚定时间 |
| **战斗描写** | 一招一式平铺 | 加入心理博弈 |

## 注意事项

1. **流水账比AI痕迹更致命**：读者可能看不出AI，但一定看得出来流水账
2. **每段都需要存在意义**：如果一个段落删掉不影响故事，它就是多余的
3. **冲突是核心**：没有冲突就没有故事
4. **情绪是连接**：没有情绪就没有代入感
5. **期待感是钩子**：没有期待感就没有追更
