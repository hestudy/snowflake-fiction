# 小说导出格式说明

## 平台格式要求

### 番茄小说
- 格式：纯文本
- 要求：
  - 用回车分段，段落之间不留空行
  - 禁用空格、Tab 缩进
  - 禁用特殊字符和格式标记
  - 平台自动处理首行缩进

### 起点中文网
- 格式：纯文本 / 作家助手
- 要求：
  - 每段首行缩进2字符（4个半角空格或2个全角空格）
  - 章节标题单独一行
  - 使用简体中文标点

### 晋江文学城
- 格式：纯文本
- 要求：
  - 遵循晋江原创写作规范
  - 禁止特殊排版
  - 正文使用标准段落格式

### 知乎盐选
- 格式：在线编辑
- 要求：
  - 直接在后台编辑器粘贴
  - 支持基础格式（加粗、斜体）
  - 短篇为主（2-5万字）

## 格式转换规则

### Markdown → 纯文本

| Markdown | 转换为 |
|----------|--------|
| `# 标题` | 删除，保留为章节名 |
| `## 小标题` | 删除 |
| `**粗体**` | 去除标记，保留文字 |
| `*斜体*` | 去除标记，保留文字 |
| `> 引用` | 转为普通段落 |
| `---` | 删除 |
| `` `代码` `` | 去除标记 |
| `[链接](url)` | 只保留文字 |
| 列表项 | 转为普通段落 |

### 段落处理

```
原文（Markdown）：
这是第一段。

这是第二段。

转换后（番茄）：
这是第一段。
这是第二段。

转换后（起点））：
　　这是第一段。
　　这是第二段。
```

## 导出提示词模板

### 导出为番茄小说格式

```
请将以下小说内容转换为番茄小说投稿格式：

## 转换规则

1. **去除所有 Markdown 标记**
   - 删除 # 标题标记
   - 删除 ** 粗体标记
   - 删除 * 斜体标记
   - 删除 > 引用标记
   - 删除 --- 分隔线
   - 删除 ` 代码标记
   - 删除 []() 链接标记

2. **段落处理**
   - 每段用单个回车分隔
   - 段落之间不留空行
   - 不添加任何缩进空格

3. **章节标题**
   - 保留章节名，单独一行
   - 格式：第X章 章节名

4. **清理**
   - 删除多余的空行
   - 确保使用简体中文标点
   - 检查特殊字符

---

[小说内容]
```

### 导出为起点中文网格式

```
请将以下小说内容转换为起点中文网投稿格式：

## 转换规则

1. **去除所有 Markdown 标记**
   - 删除所有格式标记

2. **段落处理**
   - 每段开头添加2个全角空格（　　）
   - 段落之间不留空行

3. **章节标题**
   - 格式：第X章 章节名
   - 标题单独一行，不加缩进

4. **标点规范**
   - 使用简体中文标点
   - 统一引号为 ""（中文双引号）

---

[小说内容]
```

### 导出为通用纯文本

```
请将以下小说内容转换为通用纯文本格式：

## 转换规则

1. 去除所有 Markdown 标记
2. 每段开头添加2个全角空格
3. 章节之间保留一个空行
4. 使用中文标点

---

[小说内容]
```

## 批量转换脚本

如果有多章内容需要转换，可以使用以下流程：

```bash
# 假设正文在 novel-output/[小说名]/正文/ 目录
# 每章一个 .md 文件

# 转换所有章节
for file in novel-output/*/正文/*.md; do
  # 调用 Claude 进行格式转换
  # 或使用 pandoc 工具
done
```

### 使用 Pandoc 转换

```bash
# Markdown 转纯文本
pandoc input.md -t plain -o output.txt

# Markdown 转 Word
pandoc input.md -o output.docx
```
