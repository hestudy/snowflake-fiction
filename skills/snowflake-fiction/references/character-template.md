# 人物卡片完整模板

本文档提供详细的人物卡片模板，用于步骤 3（人物介绍）和步骤 7（人物宝典）。

---

## 简版人物卡片（步骤 3 使用）

适用于快速建立角色基础设定。

```markdown
## [角色名]

### 核心设定

| 项目 | 内容 |
|------|------|
| 身份 | [年龄/职业/社会地位] |
| 目标 | [他想要什么？] |
| 矛盾 | [什么阻止他？] |
| 价值观 | [他相信什么？] |
| 顿悟 | [他最终领悟什么？] |

### 一句话经历
> [≤50字]

### 标志元素
- 口头禅：
- 习惯动作：
- 标志性物品：
```

---

## 使用建议

1. **主角**：填写所有模板
2. **反派**：重点填写心理和背景
3. **配角**：使用简版 + 部分详细项
4. **工具人**：仅填写基本信息

---

## 人物一致性检查清单

写作时，定期检查：

- [ ] 对话风格是否符合人物设定？
- [ ] 行为是否符合价值观？
- [ ] 情绪反应是否符合性格？
- [ ] 标志性语言/动作是否使用？
- [ ] 与其他人物的关系是否一致？
