# 人物卡片完整模板

本文档提供详细的人物卡片模板，用于步骤 3（人物介绍）和步骤 7（人物宝典）。

---

## 简版人物卡片（步骤 3 使用）

适用于快速建立角色基础设定。

```markdown
## [角色名]

### 核心设定

| 项目 | 内容 |
|------|------|
| 身份 | [年龄/职业/社会地位] |
| 目标 | [他想要什么？] |
| 矛盾 | [什么阻止他？] |
| 价值观 | [他相信什么？] |
| 顿悟 | [他最终领悟什么？] |
| 行为底层逻辑 | [角色做一切事情的根本驱动力，通篇遵循此逻辑] |
| 立场 | [角色所处的阵营/利益方，冲突来自立场而非善恶] |

### 一句话经历
> [≤50字]

### 标志元素
- 口头禅：
- 习惯动作：
- 标志性物品：
- 时间标签：[与时间绑定的特殊设定，如"每逢月圆夜功力尽失"]
- 记忆锚点：[身体/外貌上的独特标记，如"右臂20cm刀疤"，可作为关键伏笔]

### 反差设定（组合反差标签法）

**表层反差**（出场即可揭示，制造第一印象）：
- 视觉反差：[外表与能力/武器的反差，如"娇小女生使大铁锤"]
- 行为反差：[性格与某个行为的反差，如"高冷人设看到蘑菇就想笑"]
- 语言反差：[身份与说话方式的反差，如"硬汉口头禅是'你讨厌死了啦'"]

**深层反差**（需层层压力揭示，构成圆形人物）：
- 价值反差：[表面价值观与深层信念的反差，如"看似玩世不恭，实则蓄积力量操弄权势"]
- 目标反差：[表面追求与真实渴望的反差，如"嘴上说不在乎，实则比谁都想赢"]

**反差公式**：「看似[表层标签]，实则[深层标签]」

### 双重身份
- 表面身份：
- 隐藏身份：
- 掉马触发条件：[什么情况下身份暴露]

### CP关系设计（言情/多角色关系适用）

> 将CP作为整体设计，天然冲突+致命吸引=完整CP。

| 项目 | 内容 |
|------|------|
| CP对象 | [对方角色名] |
| CP类型 | [一见钟情/日久生情/由恨生爱/步步为营] |
| CP关系 | [单向暗恋/双向暗恋/明恋/你爱我恨/被迫绑定] |
| 天然冲突 | [三观/立场/社会身份/伦理身份/阶级地位上的对抗性] |
| 致命吸引 | [为什么会被对方吸引？基于什么特质？] |
| 核心性格标签 | [只选1个最显著的，如：抠门/冲动/理性] |
| 标签互补 | [与CP对象的标签如何形成冲突互补？] |
```

## 完整人物宝典（步骤 7 使用）

适用于深入挖掘角色，确保人物立体一致。

### 模板一：基础档案

```markdown
# [角色名] - 人物宝典

## 基本信息

| 项目 | 内容 |
|------|------|
| 全名 | |
| 昵称/别名 | |
| 性别 | |
| 年龄 | |
| 生日 | |
| 星座 | |
| 血型 | |
| 出生地 | |
| 现居地 | |
| 国籍/民族 | |

## 外貌特征

> **描写原则**：动态描写 > 静态描写。不要干巴巴列举外貌，而是通过出场动作、他人反应、环境互动来展现。
> 参考：《红楼梦》王熙凤出场——先闻笑声→再听说话→旁人反应→最后描写容貌。
>
> **速写公式**：`1个显著特征 + 1个关联动作` = 鲜活的人物形象。
> 示例：他脸上有颗痣，痣上有根毛。每次琢磨怎么害人时，手就不自觉地摸那颗痣，捋那根毛。
> 不必从头到脚面面俱到，抓住最具辨识度的那个特征，配上一个习惯性动作，人物形象就活了。

| 项目 | 内容 |
|------|------|
| 身高 | |
| 体重 | |
| 体型 | （瘦削/健壮/丰满等）|
| 发色/发型 | |
| 瞳色 | |
| 肤色 | |
| 脸型 | |
| 显著特征 | （疤痕、痣、胎记等——可作为记忆锚点和伏笔）|
| 穿衣风格 | |
| 声音特点 | |
| 气味 | （香水、烟草等）|
| 走路姿态 | |
| 标志性表情 | |
| 出场方式设计 | [如何通过动态描写让角色出场令人印象深刻] |
```

```markdown

## 黑化/觉醒触发点（3+1公式）

**3个关键事件节点**：
1. [触发事件1]：[如亲友背叛] → 转变方向：[如开始不信任他人]
2. [触发事件2]：[如力量反噬] → 转变方向：[如对自身能力产生恐惧]
3. [触发事件3]：[如信仰崩塌] → 转变方向：[如价值观重建]

**1个隐藏开关**：
- [隐藏触发]：[如童年记忆封印/血脉诅咒] → 在关键时刻激活，推动剧情转折

## 人物弧光的「同一罅隙」验证法

> 最能突出人设改变的方式：将人物逼到同一个罅隙里——同样的危机、同样的选择题，这一次人物做出了不同的反应。

**设计方法**：
1. 在故事前期设置一个关键选择场景
2. 在故事后期重现相似的选择场景
3. 人物做出不同的反应，体现弧光完成

**示例**：

| 时间点 | 场景 | 选择 |
|--------|------|------|
| 开头 | [填写初始选择场景] | [填写初始选择，如"我想修仙，荡平天下不平事"] |
| 结尾 | [填写呼应选择场景] | [填写转变后选择，如"我早就不练剑了，世道如此"] |

经典参考：《我不是药神》——同样是搞仿制药，开头为了赚钱，后来为了救人。看似选择一致，深层设定已完成反转。
```

## 使用建议

1. **主角**：填写所有模板（含模板九反差设定 + 模板十成长可视化 + CP关系设计）
2. **反派**：重点填写心理和背景 + 模板九记忆点强化 + 行为底层逻辑
3. **配角**：使用简版 + 模板九中的1个记忆点 + 立场定位（记住：记忆点 > 完整性）
4. **工具人**：仅填写基本信息
5. **言情文CP**：必须填写CP关系设计模板，确保天然冲突+致命吸引都有设定

## 人物一致性检查清单

写作时，定期检查：

- [ ] 对话风格是否符合人物设定？
- [ ] 行为是否符合价值观？
- [ ] 行为是否遵循角色的底层行为逻辑？
- [ ] 情绪反应是否符合性格？
- [ ] 标志性语言/动作是否使用？
- [ ] 与其他人物的关系是否一致？
- [ ] 角色的反常行为是否有充分铺垫？（合理反常 = 一贯表现 + 特殊触发点）
- [ ] 配角的行为是否基于立场而非简单善恶？
