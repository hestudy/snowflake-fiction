## 零、通用拆书方法论

> 来源：番茄签约作者任欢游（300万字创作经验）的实战方法论。
> 核心理念：学习火文要拆功能/作用，不拆细节。越是框架层面的东西，越方便转化为自己能运用的信息。

### 拆书的正确姿势

**❌ 错误方式：拆细节**
- 逐句分析台词含义、隐含情绪、修辞手法
- 信息太琐碎，难以消化和转化
- 看得越多越迷糊

**✅ 正确方式：拆功能/作用**
- 每一章/每一段剧情的核心作用是什么？
- 这个设定/情节服务于什么目的？

示例（拆《缚春情》前四章）：
```
第1章作用：女主守寡六年，男配跟女配在外定情还带回家 → 刻画背叛情绪
第2章作用：男配拿女主衣服给女配 → 拉仇恨
第3章作用：继续拉仇恨 → 蓄力
第4章作用：疑似男主出现 → 抛出悬念
```

### 拆书的四个核心维度

| 维度 | 关注点 | 示例问题 |
|------|--------|----------|
| 大悬念 | 全文/阶段性悬念如何设置和衔接 | 第一个大悬念是什么？第二个何时接上？ |
| 钩子 | 章节结尾如何勾住读者 | 哪个章节结尾让你最想看下一章？为什么？ |
| 节奏 | 情绪起伏的分布和密度 | 压抑了多久才爆发？爆发后多久再起新钩子？ |
| 故事性 | 人物冲突和戏剧张力的来源 | CP之间的天然冲突是什么？致命吸引是什么？ |

### 实操建议

1. 找一本同题材的爆款书，用"一句话概括每章作用"的方式快速拆完前30章
2. 标记出让你产生强烈情绪的章节，分析该章节用了什么技巧（悬念？不公平+对比？信息差？）
3. 提取该书的大悬念链条：悬念1→悬念2→悬念3，看它们如何一环扣一环
4. 将提取的框架套用到自己的大纲中，替换具体内容

### 通用大纲写作方式

> 来源：番茄小说LV5作者鹿时七（代表作《老娘心理学博士，进宫嘎嘎乱杀》）
> 三种常见大纲构建方法，适用于所有网文题材。

**方式一：起承转合（四分法）**

| 阶段 | 内容 |
|------|------|
| 起 | 故事背景 |
| 承 | 故事起因 |
| 转 | 故事高潮 |
| 合 | 故事结局 |

**方式二：五分法A（破题→结局）**

| 阶段 | 说明 |
|------|------|
| 破题 | 开头用剧情巧妙点明故事主旨，让读者立刻明白讲的是什么故事 |
| 开头 | 以冲突矛盾的剧情切入，快速吸引读者注意 |
| 发展 | 推进主线，展开人物关系和矛盾 |
| 高潮 | 核心冲突爆发 |
| 结局 | 收束故事线 |

要点：网文讲究快、准、狠，开篇不要大面积介绍背景和人物，以冲突矛盾开头，具体的主角背景、人物身份用剧情带出来。

**方式三：五分法B（目标→结果）**

| 阶段 | 说明 |
|------|------|
| 目标 | 主角想要达成什么 |
| 障碍 | 什么阻碍了目标 |
| 行动 | 主角采取什么行动 |
| 意外 | 过程中出现什么意外转折 |
| 结果 | 最终结局如何 |

示例（古言换亲文）：
```
背景：前世嫡姐嫁侯府世子→世子病死→嫡姐成寡妇受尽白眼；
      女主（庶女）嫁穷书生→书生高中状元→但用的是女主的诗词和替考。
      嫡姐嫉妒杀女主后自杀，两人双双重生。

目标：换亲（女主要嫁给世子）
障碍：父母不同意
行动：制造意外，和世子双双落水
意外：发现世子装病，世子要杀她灭口
结果：自救活下来嫁给世子，相爱后救下侯府满门并谋反成功，
      与世子幸福美满一生，嫡姐竹篮打水一场空。
```

### 明确关键点

每个故事都需要明确以下关键点，缺一不可：

| 关键点 | 作用 |
|--------|------|
| 爽点 | 让读者获得满足感的情节（打脸、逆袭、升级） |
| 爆点 | 引发强烈情绪反应的转折（反转、揭秘、高潮） |
| 反派塑造 | 极端拉仇恨，让读者期待反派被打脸 |
| 群像 | 丰满的配角群，各有性格和立场 |
| 笑点 | 调节节奏，增加趣味性和亲和力 |

### 剧情服务原则

一切情节、对话、景语都要为推动剧情而服务：

1. 清空自我，贴着人物来写（不是作者在说话，是角色在说话）
2. 尊重小说人物的意志、情感及其独立性
3. 对话要同时承担多重功能：呈现动机、推动叙事、勾画关系、构成情境、描述心理、展开空间、生成趣味、表达观念

## 一、游戏动漫类

### 1.1 题材定义与分类

以游戏、竞技元素为主的小说作品，包括虚拟网游、异世游戏、电子竞技、体育竞技、二次元轻小说等。

| 子类型 | 定义 | 代表作品 |
|--------|------|---------|
| 虚拟网游 | 现实为基础，游戏世界与现实界限清晰 | 《开局几亿个满级账号》《网游之近战法师》《亏成首富从游戏开始》 |
| 游戏异界 | 进入游戏异世成为游戏角色，或带游戏面板穿越 | 《迷雾求生：我能看到提示》《网游：开局选到了最强种族》 |
| 游戏竞技 | 电子竞技/体育竞技/游戏直播，侧重运动员刻画 | 《全职高手》《冠军之心》 |
| 轻小说 | 偏动漫质感，情节日常，人物标签二次元化 | 《开局地摊卖大力》《我加载了恋爱游戏》 |

### 1.2 写作技巧

**虚拟网游**：
1. 开头用新奇/诡异事件切入，或通过主角做任务交代游戏设定
2. 前三章就要有小高潮（主角利用金手指打脸对手）
3. 注意主角人物弧光，通过游戏成长收获个人成长
4. 有意思的NPC可以增加有趣度

**游戏异界**：
- 写法与虚拟网游大致相同，但自带新奇点，可制造笑点增强趣味性
- 世界观尽量简单方便读者消化，用共识术语帮助读者快速进入状态
- 与同类型作品有区别的金手指元素，或融合其他类型（网游+玄幻/仙侠）

**游戏竞技**：
1. 一般从逆风翻盘角度切入，或天才型选手一路开挂
2. 注重情节曲折性和生活化桥段引起共鸣
3. 世界观需要足够真实，代入感和现实逻辑更重要
4. 人物需要曲折的成长曲线，配角描写同样重要
5. 整体风格积极向上（努力、拼搏、热血、友情）

**轻小说**：
- 人物需要二次元特点（萌、可爱、画面感）
- 情节轻松少戾气，传统打脸套路不适用
- 侧重心理描写，文笔相对细腻
- 写作难点：如何不尴尬、不突兀地制造笑点

### 1.3 注意事项

- **金手指独特性**：通过对比突出独特性和爽感
- **术语控制**：开篇不要大段游戏术语，只解释当前情节最相关的，其他逐步展开
- **升级感设计**：每次升级要有明确奖励和目标，通过对比衬托让读者感受重要性
- **避免升级机器**：主角的成长是人物弧光，不是不停升级

### 1.4 选题检查专项

- [ ] 金手指是否与同类作品有明确差异？
- [ ] 开篇是否避免了大段游戏术语堆砌？
- [ ] 升级是否有明确奖励感和对比衬托？
- [ ] 主角是否有游戏之外的个人成长？
- [ ] 世界观是否简单易消化？（游戏异界类）
- [ ] 是否有跨类型融合增加新鲜感？
- [ ] 竞技类配角描写是否充分？
- [ ] 轻小说人物是否有二次元特点？

### 1.5 开篇检查专项

**第一章**：
- [ ] 是否避免了大段游戏术语堆砌？
- [ ] 金手指是否在第一章展示了独特性？
- [ ] 游戏异界类是否利用了"成为游戏角色"的新奇点？

**第三章**：
- [ ] 是否有利用金手指的小高潮（打脸对手）？
- [ ] 升级奖励是否通过对比衬托让读者感受到重要性？

**常见开篇问题**：

| 问题 | 表现 | 修复 |
|------|------|------|
| 术语堆砌 | 开篇大段游戏术语 | 只解释当前情节最相关的术语 |
| 升级机器 | 主角无感情地刷怪升级 | 前三章要有小高潮，展示人物弧光 |

### 1.6 流水账检测专项

| 流水账模式 | 表现 | 修复方向 |
|-----------|------|---------|
| 升级机器 | 主角不停升级，没有感情变化 | 通过游戏成长收获个人成长 |
| 术语说明书 | 开篇大段交代职业、技能树、装备等级 | 只解释当前情节最相关的术语 |
| 刷怪流水线 | "杀怪→经验→升级"无限循环 | 每次升级要有对比衬托 |
| NPC对话堆砌 | 大段NPC对话交代任务背景 | NPC也要有个性，通过有趣互动展示信息 |

### 1.7 角色检查专项

**虚拟网游/游戏异界**：
- [ ] NPC是否有个性（不只是任务发布器）？
- [ ] 主角是否有游戏外的人格？
- [ ] 金手指是否替代了角色成长？

**游戏竞技**：
- [ ] 队友是否有独立人格和成长曲线？
- [ ] 主角成长曲线是否曲折？
- [ ] 团队关系是否有摩擦、磨合、信任建立？

**轻小说**：
- [ ] 人物是否有二次元特点（傲娇/天然呆/腹黑等）？
- [ ] 心理描写是否细腻？

### 1.8 质量评估专项

| 评估项 | 检查要点 | 常见问题 |
|--------|---------|---------|
| 爽感设计 | 金手指独特性？升级有对比衬托？ | 金手指太泛，升级无感 |
| 升级节奏 | 每次升级有明确奖励和目标？ | 单一升级后期疲软 |
| 人物弧光 | 主角有游戏之外的个人成长？ | 只是升级机器 |
| 世界观展开 | 设定通过情节带出？ | 开篇疯狂输出设定 |
| 趣味性 | NPC有趣？世界有新奇点？ | NPC只是任务发布器 |

竞技类额外：情节曲折性、热血感、真实感
轻小说额外：笑点质量、节奏感、尺度控制

## 二、科幻类

### 2.1 题材定义与分类

用幻想艺术的形式，表现科学技术远景或社会发展对人类的影响。科幻的关键是新奇感。

| 分类 | 定义 | 代表作品 |
|------|------|---------|
| 硬科幻 | 严格技术推演，描写可能实现的新技术 | 《基地》《太空漫游2001》《三体》 |
| 软科幻 | 科技严谨性要求不高，更注重社会发展 | 《沙丘》《星球大战》 |
| 末世文 | 社会秩序崩溃后的新末世社会 | 《末世：开局先囤十亿物资》《冰河末世：我抢光千亿超市物资》《灾变：除我以外，全球人类失明了》 |
| 黑科技文 | 都市背景，金手指是未来黑科技 | — |
| 星际文 | 星际为大背景，想象星际文明不同形态 | 《深渊星芒》 |

> 网文科幻偏软科幻，科技严谨性要求不高，把"科技"或"未来"作为背景展现爽点。

### 2.2 写作技巧

**末世文**：
- 核心：用创新点抓住读者，制造陌生感和新奇感
- 创新点：类型创新（灾变文淡化丧尸）、金手指创新（结合流行元素）、副本创新
- 人性在社会崩溃后的变形是重要看点
- 示例：人物互相祝福"祝蘑菇丰收，老鼠满仓"——末世中的陌生化人际关系

末世文书名：
- 推荐格式："末世：[核心吸引点]"，先点明题材方便精准吸引读者，后半段放核心卖点
- 示例：《末世：开局先囤十亿物资》（囤物资噱头）、《末世：我能听到别人心声》（新颖金手指）
- 书名要有有效信息量，把核心特色和吸引点提炼出来

末世文世界设定：
- 基础设定：正常世界 + 丧尸病毒 → 停水停电缺粮、秩序崩溃（本质是都市文变种，可用生活常识）
- 超凡因素：引入异能觉醒（幸存者和丧尸都觉醒异能，眉心晶体，吸取晶体升级）= 网游打怪模式
- 微创新方向（在老套路中加入新元素，更容易出成绩）：
  - 冰河末世：《冰河末世：我抢光千亿超市物资》
  - 虫族入侵、海平面上升、太阳黑子等非丧尸末世类型
  - 灾变文淡化丧尸，用其他灾难类型制造差异化

末世文人设：
- 核心人设：杀伐果断、宁可我负天下人的枭雄角色
- 原因：末世秩序不存、物资短缺，不够狠就会被丧尸杀死或落入陷阱，乱世出枭雄
- 常见搭配：重生归来（抢占先机、夺取机缘的爽感）+ 枭雄人设
- 示例：《末世：开局先囤十亿物资》主角比较冷血，对有威胁的人都会杀掉
- 注意：不是只能写枭雄人设，但枭雄更容易写、更符合读者喜好，推荐新手作者

末世文剧情主线：
- 前期主线：生存（即使以重生复仇开篇，复仇后首先考虑的仍是生存）
- 后期主线：提升实力、终结末世
- 围绕主线展开的典型结构（以《末世：开局先囤十亿物资》前10万字为例）：
  1. 囤积物资（十亿物资从哪来？→ 引出有仇有钱的大伯 → 坑大伯拿资金）
  2. 建立势力（保障生存安全 → 救出雷电法王、收服苏大柱、米灵等）
  3. 有意放缓节奏（物资提前囤好但不能暴露空间异能 → 末世后一一收取 → 路上扫荡丧尸）
  4. 更高生活追求（物资到位后追求美食 → 找厨师 → 又一次行动）
  5. 核心小团体成立 → 以别墅区为基地建立更大团体
- 关键：主线明确，每一步都跟着主线走，读者粘性高

末世文金手指创新：
- 金手指是末世文差异化竞争的关键，新颖的金手指能迅速吸引大量读者
- 创新金手指案例：
  - 《末世游戏：我能给万物加前缀名》：给任何物体添加前缀名附加特殊属性
  - 《末世，我抽奖神级旅行青蛙团》：召唤不会被丧尸攻击的蛙崽帮助求生
  - 《末世：我觉醒了无限拾取异能》：拾取体质、精神、异能等
- 大框架下的小创新能让书与众不同，更容易被读者喜爱

**黑科技文**：
- 核心升级体系：科技树。科技树是黑科技文的标配，也是爽点最主要的来源
- 科技树结构：不同科技之间有等级划分（1级/2级/3级），有前后顺序，要按顺序点亮，高级科技有低级前置科技
- 科技树可分为几个大领域（能源科技、材料科技、武器科技等），每个领域内的科技可能有多个领域的前置科技
- 爬科技树=玄幻文修炼升级，让读者获得收获感、成就感、碾压感
- 科技树独特优势①：升级变强的不是主角一个人，而是主角所属群体（国家/人类），可写出更多群体内部互动情节（主角受尊敬崇拜等）
- 科技树独特优势②：一套全新的升级体系，有足够新颖度避免同质化
- 搭建科技树四要点：
  1. 用关键科技划分不同层次（如1级文明关键科技是"可控核聚变"），高层次对低层次有绝对力量差距（参考《三体》水滴全灭人类舰队、歌者二向箔毁灭太阳系）
  2. 让读者明确感知科技直接转化成力量：更强大的能源→更强大的材料→更强大的武力→打开更高级的地图
  3. 不必受太多现实合理性束缚，核心考量是作为升级节点能否发挥爽点（类似玄幻不考量功法是否合理）
  4. 充分发挥想象力，每个新科技及其转化方式都可以不断创新，让读者持续受到吸引
- 可与其他类型结合（黑科技+末世/灾变）
- 代表作品：《灾变时代：我能点亮科技树》《火星遗迹曝光，忠烈十三子》

**星际文**：
- 星际文最大特点：大量的、自由度非常高的不同地图和不同种族，这是太空歌剧的最大魅力
- 每个星球=一个世界/副本，核心是写出各世界之间的互动
- 给每个副本设置不同特色、种族、势力、文明形态
- 黑科技文与星际文相通：同一个科技树升级体系，开局在都市=黑科技文，开局在星际=星际文，只是开局位置在科技树的不同位置
- 黑科技文主线发展一定会开新地图，新地图往往在太空中，所以大部分会逐渐进入星际地图
- 星际元素为黑科技文提供源源不断的新地图、新种族，让升级之路上可以不断写出新情节
- 偏修炼的星际文：升级体系不是科技而是修炼，实际上可当玄幻文来写，星际元素提供更多地图和种族的多样性，增加区分度和记忆点
- 可加入高魔元素（原力、幽能、灵能等）
- 重视远古遗迹、上古神明等偏奇幻的设定
- 代表作品：《对不起，我的星际舰队只招女兵》《深渊星芒》

### 2.3 选题检查专项

- [ ] 是否为读者制造了陌生感和新奇感？
- [ ] 末世文：是否与其他末世文拉开区分度（类型/金手指/副本创新）？
- [ ] 末世文：书名是否采用"末世：[吸引点]"格式，核心卖点是否在书名中体现？
- [ ] 末世文：世界设定是否有微创新（不只是千篇一律的丧尸末世）？
- [ ] 末世文：人设是否契合末世混乱无序的环境（枭雄/果断型更易出成绩）？
- [ ] 末世文：剧情主线是否明确（生存→变强→终结末世）？
- [ ] 末世文：金手指是否有创新（与同类作品有明确差异）？
- [ ] 黑科技文：是否搭建了科技树升级体系（层次划分+前置科技路径）？
- [ ] 黑科技文：科技是否能直接转化为力量（能源/材料/武力/地图）？
- [ ] 黑科技文：科技树是否有足够想象力（不受现实合理性过度束缚）？
- [ ] 星际文：不同星球/世界是否有差异化设计（种族/文明/特色）？
- [ ] 星际文：升级体系是科技树还是修炼？是否明确？
- [ ] 同质化规避：是否有足够的创新点？

### 2.4 开篇检查专项

**第一章**：
- [ ] 是否用具体的陌生化细节切入，而非大段世界观说明？
- [ ] 前300字内是否让读者感受到"这个世界不一样"？
- [ ] 末世文：重生类是否快速展示主角的信息优势（知道末世要来）？
- [ ] 末世文：金手指是否在第一章有所展示或暗示？
- [ ] 黑科技文：科技树是否快速展示（让读者感知到升级体系的存在）？

**第二章**：
- [ ] 末世文：是否展示了社会崩溃后的新规则和新人际关系？
- [ ] 末世文：主角是否围绕"生存"主线有明确行动（囤物资/建势力/收服队友）？
- [ ] 星际文：是否展示了第一个"星球副本"的独特特色？
- [ ] 黑科技文：科技升级是否让读者直观感受到力量变强？

**第三章**：
- [ ] 末世文：是否有利用金手指/信息优势的小高潮？
- [ ] 末世文：主角的枭雄人设是否通过具体行动体现？

**常见开篇问题**：

| 问题 | 表现 | 修复 |
|------|------|------|
| 设定先行 | 开篇大段描写末世/星际世界观 | 用陌生化细节切入，通过情节展示设定 |
| 科技说明文 | 大段解释科技原理和参数 | 重点展示科技转化为力量后的爽感 |

### 2.5 流水账检测专项

| 流水账模式 | 表现 | 修复方向 |
|-----------|------|---------|
| 设定百科全书 | 大段描写世界观、科技体系、文明等级 | 通过角色遭遇逐步展示设定 |
| 副本流水线 | 一个副本接一个副本，缺乏情感连接 | 每个副本要有角色成长和情感变化 |
| 科技说明文 | 详细解释黑科技原理和参数 | 重点是科技转化为力量后的爽感，而非科学严谨性 |
| 探索日志 | "到达A星球→描写→到达B星球→描写" | 每个星球要有独特冲突和种族互动 |
| 升级无感 | 科技树升级了但读者感受不到力量变化 | 每次升级要安排情节展示科技→能源/材料/武力/地图的转化 |
| 囤物资流水账 | 末世文"找物资→收物资→找物资"无限循环 | 每次行动要有人际冲突、收服队友、遭遇危机等情节穿插 |
| 打丧尸流水线 | 末世文"遇丧尸→杀丧尸→升级"模式单一 | 丧尸战斗要有策略变化，穿插人性冲突和团队互动 |

### 2.6 角色检查专项

- [ ] 末世文：角色在社会崩溃后是否展现了人性变化？
- [ ] 末世文：主角人设是否契合末世环境（杀伐果断/枭雄型更易出成绩）？
- [ ] 末世文：主角是否有明确的行为逻辑（不是无脑善良也不是无脑杀戮）？
- [ ] 末世文：团队成员是否各有特色和独立人格（不只是主角的跟班）？
- [ ] 星际文：不同文明/种族角色是否有差异化行为逻辑？
- [ ] 避免所有外星种族都像"换了皮肤的人类"
- [ ] 黑科技文：主角群体升级后，群体内部互动是否有情节展开（尊敬/崇拜/嫉妒等）？
- [ ] 黑科技文：主角是否有独立人格（不只是科技树的点亮工具）？

### 2.7 质量评估专项

| 评估项 | 检查要点 | 常见问题 |
|--------|---------|---------|
| 新奇感 | 世界观是否有想象力？ | 设定平庸 |
| 创新度 | 与同类作品是否有差异？ | 同质化严重 |
| 代入感 | 科幻设定是否影响代入感？ | 设定过于复杂 |
| 社会影响 | 科技/灾变是否对社会产生深层影响？ | 只用科幻元素装逼 |
| 升级体系 | 科技树层次是否清晰？升级是否有爽感？ | 科技升级无感、层次模糊 |
| 力量转化 | 科技是否直观转化为力量（能源/材料/武力/地图）？ | 升级了但读者感受不到变强 |

黑科技文额外：科技树想象力、群体升级互动、科技层次碾压感
星际文额外：地图多样性、种族差异化、升级体系与地图扩展的联动
末世文额外：金手指创新度、主线明确度（生存→变强→终结末世）、人设契合度（枭雄型）、世界设定微创新

## 三、历史类

### 3.1 题材定义与分类

以古代社会为背景，通过描写特定历史人物和事件，再现一定历史时期的经济文化、政治军事、社会风俗及历史发展进程。

**品类细分**：
1. 金手指历史 vs 非金手指历史（是否带系统）
2. 架空历史 vs 非架空历史（非架空要求更严谨）
3. 朝代划分：夏商周/秦汉/两晋南北朝/隋唐/五代十国/宋元明清
4. 标签划分：种田、朝堂、智谋、争霸、科技、科举、架空、系统

**特点**：严谨 + 有趣
- 严谨：戴着镣铐跳舞，以史为鉴，不能悖谬历史规律
- 有趣：小说本质是讲故事，让历史人物鲜活起来

### 3.2 写作技巧

1. **史料查阅**：开书前看对应朝代历史书（编年史如《资治通鉴》+ 纪传体如《二十四史》）
2. **清晰主线**：人事时地物，大目标拆分为小目标，主角时刻知道在干什么
3. **开篇简洁**：背景清晰，人物不超过3个，视角不随意切换
4. **人物出彩**：发掘小人物，突出人物弧光，通过反差写出立体感

### 3.3 注意事项

**规避历史虚无主义**：
- 不过分夸大主人公能力、不随意处置历史事件
- 不美化反面人物、不为反面人物翻案
- 不猎奇、不剑走偏锋、不片面追求阅读量
- 兼顾历史真实与艺术真实

**正确价值观**：
- 树立正确的历史价值观和使命感
- 无论主角是统治阶层还是小人物，本质都是对美好生活的向往

**红线**：严禁涉政、涉黄

## 四、悬疑推理类

### 4.1 题材定义与分类

悬疑小说的核心在"悬疑"两个字。"悬"是悬念，"疑"是怀疑，"悬疑"指的是因未知而产生的怀疑心理。悬疑小说是以解决悬念、探索未知作为主要剧情的小说类型，往往具备三大特点：曲折的情节、强烈的悬念和严谨的逻辑。

| 子类型 | 定义 | 代表作品 |
|--------|------|---------|
| 灵异风水 | 以神鬼事物为题材，出现鬼魂、异变生物等惊悚元素，包含朴素因果思想 | 《我有一座恐怖屋》《鬼吹灯》《盗墓笔记》《魔临》 |
| 推理探案 | 以严密逻辑推理解开谜题，含凶杀案/失踪/离奇事件 | 《心理罪》《法医秦明》《无证之罪》《长夜难明》 |
| 惊悚游戏 | 现实世界部分人类被选中参与死亡率极高的惊悚游戏，依靠聪明才智和金手指通关或反杀鬼怪 | 《十日终焉》《惊悚游戏：开局继承祖宗万亿冥币》《我纹身大哥吊打阴阳两界很合理吧》 |
| 规则怪谈 | 部分人类被选中参与高死亡率副本，重在勾勒逻辑严密的死亡规则，玩家通过推理规避假规则存活 | 《我在诡异世界谨慎求生》 |
| 诡异复苏 | 现实世界灵气和鬼怪同时复苏，人类与鬼怪两大阵营的战斗，主角拥有系统金手指碾压鬼怪 | 《全球诡异时代》《诡秘之主》 |
| 恐怖末世 | 鬼怪复苏过于猛烈导致人类阵营崩溃，主角重生归来或先知先觉提前储备物资，在恐怖末世中占尽先机 | 《末日蟑螂》 |

### 4.2 写作技巧

**灵异风水**：

背景设置——三种恐怖氛围营造手法：
1. **人称转换**：第三人称交代背景和来龙去脉，第一人称刻画恐怖气氛和人物内心以增强代入感
2. **信息不对称**：刻意制造人物与读者间的信息差，让紧张感升级。如《我有一座恐怖屋》中陈歌被剥夺视力，读者知道鬼魂位置但主角不知，读者替他捏汗
3. **反差对比**：以轻松惬意的表象反衬背后的恐怖。生活中随处可见的平常之处突然变得不平常，猝不及防间汗毛倒竖

人物设置：
- 主角事件前比较普通，因一件事改变人生轨迹（发现身世秘密/意外得到金手指/卷入诅咒）
- 生活背景可以普通，但性格必须随恐怖程度提升而有显著成长
- 示例：陈歌从被保护者蜕变为保护者

事件设置：
- 以离奇小事切入，环环相扣，事件不断升级
- 允许超自然力量，但基本逻辑必须遵循
- 不能把起因、发展和解决都推给超自然力量
- 故事真相应当与讲述同样精彩，不能"一切都是一场梦"

**推理探案**：

背景设置：
- 以凶杀案、诡异事件为主，开篇故事就要离奇
- 每层环环相扣，后续不断反转
- 案件可以是单元剧模式（如《福尔摩斯》《法医秦明》），主线在人物关系而非事件升级

人物设置：
- 主角需要高智商、敏锐观察力，但不能千篇一律的"冷静机智"
- 突出与其他推理主角不一样的地方（如福尔摩斯的高智商低情商，薄教授的清冷孤僻）
- 真凶/幕后黑手要有迷惑性，不能一出场就暴露
- 必要时制造烟雾弹，让读者在确定与不确定间反复摇摆

事件设置：
- 讲究关联性和逻辑性，不停发现新线索，前面的道具不能丢掉
- 最终解决要出人意料又在情理之中
- 不能变成偶发事件/意外事件收尾（虎头蛇尾）

**惊悚游戏**：

> 参考：番茄资深编辑青玄（带出《冒姓琅琊》《时停起手》等多部畅销作品）的惊悚游戏写作技巧分享。番茄爆款《十日终焉》也有惊悚游戏标签。

背景设置：
- 现实世界有部分人类被选中作为玩家参与死亡率极高的惊悚游戏（如青山精神病院、死亡学校等）
- 必须依靠自己的聪明才智和金手指达成通关或反杀鬼怪
- 在诡异元素中融合游戏副本设定，可拓展空间大，适合新作者入门也适合老作者突破

书名策略：
- **新人推荐**：使用《惊悚游戏：XXXX》格式，利用"惊悚游戏"四字的搜索流量，稳妥不出错。案例：《惊悚游戏：开局继承祖宗万亿冥币》《诡异游戏入侵，氪金十亿当鬼王》
- **进阶取名**：突出书的亮点，聚焦前二十章读者更容易看到的内容，多参考阅读榜取名模式。案例：《二十五岁，害怕鬼不够抗揍》《我纹身大哥吊打阴阳两界很合理吧》

副本设计（核心竞争力）：
- **场景选择**：选天然带有惊悚感的场景（废弃精神病院、荒郊野岭古怪村庄、闹鬼大楼等），可直接采用恐怖电影/游戏作为副本灵感。案例：《惊悚游戏：开局继承祖宗万亿冥币》的404号血色列车，《我纹身大哥吊打阴阳两界很合理吧》的裂口女
- **难度设定**：死亡率必须极高，设定有难度的推理副本和较低通关率，通过配角的失败死亡塑造惊悚氛围。人人轻松通关=游乐项目，失去惊悚感
- **智谋优先**：主角自身的智谋和推理比金手指直接平推更有个人魅力，更能吸引读者喜爱（参考《十日终焉》齐夏的冷静智慧）
- **多样化**：避免副本同质和单调重复，几十个副本世界必须各有特色，否则读者吸引力下滑
- **伏笔串联**：副本之间埋设隐秘联系，最终规划在一个完整世界观下，剧情更流畅自然，给读者更多惊喜。同时用积分系统/玩家排名/幕后黑手等主线串联

金手指设计：
- 金手指的创新和趣味性是核心竞争力，不同金手指带来不同剧情走向和人设风格
- **洞察型**：如"洞察之眼"，提前洞察诡异的杀人规则，规避甚至反向利用
- **反骨型**：不走解谜模式，违背游戏规则获得系统奖励，平推副本的快感
- **身份反转型**：如主角是黑白无常，诡异副本对别人是死亡副本，对主角是幼儿园难度
- **其他案例**：观察直播间观众厄运值收获怨念值变强、继承祖宗万亿冥币等

**规则怪谈**：

背景设置：
- 与惊悚游戏类似，但重在勾勒逻辑严密的死亡规则
- 提示给玩家的规则中有部分假规则和互相矛盾的规则，需要玩家推理规避

写作要点：
- 规则设计是核心：规则要有内在逻辑，假规则要有迷惑性，真假规则之间要有可推理的线索
- 爽点来源：推理解谜一步步揭开副本世界观，或利用金手指碾压副本
- 每个副本的规则体系要有差异化，避免"换皮"感
- 读者参与感很重要：给足线索让读者也能推理，揭晓时有"原来如此"的满足感

**诡异复苏**：

背景设置：
- 现实世界灵气和鬼怪同时复苏，人类在拥有灵气可以修炼的同时必须面临鬼怪威胁
- 本质是人类和鬼怪两个阵营的战斗

写作要点：
- 金手指的创新和趣味性是关键差异化手段
- 爽点来源：主角拥有系统金手指碾压鬼怪，爽感十足
- 世界观设计：灵气复苏的规则、鬼怪的等级体系、人类的修炼方式需要自洽
- 可融合都市元素：现代社会+超自然力量的反差感是天然看点
- 恐怖氛围与爽感要平衡：前期偏恐怖建立紧张感，中后期主角变强后偏爽

**恐怖末世**：

背景设置：
- 鬼怪复苏过于猛烈导致人类阵营崩溃，来到鬼怪统治的末日
- 主角重生归来或先知先觉，提前储备恐怖末世需要的物资（冥钞、古董、香火等）

写作要点：
- 与科幻末世的区别：恐怖末世的威胁是鬼怪而非丧尸/灾变，物资体系不同（冥钞/香火 vs 食物/武器）
- 爽点来源：主角占尽先机，在恐怖末世中逍遥自在，别人恐惧的鬼怪对主角构不成威胁
- 物资体系要有创意：冥钞、古董、香火等非常规末世物资是差异化看点
- 可融合灵异风水元素：风水布局、驱鬼术法等增加专业感

### 4.3 注意事项

- 不宣传封建迷信，如需出现应一笔带过并作为批判对象
- 部分陋习（重男轻女、吃绝户）可作批判对象，部分陋习（食用同类尸体、陪葬人殉）不能出现
- 女性向：不能出现冥婚、结阴亲、怀鬼胎等元素
- 男性向：不许出现冰恋元素
- 血腥场面尽量模糊化处理，不许出现屠城、毁灭世界等反人类场面
- 灵异类允许神鬼事物，但不能出现真实地名、政府机关及领导人名称
- 不允许以悬疑之名行色情之实

### 4.4 选题检查专项

- [ ] 悬念是否足够强烈，能驱动读者持续阅读？
- [ ] 灵异类：恐怖氛围是否运用了信息不对称或反差对比手法？
- [ ] 推理类：主角人设是否有独特性（不只是"冷静机智"）？
- [ ] 真凶/幕后黑手是否有足够迷惑性？
- [ ] 逻辑是否严谨，是否避免了"一切都是梦"式结局？
- [ ] 是否规避了平台内容红线（封建迷信/血腥/涉政）？
- [ ] 惊悚游戏/规则怪谈：金手指是否有创新和趣味性？
- [ ] 规则怪谈：规则设计是否有内在逻辑，假规则是否有迷惑性？
- [ ] 诡异复苏：世界观是否自洽（灵气规则/鬼怪等级/修炼方式）？
- [ ] 恐怖末世：物资体系是否有创意（冥钞/香火等非常规物资）？
- [ ] 恐怖末世：与科幻末世是否有明确差异化？

### 4.5 开篇检查专项

**第一章**：
- [ ] 灵异类：是否以离奇小事切入（而非直接大段恐怖描写）？
- [ ] 推理类：开篇案件是否足够离奇抓人？
- [ ] 是否运用了人称转换或信息不对称增强代入感？
- [ ] 惊悚游戏/规则怪谈：是否快速进入副本，展示规则的新奇感？
- [ ] 诡异复苏：是否展示了"日常世界突然变异"的反差感？
- [ ] 恐怖末世：是否展示了主角的信息优势或先知先觉？

**第二章**：
- [ ] 事件是否环环相扣、开始升级？
- [ ] 推理类：是否有新线索出现，推动读者继续阅读？
- [ ] 规则怪谈：是否开始展示规则的真假矛盾？
- [ ] 诡异复苏/恐怖末世：世界观是否通过角色遭遇展示（而非说明文）？

**第三章**：
- [ ] 灵异类：主角是否开始展现成长（从普通人到应对者）？
- [ ] 推理类：是否制造了烟雾弹，让读者在确定与不确定间摇摆？
- [ ] 惊悚游戏/规则怪谈：是否有利用金手指/推理的小高潮？
- [ ] 诡异复苏：主角是否展示了碾压鬼怪的爽感？

**常见开篇问题**：

| 问题 | 表现 | 修复 |
|------|------|------|
| 恐怖堆砌 | 开篇大段恐怖描写，缺乏节奏 | 以离奇小事切入，用反差对比制造恐怖 |
| 真凶暴露 | 反派一出场就能猜到是谁 | 制造烟雾弹，增加迷惑性角色 |
| 逻辑缺失 | 事件全靠超自然力量推动 | 保持基本逻辑，超自然只是背景 |
| 规则堆砌 | 规则怪谈开篇大段罗列规则 | 让主角在探索中逐步发现规则 |
| 副本同质化 | 惊悚游戏每个副本模式雷同 | 每个副本要有独特规则和恐怖元素 |

### 4.6 流水账检测专项

| 流水账模式 | 表现 | 修复方向 |
|-----------|------|---------|
| 恐怖走马灯 | 一个灵异事件接一个，缺乏情感连接 | 每个事件要有角色成长和情感变化 |
| 线索堆砌 | 大量线索罗列但缺乏推理过程 | 展示主角的推理思维链，让读者参与推理 |
| 案件流水线 | 单元剧模式下案件缺乏关联 | 用人物关系主线串联各案件 |
| 设定说明书 | 大段交代灵异规则/能力体系 | 通过角色遭遇逐步展示规则 |
| 副本流水线 | 惊悚游戏一个副本接一个，模式雷同 | 每个副本要有独特规则，用积分/排名主线串联 |
| 规则罗列 | 规则怪谈大段列出规则条目 | 让主角在探索中逐步发现和验证规则 |
| 打鬼流水线 | 诡异复苏"遇鬼→杀鬼→升级"无限循环 | 穿插人类阵营内部冲突和社会变化 |
| 囤物资流水账 | 恐怖末世"找物资→收物资"无限循环 | 每次行动要有人际冲突和恐怖遭遇穿插 |

### 4.7 角色检查专项

**灵异风水**：
- [ ] 主角是否有从普通人到应对者的成长弧光？
- [ ] 金手指是否替代了角色成长（主角只靠外挂解决问题）？
- [ ] 配角（鬼魂/队友）是否有独立人格？

**推理探案**：
- [ ] 主角人设是否有独特性（不只是"冷静机智"模板）？
- [ ] 真凶是否有迷惑性，不能一出场就暴露？
- [ ] 是否有烟雾弹角色让读者反复摇摆？
- [ ] 受害者是否有足够的人物刻画（让读者在意案件结果）？

**惊悚游戏/规则怪谈**：
- [ ] 主角是否有推理能力（不只靠金手指碾压）？
- [ ] 副本中的NPC/队友是否有独立人格？
- [ ] 金手指是否替代了角色的智慧和成长？

**诡异复苏**：
- [ ] 主角变强后是否仍有挑战（不是无脑碾压）？
- [ ] 人类阵营内部是否有利益冲突和人性博弈？
- [ ] 鬼怪是否有差异化设计（不是换皮怪物）？

**恐怖末世**：
- [ ] 主角的先知优势是否有合理来源和限制？
- [ ] 末世中的人际关系是否有深度（不只是主角独行）？
- [ ] 恐怖元素是否与末世生存有机结合？

### 4.8 质量评估专项

| 评估项 | 检查要点 | 常见问题 |
|--------|---------|---------|
| 悬念强度 | 悬念是否层层递进？ | 悬念单一或过早揭晓 |
| 逻辑严谨性 | 线索是否前后呼应？ | 逻辑漏洞、强行解释 |
| 恐怖/紧张感 | 是否运用了信息不对称/反差对比？ | 只靠血腥场面制造恐怖 |
| 人物弧光 | 主角是否有显著成长？ | 主角始终是工具人 |
| 反转质量 | 反转是否出人意料又在情理之中？ | 强行反转、虎头蛇尾 |

惊悚游戏/规则怪谈额外：副本差异化、规则设计逻辑性、读者参与推理的空间
诡异复苏额外：恐怖与爽感的平衡、人类阵营内部博弈、鬼怪等级体系
恐怖末世额外：物资体系创意、与科幻末世的差异化、恐怖元素与生存的结合

## 五、新媒体小说类

### 5.1 新媒体文通用规则

新媒体文即"适合在新媒体渠道推的文"，推广渠道包括头条、抖音、微信、微博、浏览器等。

**用户阅读流程**：推广素材 → 文案章节（前2-3章）→ 免费章节 → 充值付费 → 跟读复充

**理想标准**：
1. 开篇有噱头，方便制作吸睛推广素材
2. 开篇切入快、期待感强，迅速抓住读者
3. 免费章节（约3万字）情节紧凑、一环扣一环、情绪调动强
4. 计费卡点期待感强，充值率高
5. 计费章节期待感保持好，跟读复充率高

**开篇创作建议**：
- 整体设计：把开篇几万字当作整体设计，构思框架、填充素材、优化卡点
- 笔墨分配：修炼升级、背景设定、场景转换等能一句话讲清就不用两句话
- 开篇思路：制造爽感/期待感/情绪调动，无帮助的情节舍弃，帮助小的简写

### 5.2 男频子类型

#### 5.2.1 女婿文

底层逻辑——小人物：利用女婿身份的普遍性引发读者共鸣，用"不公平对待"制造情感认同。

| 要素 | 说明 |
|------|------|
| 身份设计 | 女婿（明面身份）+ 金手指身份（首富/战神/神医/仙帝/风水师） |
| 核心人物 | 女主家庭图像（丈母娘是关键角色，市井势利但不能退场） |
| 开篇核心 | 围绕女婿与某角色的人物关系反常化，制造噱头冲突点 |
| 情绪延续 | 围绕开篇情感情绪点书写，把某一情绪极致化，不突兀转场 |

代表作品：《超级上门女婿》《豪婿》《最佳女婿》《上门龙婿》

#### 5.2.2 战神文

传统兵王文的升级版，核心是"王者回归"。战神人设：睥睨天下、傲骨嶙峋、杀伐果断。

**塑造战神三要素**：
1. 行为影响力：战神的行为对社会产生巨大影响（回国战机护航、一怒国破族亡）
2. 配角衬托：桀骜兵王只是小跟班、名震神医听候差遣、权势贵族甘愿跪门
3. 言行设计：惜字如金、一言九鼎，语言带满满自信和装逼感

**丰富情节线**：
- 主角社会关系立体化：家人、朋友、下属各引出一条情节线
- 反派关系网络化：核心反派 + 家族/盟友/靠山，形成利益集团

代表作品：《都市之至尊战神》《天神殿》《都市巅峰战神》《战神奶爸》

#### 5.2.3 都市生活

对现实生活的艺术加工，普通人通过努力取得成功。

**创作三步法**：
1. 先选故事再定人称：故事比人称重要，核心是引起兴趣和共鸣
2. 用套路讲不套路的故事：把故事卖点用一个词/一句话总结，放到第一章展开（黄金单章）
3. 投入自己情感：不止爱情，还有亲情、友情、事业、理想

**关键提醒**：不要开篇拖沓铺垫，三五章不知道讲啥；后期不要陷入男女主反复离合的误区

#### 5.2.4 超短爽文（男频悬疑向）

> 来源：番茄作家课堂·爆款超短《高额电费暴露老婆第二个家》作者江行舟创作分享。

超短故事是番茄小说平台高热度品类，篇幅短、节奏快、情绪密度高，适合新媒体渠道推广。

**五大核心特点**：

| 特点 | 说明 |
|------|------|
| 代入感强 | 主角多为普通中年男性、职场边缘人、被辜负者，易引发共鸣 |
| 以小见大的冲突 | 将日常矛盾升级为戏剧性冲突，涉及权力、尊严、公平、背叛 |
| 身份反转与信息差 | 主角表面弱势，实则具备隐藏能力或反转空间，利用信息差制造悬疑 |
| 节奏快、钩子密 | 开篇即抛核心矛盾，短段落、强对话、场景化描写 |
| 结局爽快、闭环完整 | 体现"善恶有报""扬眉吐气"的正义感，完成情绪宣泄 |

**题材选择方法**：
1. 观察日常，挖掘情绪：从生活中常见但易被忽视的细节入手（账单异常、监控盲区、邻里对话），寻找能引发猜疑、冲突或反转的切入点
2. 结合热点，贴近市场：关注平台近期热门题材和情绪风向（"信息差""身份反转""隐藏真相"），将个人灵感与读者兴趣结合

**人物设计要点**：
- 反派/冲突方行为应有内在逻辑，甚至能让部分读者理解，增强张力和真实感
- 利用人物身份、认知与真相之间的"信息差"构建持续情绪牵引
- 主角的认知偏差本身就是悬疑引擎（如高社会地位者自动过滤"不可能"的嫌疑人）

**三层钩子架构**：

超短悬疑文的钩子设计由三层组成，从事件到情绪层层递进：

| 层级 | 功能 | 示例 |
|------|------|------|
| 主钩子 | 核心悬念，贯穿全文 | "情夫究竟是谁？" |
| 次钩子 | 过程悬念，维持好奇 | "二人如何避开监控完成偷情？" |
| 情绪钩子 | 读者情绪期待 | "男主何时发现真相？读者何时获得揭晓快感？" |

**剧情构造原则**：
- 每次主角接近真相时设置合理障碍或误导，保持"只差一步"的紧张感
- 通过配角反应或反派行为不断强化读者情绪（如旁观者的嘲笑暗示真相）
- 最终揭晓既在意料之外又在逻辑之中，完成情绪释放与故事闭环

**情绪营造三段式**：

| 阶段 | 读者情绪 | 写法 |
|------|---------|------|
| 前期 | 怀疑、疑虑 | 主角无证据猜疑，引发"是否误会"的悬念 |
| 中期 | 同情、愤怒 | 真面目逐渐显现，配角反应加剧处境 |
| 后期 | 爽感、宣泄 | 真相浮现，主角从被欺瞒到反击 |

**爆款案例参考**：
- 《不发月饼被网暴，我取消购物卡后他们崩溃了》——职场权力反转
- 《重生后，不再资助清高校花》——情感背叛+断舍离
- 《这铁疙瘩我能修，老板拿命和我赌》——隐藏实力+身份反转
- 《上秒缴费下秒搬走？我直接让你豪房变毛坯，房东傻眼了》——日常矛盾升级

### 5.3 女频子类型

#### 5.3.1 虐文

开篇讲述女主被男主从精神和身体上带来双重打击的现代言情作品。

**五大核心要素**：

| 要素 | 说明 |
|------|------|
| 情绪 | 虐文的灵魂。写好女主的心酸、无奈、渴望爱、空虚绝望等情绪，情绪塑造成功即使节奏稍慢读者也会读下去 |
| 人物关系 | 脉络主干。每个人物需要合理背景说服读者（如女主不离婚的原因必须合理） |
| 悬念 | 连接脉络的桥梁。随剧情展开不断抛出新悬念（女主何时死？男主何时发现真相？） |
| 视角 | 开篇以女主视角展开，通过女主眼睛感知情绪波动，快速带入 |
| 噱头 | 开篇直接切入矛盾或强冲突，狗血情节只要逻辑自洽即可 |

**常见开篇模式**：
- 女主癌症晚期 + 被男主误解 + 以身补偿
- 被前夫/前男友背叛 + 怀孕 + 带球跑
- 青梅竹马 + 家庭仇恨 + 相爱相杀

#### 5.3.2 现实婚恋文

以婆媳矛盾、夫妻矛盾为主元素，讲述女性在婚姻生活里遭遇不公。

**特点**：第一人称为主，"老公""婆婆"多是反面角色，与男主对手戏偏"甜"。

**创作三步法**：
1. 挖掘现实矛盾：婚内出轨、婆媳关系、生育问题、闺蜜变小三（关注微博热搜/知乎/头条）
2. 理顺情感逻辑：女主平凡但漂亮，想突破枷锁；男主成熟多金，给予安全感
3. 感情线之外的事业线：女主个人成长 或 寻求外人援助

**常见开篇模式**：
- 女主难产 + 婆媳关系差 + 情人怀孕
- 老公出轨 + 女主被害流产 + 男主拯救
- 家庭不和谐 + 女主自立自强

#### 5.3.3 超短爽文（女频年代向）

> 来源：番茄作家课堂·爆款超短《五零年代，我给外婆当虐渣系统》作者钮钴禄莹莹创作分享。

超短故事的女频年代向是当前高热品类，将年代背景与多元素叠加，通过时代差异制造天然矛盾，叠加女性觉醒主题实现情绪共鸣。

**四大核心特点**：

| 特点 | 说明 |
|------|------|
| 年代背景差异化 | 区别于现代都市，将故事置于上世纪60-90年代，利用新旧思想交替、城乡差距制造天然矛盾冲突与信息差 |
| 多元素融合（叠甲） | 在年代文地基上叠加"重生""系统""虐渣""女性成长"等热门元素，提升题材辨识度 |
| 女性觉醒与救赎 | 核心题材不局限于世情，延伸至女性成长、命运觉醒、亲情救赎、代际更迭 |
| 家国情怀升华 | 将个人命运（离婚、独立）与时代洪流（建设国家、妇女解放）结合，使爽感超越家长里短 |

**题材定位公式**：年代背景 + 年代重生/穿越 + 变身系统/金手指 + 虐渣 + 女性成长

**核心角色设计**：

| 角色类型 | 设计要点 | 示例 |
|---------|---------|------|
| 女主（觉醒者） | 人设弧光：从逆来顺受→逐步觉醒→发出自己的光芒 | 周秀兰：农村妇女，在引导下通过学习与抗争最终独立 |
| 引导者/系统 | 功能定位：带着前世遗憾的救赎者，引导女主成长，弥补亲情亏欠 | 外孙女变系统：视角新颖，代入感独特 |
| 渣男（反派） | 精致利己主义者的时代缩影，借时代之名行自私之实，最终自食恶果 | 陈建国：一切以个人利益为核心 |
| 配角团 | 各有功能定位：命运转折点、虐渣工具人、成长见证者 | 沈静姝教授、秦烈、小红 |

**反套路设定（创新点）**：
- 视角创新：区别于传统"穿越"或"自带系统"，让主角直接变成系统，代入感更独特
- 格局打开：不是单点逆袭，而是两代女性完整的觉醒光谱——经济独立→地理迁移→精神自立→实现社会价值，从小爱走向大爱
- 拒绝"火葬场"：渣男全程自作自受，无"追妻火葬场"情节，不在烂人身上浪费时间，渣男每次落魄对应女主一次向上攀登

**剧情设置四要素**：

| 要素 | 写法 | 效果 |
|------|------|------|
| 开局即高潮 | 开篇休书/背叛，迅速展现矛盾；借力打力（妇联、舆论）完成第一轮反击 | 快速建立阅读爽感 |
| 地图切换与升级 | 空间跃迁（农村→大院→首都），每阶段设置明确目标（离婚补偿→落户→学习→贡献国家） | 成长线清晰可见 |
| 双线交织叙事 | 明线：女主在新环境的奋斗与成长；暗线：渣男在老家的一地鸡毛 | 双线对比形成强烈反差，持续输送虐渣爽点 |
| 格局升华 | 中期将个人命运与国家发展挂钩；后期女主参与时代洪流（妇联骨干等） | 故事超越家长里短，拥有时代情怀 |

**爆款案例参考**：
- 《重回七五，我不再亏欠自己》——军营背景+重生+女性独立，开篇以临终遗言制造情绪冲击
- 《80年买楼送破烂？我撬开红木柜，全家当场吓瘫》——年代寻宝+悬念反转，日常细节引爆惊喜
- 《我在荒原等春来》——大西北军嫂+女性觉醒，以细节（大白兔奶糖、雪花膏味）制造情绪张力
- 《为国为人民决不为负心汉》——民国背景+重生+女性独立，将个人命运与家国情怀深度绑定

## 六、古代言情类

### 6.1 题材定义与分类

古代言情小说，即以古代背景展开的言情故事。作品可发生在真实历史背景、架空古代世界、仙侠世界或神话世界等。

| 子类型 | 定义 | 代表作品/特点 |
|--------|------|-------------|
| 女玄 | 女频玄幻/仙侠，修炼升级为主线，感情为辅线 | 东方玄幻、异世大陆、西方魔法 |
| 萌宝 | 萌宝推动男女主感情线发展 | 智商碾压型、有辨识度型、推动剧情型 |
| 宅斗 | 商贾/官宦后宅的嫡庶之争、地位之争 | 重生复仇、穿越逆袭 |
| 种田 | 古代背景下发家致富，从无到有的积累过程 | 农业/畜牧/商业/经营基建 |
| 脑洞 | 新颖创意设定，反常规人设 | 系统文、快穿、女主非人类 |

**背景分类**：穿越、架空、重生、历史、仙侠、远古、神话
**感情分类**：甜宠、虐恋、先虐后宠、先宠后虐

### 6.2 身份背景与基调确定

> 来源：番茄小说LV5作者鹿时七（代表作《老娘心理学博士，进宫嘎嘎乱杀》）

根据女主的不同身份背景，确定全文基调和爽点方向：

| 身份背景 | 金手指特点 | 爽点方向 | 写作要点 |
|---------|-----------|---------|---------|
| 重生 | 先知（前世记忆） | 信息差、将计就计、扮猪吃老虎 | 围绕先知金手指写，提前安排突发事件，主打信息差爽感 |
| 穿越（今穿古） | 现代知识带入古代 | 精神状态美好、不按套路出牌、帮助身边人走出困境 | 不仅限于宅斗，走出大宅院提高读者期待值 |
| 穿书 | 先知（原书剧情） | 改变原书角色命运、救下家人、逆转悲剧结局 | 与重生类似但多一个爽点：改变命运 |
| 无金手指原住民 | 无外挂，纯靠自身 | 扮猪吃老虎、运筹帷幄 | 性格塑造需要功底，"穿越女凭什么斗得过世家贵女？" |

**关键原则**：人物不同的背景决定行事风格和性格塑造方向，先确定身份背景再展开剧情。

### 6.3 通用写作技巧

**文笔和叙事风格**：
- 语言符合古代背景，但不过于文绉绉，通俗易懂
- 纯古代人物不能使用现代网络流行语
- 用事件推动故事发展，外物描写起辅助作用，不大段描写环境/服饰/外貌
- 叙述风格（轻松/欢脱/幽默/细腻）要与故事基调匹配

**世界观架构**：
- 仙侠玄幻：体系宏大，上限天花板高，修炼等级层层分级（金字塔结构）
- 古代架空：体现时代背景特色，种田重民俗、宅斗重嫡庶制度、宫斗重后妃等级

**金手指设定**（两大类）：
1. 信息不对等：女主提前预知剧情（重生/穿书/预知未来）
2. 能力不对等：拥有别人没有的东西和技能（空间/灵泉/透视/读心术/神医）

金手指要有限制，不能让女主无所不能。种田文的空间可以种出更好的瓜果，但不能直接产出仙丹。

**人物塑造**：
- 男主：符合女性幻想，人设讨喜实力强大，不能抢女主打脸风头，可设置强绑定关系
- 女主：人设有亮点，善恶分明不圣母，金手指与自身优势结合
- 正面配角：扶持主角成长，可承担副线CP，可有立场转变
- 反面配角：极端塑造拉仇恨值，前中期不让反派形象摇摆

**爽点设计**：
- 欲扬先抑：先营造不被看好，再奋起打脸
- 营造紧迫感：金手指在身处险境时获得最佳
- 极端人设：凡出场就让读者相信主角一定没事
- 发扬长处：比试围绕女主优势写

### 6.4 各子类型创作要点

**女玄文**：
- 世界观宏大，为后期换地图换大陆做准备
- 等级设定层层分级，技能体系丰富且画面感强
- 战斗场景用短句、拟声词、省略号增加画面感
- 设计阶段性任务目标串联成大目标
- 言情线不能忽略，男主要有存在感
- 友情、亲情线也是重要看点

**萌宝文**：
- 萌宝人设：智商碾压型 / 有辨识度型 / 推动剧情型
- 出场时间：开篇出场（节奏快）或中后期出场（拉期待值）
- 忌讳：萌宝讨厌女主、三观不正、失去幼儿感变成成年人思维

**种田文**：
- 满足感来自从无到有的积累过程，不是一夜暴富
- 男主选择：青梅竹马型（早出场）或天降型（可晚出场，多有隐藏身份）
- 地图线铺开：村子→镇上→更大地方，或种田+科举/宅斗杂糅
- 感情线不能喧宾夺主，金手指不能过于粗暴

**宅斗文**：
- 矛盾冲突根本是欲望对立（嫡庶之争/地位之争）
- 四大期待点：极品下场、男主出现、感情进展、跳出宅院
- 权谋四层：谋感情→谋地位→谋权利→谋家国
- 女主要有职业线（医术/厨艺/刺绣/经商）实现自我价值
- 避免为了斗而斗，节奏不能过快也不能过慢
- 风格多元：不必全程"斗"，可走欢脱温馨、家长里短、姐妹齐心发展事业等路线，把阴暗斗争安排给与主角关系密切的配角
- 金手指规则：只允许弱金手指（如医术空间、前世记忆），金手指只作辅助，女主主观利用时才出现，不能让金手指干预或绝对推动主线
- 五种爽点类型：①弱→强逆袭打脸（丑女变美女、人人可欺→立威占位）②跳出宅院实现自我价值（不再服小做低，发展事业线）③获得真挚感情（前世被渣男欺骗→今世遇到真爱）④上帝视角信息差（穿越/重生/穿书，知道所有人不知道的未来走向）⑤多重身份逆袭（不被看好的皇子→权力霸者）

**脑洞文**：
- 背景创新：传统题材+系统 或 传统题材+新颖背景（直播/游戏等）
- 形象创新：女主萌宝（团宠风）或女主非人类（猫猫/狐妖/人参精）
- 特殊身份必须在剧情中发挥作用，不能沦为背景板

### 6.5 常见误区

| 误区 | 表现 | 修复 |
|------|------|------|
| 书名太文艺 | 穿越医妃复仇文起名《浮生梦》 | 书名要展示故事亮点、元素、题材 |
| 语言太现代 | 古代人物说"爸爸妈妈"、网络流行语 | 符合时代背景的称呼和说话方式 |
| 辞藻堆砌 | 五六百字描写人物外貌服饰 | 外物描写辅助即可，不拖慢节奏 |
| 背景交代过长 | 穿越前/重生前内容描写过多 | 尽快推进后续情节 |
| 开篇人物过多 | 一下子出场大量人物 | 控制出场人数，突出主要情节 |
| 切入点无吸引力 | 开篇几千字交代重生前背景 | 以有亮点的情节切入 |
| 主线进展缓慢 | 十几章围绕一个小事件 | 一到两章制造一个爽点 |
| 人名不合时代 | 古代皇帝叫"王浩晨" | 人名符合古代背景，避免生僻字 |

### 6.6 选题检查专项

- [ ] 金手指是否有限制（不能让女主无所不能）？
- [ ] 宫斗宅斗文：金手指是否只作辅助（不能绝对推动主线，不能让女主依附金手指）？
- [ ] 女玄文：世界观是否足够宏大，等级体系是否完善？
- [ ] 种田文：满足感是否来自积累过程（而非一夜暴富）？
- [ ] 宅斗文：矛盾冲突是否基于欲望对立？
- [ ] 脑洞文：特殊设定是否在剧情中发挥了作用？
- [ ] 书名是否展示了故事亮点和题材（不过于文艺）？

### 6.7 开篇检查专项

**第一章**：
- [ ] 是否以有亮点的情节切入（而非大段背景交代）？
- [ ] 出场人物是否控制在合理数量？
- [ ] 语言风格是否符合古代背景？
- [ ] 穿越/重生文：是否尽快进入"后世"情节？

**第二章**：
- [ ] 是否有矛盾冲突点推动剧情？
- [ ] 女玄文：修炼体系是否开始展示？
- [ ] 宅斗文：嫡庶/妻妾矛盾是否初现？

**第三章**：
- [ ] 是否有至少一个爽点（打脸/逆袭/金手指展示）？
- [ ] 男主是否已出场或有出场暗示（言情主线类）？

**常见开篇问题**：

| 问题 | 表现 | 修复 |
|------|------|------|
| 背景灌输 | 开篇几章交代世界观/技能体系 | 通过情节展示设定 |
| 辞藻堆砌 | 大段外貌/服饰/环境描写 | 精简外物描写，聚焦情节 |
| 人物混乱 | 开篇出场人物过多 | 控制人数，突出主角 |
| 节奏拖沓 | 穿越前/重生前描写过多 | 尽快进入核心情节 |

### 6.8 流水账检测专项

| 流水账模式 | 表现 | 修复方向 |
|-----------|------|---------|
| 升级流水线 | 女玄文不停升级打怪无情感变化 | 每次升级要有情感成长和人际变化 |
| 种田日记 | "种菜→收获→卖钱"无限循环 | 每个阶段要有新挑战和人际冲突 |
| 宅斗走马灯 | 女配轮番出场被打脸，模式单一 | 矛盾冲突要有递进，从宅院到更大舞台 |
| 辞藻填充 | 大段环境/服饰/外貌描写凑字数 | 外物描写辅助即可，聚焦情节推进 |
| 设定百科 | 大段交代修炼体系/世界观/等级 | 通过角色遭遇逐步展示 |

### 6.9 角色检查专项

**女玄文**：
- [ ] 女主是否有废柴逆袭的成长弧光？
- [ ] 男主是否有存在感（不被遗忘）？
- [ ] 配角技能和等级是否设计好（避免脸谱化）？
- [ ] 友情/亲情线是否有打动人的地方？

**萌宝文**：
- [ ] 萌宝是否保持了幼儿感（不像成年人思维）？
- [ ] 萌宝是否推动了剧情发展？
- [ ] 萌宝人设是否正面（不是熊孩子）？

**种田文**：
- [ ] 感情线是否喧宾夺主？
- [ ] 男主选择是否与种田主线匹配？
- [ ] 金手指是否过于粗暴（直接产出而非辅助积累）？

**宅斗文**：
- [ ] 反派是否极端塑造拉足仇恨值？
- [ ] 女主是否有职业线/成长线？
- [ ] 权谋是否基于主线剧情（不为斗而斗）？
- [ ] 主角性格属性是否开篇立现（如病娇偏执/睿智冷静/无坚不摧），让读者一眼判定是否想看？
- [ ] 配角是否有反差感设计（如对外高冷对内温暖、外表乞丐实则高手），而非单面好人或坏人？
- [ ] 金手指是否只作辅助（女主主观利用时才出现，不干预主线）？

### 6.10 质量评估专项

| 评估项 | 检查要点 | 常见问题 |
|--------|---------|---------|
| 文风匹配度 | 语言是否符合古代背景？ | 现代化用语、网络流行语 |
| 金手指平衡 | 金手指是否有限制？ | 女主无所不能，缺乏挑战 |
| 爽点设计 | 是否有欲扬先抑的打脸节奏？ | 爽点单一或缺乏起伏 |
| 感情线平衡 | 感情线与主线是否平衡？ | 感情线喧宾夺主或男主无存在感 |
| 世界观完整度 | 体系是否完善且有侧重？ | 设定混乱或过于简单 |

### 6.11 毒点与过度检测专项

> 来源：番茄小说LV5作者鹿时七（代表作《老娘心理学博士，进宫嘎嘎乱杀》）
> 虽以古言为例，但轻毒点和五过度对所有网文题材均适用。

**五大轻毒点**（读者不适但可能忍耐）：

| 毒点 | 表现 | 后果 | 修复方向 |
|------|------|------|---------|
| 成长太快 | 没有过渡，主角突然变得很厉害 | 读者观感分裂，一头雾水 | 设置成长阶梯，每次提升有铺垫和代价 |
| 报应太慢 | 反派作恶后迟迟不受惩罚 | 读者堵气，耐心耗尽 | 有仇当场就报，除非大Boss需要埋伏笔后期填坑 |
| 人设偏离 | 前期高智后期恋爱脑，性格前后不搭 | 读者困惑，角色失去可信度 | 性格转变需要铺垫和触发点 |
| 单剧情太漫长 | 一个打脸小剧情拉了十几章 | 读者失去耐心 | 小剧情快速解决，大Boss才适合长线蚕食 |
| 换地图太频繁 | 频繁更换场景，不断出现新配角 | 读者对新人物产生抵触心理 | 控制换地图频率，让读者先与当前角色建立感情 |

**五大重毒点**（读者直接弃书）：

| 毒点 | 说明 |
|------|------|
| 太弱智 | 主角智商下线，明显的蠢行为推动剧情 |
| 太圣母 | 无底线原谅伤害自己的人，让读者受气 |
| 总被虐 | 主角持续受虐无反击，读者压抑到窒息 |
| 反三观 | 价值观扭曲，美化不当行为 |
| 拆CP | 破坏读者磕的CP，引发强烈反感 |

**五过度**（好的人设写过头变成毒点）：

| 过度类型 | 初衷 | 过度后果 | 把控要点 |
|---------|------|---------|---------|
| 过度调皮 | 活泼可爱、古灵精怪 | 变蠢货，失去甜变成累赘 | 调皮要有底线和分寸，不影响正事 |
| 过度深情 | 展现CP感 | 变油腻，读者反感 | 少说多做，用细微行动体现感情（如悄悄递糖） |
| 过度高冷 | 酷帅有距离感 | 变傲慢，伤害读者感情 | 高冷对外、温柔对特定人，有反差 |
| 过度深刻 | 传达思想 | 变说教，爹味输出 | 不强行输出观点，用剧情自然传达 |
| 过度善良 | 善良底色 | 变圣母，无底线原谅 | 善良但不软弱，该出手时就出手 |

**毒点自检清单**：

- [ ] 主角成长是否有合理过渡（不是突然变强）？
- [ ] 反派作恶后是否及时受到惩罚？
- [ ] 人设是否前后一致（转变有铺垫）？
- [ ] 单个小剧情是否控制在1-2章内解决？
- [ ] 换地图是否有必要且不过于频繁？
- [ ] 主角是否避免了弱智/圣母/总被虐的表现？
- [ ] 人设特质是否把控了度（不过度）？
- [ ] 感情戏是否"少说多做"（不油腻）？

## 七、现代言情类

### 7.1 题材定义与分类

现代言情小说，在现代背景（都市/年代）下展开，以女主为主视角。故事可以言情为主（爱恨情仇），也可以剧情为主（打脸虐渣），或以日常为主（温馨互动）。

| 子类型 | 定义 | 核心看点 |
|--------|------|----------|
| 总裁文 | 出身高贵的霸总与普通女主的故事 | 甜宠互动、追妻火葬场、虐渣打脸 |
| 萌宝文 | 文中有萌宝存在，分总裁延伸型和萌宝主角型 | 萌宝推动感情线、马甲掉落、团宠 |
| 年代文 | 重生/穿越回60-90年代 | 信息差致富、高考/改革开放风口、空间/系统 |
| 马甲文 | 主角身披多个马甲（黑客/首富/特工等） | 不断掉马打脸、身份反转 |
| 快穿文 | 系统辅助穿越不同世界完成任务 | 多世界短篇、节奏快、世界观多变 |
| 穿书文 | 穿越成原书炮灰女配/悲惨角色 | 预知剧情改命、逆转结局 |
| 打脸复仇文 | 重生/穿书后对前世仇人复仇 | 纯爽文、节奏流畅、持续打脸 |
| 团宠文 | 女主被多人极尽宠溺 | 哥哥疼父母爱、锦鲤属性、真千金回归 |
| 虐文 | 压抑苦情氛围，虐身虐心 | 情绪调动、追妻火葬场、带球跑 |
| 脑洞文 | 含创新设定（星际/末世/系统/非人类等） | 新奇感、想象力、跨类型融合 |
| 校园文 | 校园背景，学生时期故事 | 青春色彩、双向救赎、奋发向上 |

### 7.2 常用元素与多元组合

各子类型有常用元素点，跨类型组合可产生新亮点。

**总裁文**：一夜情、先婚后爱、别后重逢、替嫁、契约、萌宝带球跑、撩夫、娱乐圈
- 组合示例：一夜情+萌宝带球跑+别后重逢 → 经典萌宝文开篇
- 组合示例：替嫁+契约婚姻+先婚后爱 → 经典婚恋文开篇

**年代文**：系统、空间、家长里短、发家致富、学霸、锦鲤、种田、养娃
- 组合示例：空间+发家致富+种田+家长里短 → 女主依靠空间种田致富

**团宠文**：女主萌宝、哥哥舅舅爸爸大佬、福宝、美食
- 组合示例：锦鲤+女主萌宝+团宠+种田+发家致富 → 锦鲤福宝带全家致富

**女强爽文**：真假千金、马甲、重生、打脸复仇、异能、女战神、病娇偏执
- 组合示例：真假千金+马甲+重生+打脸复仇 → 真千金回归虐渣之路

**校园文**：学霸、学渣、校霸、校草、系统、金手指、双向救赎
- 组合示例：校霸+校草+双向救赎 → 少男少女互相拯救

**快穿/末世/星际/无限流/悬爱**：
- 快穿：穿越各大位面完成系统任务，每个世界篇幅短、节奏快
- 末世：重生末世前，囤物资、建基地、抵御丧尸
- 星际：穿越星际世界，靠异能/精神力/机甲一步步登顶
- 无限流：被迫进入重构世界/游戏空间，生死考验+人性审视
- 悬爱：发现杀人犯可能就在身边

### 7.3 写作技巧

**1. 在热门套路中找新切入点**

真假千金套路中有四个女性形象（真千金、假千金、生母、养母），视角和人设变化可演绎完全不同的故事：
- 以假千金为主人公 → 《蓝色生死恋》
- 以真假千金为双女主 → 《还珠格格》
- "假千金"为男性设定 → 《海豚湾恋人》

**核心方法**：换主人公、调整身份性格、改变视角、找新情节切入点。

**2. 经典故事在新时代背景下重新演绎**

- 契约结婚+先婚后爱 ← 《美女与野兽》（被迫同居→日久生情→发现真实身份）
- 虐文梗 ← 《小美人鱼》（牺牲自我→误会无法解开→悲剧结局）
- 团宠+逆袭打脸 ← 《白雪公主》（被迫害→被团宠→遇真爱→获新生）

**3. 感情线和剧情线双线并行**

| 常见剧情线 | 常见感情线 | 组合示例 |
|-----------|-----------|----------|
| 发家致富 | 先婚后爱 | 种田文+日久生情 |
| 打脸复仇 | 追妻火葬场 | 重生文+男主挽回 |
| 娱乐圈升级 | 隐婚 | 女明星+总裁丈夫双线发展 |

**4. 总结套路共性，推导新情节**

常见开篇套路的本质是"主人公命运发生重大转折"：
- 村里的女主进城 / 修行的女主下山 / 纯洁的女主一夜情
- 未婚的女主被迫结婚 / 死亡的女主获得重生 / 平凡的女主身份被揭开

→ 找到你故事中命运转折的节点，以此作为开篇起点。

**5. 设置悬念，铺垫情节，建立期待**

三步法：
1. 给读者细节和线索，设置悬念，建立期待目标
2. 铺垫情节，延迟满足，不断提高期待值
3. 满足预期，并建立新的期待目标

示例：
- 马甲文：读者从书名/简介/细节暗示得知女主是大神 → 期待掉马时刻 → 配角震惊反应
- 追妻火葬场：男主初期态度恶劣 → 读者搜寻男主爱女主的证据 → 期待男主意识到感情+挽回

**6. 脑洞文创新三板斧**

> 来源：番茄小说女频脑洞文创作指南

脑洞题材的创新可以从三个方向思考：

- **元素组合**：把多种元素融合到一篇作品里，实现爽点看点多样化，抓住不同读者的胃口
- **套路反转**：把已有的成熟套路反转改写，从另一个角度切准读者爽点。这种方式主线可以按套路文延展，降低新手写长的难度
- **金手指翻新**：不拘泥于常见的空间/商城/神豪，金手指的创新会很大程度创新作品的主线

**7. 脑洞文开篇：金手指出场**

金手指是脑洞文独有的配置，贯穿全文、牵引主线。开篇除了找好切入点、立人设、明确主线、设置悬念之外，还需要让金手指尽早出场，让读者知道这个故事的"脑洞"在哪里。

### 7.3a 快穿文专项

> 来源：番茄小说女频脑洞文创作指南

快穿题材一般是主角出于某种原因穿越到不同位面完成任务。位面可以五花八门涵盖各种题材和元素，女主快穿的原因构成整个作品的主线，保证紧凑性。每个位面5-8万字为主，节奏轻快爽点密集。

**按主线类型分类**：

| 主线类型 | 核心机制 | 看点 | 注意事项 |
|----------|----------|------|----------|
| 攻略型 | 任务是攻略男主或收集男主碎片 | 感情线刻画、女主"手段"和"心机"、以甜为主 | 需要擅长写感情线，详细刻画两人感情发展脉络 |
| 逆袭型 | 穿越成炮灰/女配/小可怜，大佬帮原主翻盘 | 大女主成长线、复仇打脸线、可写无CP | 部分成长线可用时间大法快速过进度 |
| 拯救型 | 穿越成边缘人物，带任务拯救特定人物（反派/男配） | 已知剧情的信息差、甜+爽双线 | 结合了攻略型和逆袭型特点，主角提前预知剧情 |
| 修罗场型 | 万人迷设定，每个位面引来男性角色雄竞 | 乙女游戏般的爽感、多人追求 | 女主可以被喜欢但不要主动暧昧多人，建议1v1，避免后宫风险 |
| 单一题材型 | 全位面单一背景（年代/末世/美食等） | 深挖单一题材多种写法，固定吸引对应读者 | 适合擅长某一题材的作者 |

**快穿 vs 无限流区分**：

| 维度 | 快穿 | 无限流 |
|------|------|--------|
| 世界观 | 不强调快穿世界的世界观，强调位面世界观和任务 | 有完整的世界观设置 |
| 参与人数 | 女主单人穿越 | 一群人做任务，有淘汰和组队机制 |
| 画风 | 以爽甜为主 | 偏灵异恐怖，有紧迫感和死亡可能 |

**快穿开篇注意**：
- 系统绑定切入不要话痨，直奔主线，最好以位面情节/场景开篇
- 位面背景要简洁，可随任务发展后面补充，不要一两章阐述背景导致节奏拖沓

### 7.3b 现言脑洞文专项

> 来源：番茄小说女频脑洞文创作指南

背景以现代、未来、星际、末世为主，内容包含系统、金手指、异能等非现实元素，主线推进和悬念塑造与脑洞元素强关联，画风创新生动有趣。

**热门子题材要点**：

| 子题材 | 核心看点 | 金手指方向 |
|--------|----------|------------|
| 女主萌宝文 | 萌宝可爱卖萌、养崽幸福感、虐渣逆袭成长 | 萌宝有金手指，团宠设定 |
| 末世文 | 末世生存、重建文明，可结合囤货/逃荒/基建/直播 | 空间囤货、异能傍身，与普通人制造反差 |
| 玄学文 | 降维打脸的爽感，构建玄学世界观 | 玄学作为金手指，可结合真假千金/豪门/娱乐圈/重生 |
| 星际文 | 星际背景下的幻想生活，开放性包容性强 | 可围绕基建/升级/美食/直播/种田/校园等多角度展开 |
| 娱乐圈脑洞 | 女主升级吸粉，升级路径清晰，适合新人 | 绑定金手指让女主更快吸粉，通过弹幕强化人设 |

**注意事项**：
- 脑洞可以天马行空，但人设和行为要符合正常逻辑，不要为了欢脱文风放弃人物逻辑
- 写爽点或虐点注意分寸，前期压爽点虐女主程度太过、或虐配角太过，都会降低主角逼格
- 萌宝文注意萌宝人设要符合小孩子天真可爱纯真，不要有太多成年人性格特点（邪恶/粗口/油腻/报复心过强）
- 创新的点可以冷门但不能邪门，要符合读者对作品的基本需求（代入感+爽感）
- 言情小说中男主基本要求是强大有魅力，奶狗人设也要在背后有实力，不是整天哭唧唧无所事事

### 7.3c 玄幻言情专项

> 来源：番茄小说女频脑洞文创作指南

世界背景有明显仙侠玄幻元素，剧情主线以女主升级线和感情线为主，以女主视角为主视角。

**两大方向**：

| 方向 | 特点 | 人设 | 注意事项 |
|------|------|------|----------|
| 传统打脸虐渣向 | 爽感强、爽点密集、节奏明快 | 女强设定，满级大佬穿成废柴，性格爽飒果决 | 避免爽点重复，通过新剧情和配角反差设定增加新颖度 |
| 创新人设轻松向 | 风格更轻松，人设反差强 | 结合现代网络潮流（内卷/摆烂/咸鱼），现代思维与古代规则碰撞 | 抓牢人设趣味点的同时配合明确升级主线，不要一味追求人设忽略主线 |

### 7.4 选题检查专项

- [ ] 题材元素组合是否有新意（不是单一套路）？
- [ ] 总裁文：开篇相遇方式是否有吸引力（一夜情/契约/替嫁等）？
- [ ] 萌宝文：萌宝是否有推动剧情的作用（不只是卖萌）？
- [ ] 年代文：金手指是否与时代风口结合（高考/改革开放/房地产）？
- [ ] 马甲文：马甲数量和掉马节奏是否设计好？
- [ ] 打脸复仇文：爽点是否持续递进（不是重复打脸）？
- [ ] 虐文：人物行为逻辑是否自洽（为什么不离开/不反抗）？
- [ ] 校园文：是否有积极向上的核心思想？
- [ ] 快穿文：主线类型是否明确（攻略/逆袭/拯救/修罗场/单一题材）？
- [ ] 快穿文：每个位面篇幅是否控制在5-8万字？
- [ ] 快穿文：修罗场型是否避免了后宫风险（建议1v1）？
- [ ] 脑洞文：金手指是否有创新（不是常见的空间/商城/神豪）？
- [ ] 脑洞文：创新是否符合读者基本需求（代入感+爽感），不邪门？
- [ ] 脑洞文：人设和行为是否符合正常逻辑（不为欢脱放弃逻辑）？
- [ ] 萌宝文：萌宝人设是否符合小孩子天真可爱（无成人性格特点）？
- [ ] 感情线和剧情线是否双线并行？
- [ ] 是否设置了悬念和期待目标？

### 7.5 开篇检查专项

**第一章**：
- [ ] 是否直接切入命运转折点（不拖沓铺垫）？
- [ ] 总裁文：男女主是否已相遇/绑定？
- [ ] 重生/穿书文：是否尽快进入"新生"情节？
- [ ] 马甲文：是否暗示了女主的隐藏身份？
- [ ] 虐文：是否以女主视角切入，情绪是否到位？
- [ ] 快穿文：是否以位面情节/场景开篇（而非系统绑定话痨）？
- [ ] 快穿文：位面背景是否简洁（不要一两章阐述背景）？
- [ ] 脑洞文：金手指是否在开篇尽早出场？

**第二章**：
- [ ] 冲突是否强化（紧迫目标+具体阻碍）？
- [ ] 人物关系是否初步建立？
- [ ] 是否有悬念驱动读者继续阅读？

**第三章**：
- [ ] 是否有至少一个爽点/情绪高潮？
- [ ] 马甲文：是否有第一次掉马暗示？
- [ ] 打脸文：是否有第一次小打脸？
- [ ] 结尾是否有钩子？

**常见开篇问题**：

| 问题 | 表现 | 修复 |
|------|------|------|
| 背景交代过长 | 重生前/穿越前大段描写 | 尽快进入核心情节 |
| 人物过多 | 开篇登场大量角色 | 控制在3-4人，逐步引入 |
| 套路感过强 | 开篇与同类作品高度雷同 | 换视角/换切入点/调整人设 |
| 情绪缺失 | 虐文开篇只有事件没有情绪 | 以女主视角写心理感受 |

### 7.6 流水账检测专项

| 流水账模式 | 表现 | 修复方向 |
|-----------|------|----------|
| 打脸流水线 | 反派轮番出场被打脸，模式单一 | 打脸要有递进和变化，反派关系网络化 |
| 掉马流水线 | 马甲文一个接一个掉马，缺乏情感连接 | 每次掉马要有情感变化和人际影响 |
| 甜宠日记 | 总裁文只有甜蜜互动，缺乏冲突 | 感情线需要剧情线支撑 |
| 种田/致富流水账 | 年代文"赚钱→升级"无限循环 | 每个阶段要有新挑战和人际冲突 |
| 情绪真空 | 虐文只有虐的事件，缺少情绪描写 | 虐文灵魂是情绪，每个事件都要有情绪反应 |
| 世界走马灯 | 快穿文世界切换缺乏连贯性 | 用主线（灵魂碎片/系统成长）串联各世界 |
| 系统话痨 | 快穿/脑洞文主角与系统大量无意义对话 | 系统绑定直奔主线，不要过多笔墨互动 |
| 脑洞背景板 | 金手指/脑洞设定只在开头出现，后续沦为背景 | 金手指要贯穿全文，持续推动主线 |

### 7.7 角色检查专项

**总裁文**：
- [ ] 男主是否有"霸总"之外的人格层次？
- [ ] 女主是否有独立人格（不只是被宠对象）？
- [ ] 配角（闺蜜/秘书/情敌）是否有独立功能？

**萌宝文**：
- [ ] 萌宝是否保持幼儿感（不像成年人思维）？
- [ ] 萌宝是否推动了剧情发展（不只是卖萌工具）？
- [ ] 萌宝人设是否正面（不是熊孩子）？

**马甲文**：
- [ ] 女主各马甲身份是否有差异化表现？
- [ ] 配角对女主的认知是否有层次（不是所有人同时掉马）？
- [ ] 掉马后配角反应是否有差异化？

**虐文**：
- [ ] 每个人物是否有合理背景说服读者？
- [ ] 女主行为逻辑是否自洽（为什么不离开）？
- [ ] 男主的"虐"是否有合理动机（不是无脑渣）？
- [ ] "白月光"角色是否有足够存在感？

**校园文**：
- [ ] 角色是否有青春期特有的情感表达？
- [ ] 双向救赎是否有具体的情感逻辑？
- [ ] 是否避免了成人化的处理方式？

**快穿文**：
- [ ] 攻略型：女主的"手段"和"心机"是否有层次感？
- [ ] 逆袭型：大女主成长线是否清晰？
- [ ] 修罗场型：女主是否避免了主动暧昧多人？
- [ ] 各位面男主/配角是否有差异化（不是换皮同一人）？

**脑洞文**：
- [ ] 萌宝人设是否保持幼儿感（不像成年人思维）？
- [ ] 男主是否强大有魅力（奶狗人设也要有实力底牌）？
- [ ] 反派/配角是否有基本智商（不为追求爽感而降智）？

### 7.8 质量评估专项

| 评估项 | 检查要点 | 常见问题 |
|--------|---------|----------|
| 元素组合 | 多元素是否有机融合？ | 元素堆砌、主次不分 |
| 悬念设计 | 是否有持续的期待目标？ | 悬念单一或过早揭晓 |
| 情绪调动 | 读者是否被情绪带入？ | 只有事件没有情绪 |
| 爽点递进 | 打脸/掉马是否有递进？ | 模式单一、后期疲软 |
| 双线平衡 | 感情线与剧情线是否平衡？ | 感情线喧宾夺主或剧情线薄弱 |
| 人设说服力 | 角色行为是否有合理背景？ | 行为逻辑不自洽 |

虐文额外：情绪饱满度、虐的节奏控制、追妻火葬场的期待值管理
马甲文额外：掉马节奏、身份反差感、配角反应差异化

## 八、玄幻类

### 8.1 题材定义与分类

玄幻文大致分为三类：玄幻仙侠、西方奇幻、武侠。

| 大类 | 背景 | 核心 |
|------|------|------|
| 玄幻仙侠 | 虚构东方背景 | 主角实力成长为主线，不断升级打败敌人 |
| 西方奇幻 | 虚构西方背景 | 以西方认知体系为基础，描述主角成长 |
| 武侠 | 古代中国 | 江湖恩怨情仇，家国天下 |

**仙侠文核心定义**：
- 特定升级体系：练气/筑基/金丹/元婴/化神等（源自《缥缈之旅》）
- 修行之人寻求长生，资源归属成为矛盾来源
- 修仙四艺：炼丹、制符、布阵、御兽
- 各种修行流派（剑修/体修/符修/丹修等）

**玄幻仙侠细分**：

| 流派 | 子类型 | 特点 |
|------|--------|------|
| 传统流 | 凡人流仙侠、古典仙侠、废柴逆袭流、强者重生流、日常风仙侠 | 成长平缓、金手指不超出故事限制、字数较长 |
| 脑洞流 | 洪荒流、签到系统流、全球神祇、迪化流、徒弟文、神宠文、女帝文 | 成长快速、金手指独立于故事之外、字数较短 |

### 8.2 传统玄幻仙侠写作技巧

**凡人流仙侠**（代表：《凡人修仙传》《仙逆》《凡骨》《宝塔仙缘》）：
- 设定：主角天赋/机缘相对一般（杂灵根/四五灵根），金手指偏辅助（加速灵药生长的空间、灵气提纯器皿等）
- 金手指示例：加快时间流速的小镜子、提炼丹药纯度的小鼎、推演完善功法的经书、复制灵药的罐子、废丹变灵丹的丹炉
- 剧情：风格写实，突出由弱变强的成长过程，人物心理和情感变化需细致推敲
- 人设：主角聪慧+果敢（聪明才能在残酷修仙世界生存，果敢才能抓住机遇），升级不要太快也不要太慢，配角不能脸谱化
- 节奏优化：省略凡人世界生活刻画，直接从奇遇情节切入→得到金手指→快进到修仙路线（"删减平淡部分，将有趣看点作为切入点"）
- 境界：凝气/筑基/金丹/元婴/化神等

> **仙侠与玄幻的核心区别**：玄幻文有降智反派来嘲讽主角、给主角垫脚装逼；仙侠文每个角色都智商在线，情节侧重点不在打脸，而在主角如何算计别人、如何苟发育、如何惩治恶人。

**古典仙侠**（代表：《剑来》《赤心巡天》《烟雨楼》《灭运图录》）：
- 核心：侧重于"侠"字，不光写仙人高深实力，也要体现心怀天下、大义凛然的人物形象
- 世界观：高武版的江湖——有市井小人也有文人墨客，有杀人夺宝也有门派纷争，有英雄气短也有儿女情长
- 语言：不能太白话，文风要有"古气"，语言本身成为卖点
- 体系：需要完善的自洽体系（经济贸易、门派架构、修炼、功法、法宝等），涉及道家佛教等宗教知识
- 细节：几百万字承载一个世界，需要丰满的细节增强代入感
- 人物：群像刻画是核心，人物是珠子、剧情是丝线，将人物写"活"才能留住读者
- 空间：一开始就设定好地图，过小难展开、过大超出控制
- 反派：不能太弱智，反派越强、主角胜之的爽感越强
- 金手指：相对偏弱，没有凡人流那么有存在感，主要靠主角主观能动性提升实力
- 节奏：可以不必快节奏，更多倾向于树立人设，起承转合需要更多设计

**废柴逆袭流**（代表：《斗破苍穹》《武动乾坤》）：
- 核心：读者最感兴趣的不是修炼过程，是修炼后带来的优越性
- 开篇：一两章内让主角获得金手指，不要让主角被欺凌十几章
- 打斗：多用象声词、短句、宏大夸张描写，展现反转变化
- 写长技巧：①丰富反派关系网 ②开辟新小地图（秘藏/传承）③做好人物群像 ④明确力量体系 ⑤塑造长线期待感（如"三年之约"）

**强者重生流**（代表：《万古神帝》《逆天邪神》）：
- 本质：由小到大、由弱到强重新站上巅峰
- 金手指：前世秘术/功法带到弱小时期，降维打击
- 注意：金手指容易开大，升级太快会后继无力、情节崩塌，一定要克制

**日常风仙侠**（代表：《我师兄实在太稳健了》《那些年，我们一起砍过的修仙者》）：
- 风格：轻松幽默，大量师门日常，没有苦大仇深的剧情
- 特点：近几年发力的仙侠新方向，读起来轻松愉快
- 人设技巧：利用强烈反差感增强立体感（我见犹怜的小师妹其实是力大如牛的盘古后裔、一向稳重的师傅其实是心机boy）
- 搞笑素材：关注漫画/影视/短视频热门段子（如《银魂》），人物身份/外表/个性的巨大反差、偶发事件集中爆发、小误会引发连锁反应
- 适合：喜欢轻松文风、不擅长写沉重剧情的作者
- 注意：日常风≠平淡无趣，要写生动有趣、温馨轻松、有生活气息的日常，仍需有主线推动

### 8.3 脑洞玄幻仙侠写作技巧

**洪荒流**（代表：《洪荒：开局给女娲剧透》《洪荒二郎传》《洪荒关系户》）：
- 基础：以《封神演义》《西游记》《山海经》《白蛇传》《蜀山传》为素材，以《佛本是道》为开山之作和框架（开启了网文对神话故事整合改编的全新篇章）
- 背景线：开天量劫→龙汉初劫→道魔之争→巫妖量劫→百教兴起→逐鹿之战→封神之战→西游之行（已形成体系，创作时尽量不要打乱）
- 核心人物：盘古三清、西方二释、地母一皇六位大道圣人，以大道（鸿钧）为最高存在
- 注意：①先搞清整体背景再动笔 ②写好大纲控制节奏（避免50万字写完所有背景）③神话人物耳熟能详可有效提高代入感

**签到/选择系统流**（代表：《开局签到荒古圣体》）：
- 特点：省略痛苦修炼过程，保留升级后的爽感
- 人设：无敌天骄流→言行霸气高冷；苟到无敌流→言行稳健，不主动惹事但斩草除根
- 期待感：通过即将获得的系统奖励塑造短线期待感
- 禁忌：不要让主角与系统大量对话拖慢节奏
- 写长：保证主角在当前局部范围内无敌即可，不是开局世界无敌

**万倍返还文**（代表：《我给万物加点，万倍返还》）：
- 核心机制：主角馈赠给别人东西，就能获得万倍返还的奖励
- 爽点来源：以小博大的极致反差——送出一颗丹药，返还万颗；赠予一本功法，返还神级秘术
- 写长技巧：①控制返还节奏，不要一开始就返还逆天物品 ②馈赠对象要有故事（不是随便找人送东西）③返还物品要推动剧情（获得新能力→解锁新地图/新冲突）
- 注意：返还机制容易让主角过快变强，需要设计合理的限制条件（冷却时间/馈赠条件/返还随机性）

**宗门建设文**（代表：《我，截教大师兄，加入聊天群》）：
- 核心机制：将落魄宗门进行种田经营，使宗门变强大，主角实力随之提升
- 本质：无敌文的变种——升级单位从个人变成宗门，爽感从个人碾压变成集体碾压
- 写长技巧：①宗门发展阶段化（破败→小有名气→区域强宗→天下第一）②招募弟子各有特色（不是批量NPC）③宗门之间的势力博弈提供持续冲突
- 爽点设计：宗门排名赛、弟子出山惊艳四方、其他宗门求合作/求庇护
- 可融合：种田文写法（经营积累的满足感）+ 收徒文（师徒互动）

**长生苟道流**（代表：《我师兄实在太稳健了》《万相之王》）：
- 核心人设：主角拥有无穷寿命或极长寿命，稳健修仙发育变强，人设很"苟"——不冒险、不出头、坐看世间沉浮
- 爽点来源：①苟了千年终于出手，一击必杀的反差 ②别人拼死争夺的资源，主角靠时间积累轻松获得 ③世人以为他是小人物，实则隐藏大佬
- 与签到流的区别：签到流靠系统被动变强，苟道流靠主角主动选择低调发育
- 写长技巧：①设计"不得不出手"的节点打破苟的状态 ②通过时间跨度展示世事变迁（宗门兴衰、故人老去）③苟的过程中穿插日常趣味（种田/炼丹/养宠）
- 注意：纯苟容易写成流水账，需要定期安排"被迫出手"的高潮事件

**迪化流**（代表：《原来我是绝世高人》《亏成首富从游戏开始》）：
- 核心：心理与现实的巨大反差（配角迪化主角 或 主角迪化配角）
- 文风：诙谐幽默，不建议涉及苦大仇深的复仇线
- 关键：人物关系循序渐进，制造巨大信息差，形成密集爽点

**徒弟文**（代表：《徒弟都是大魔头》）：
- 两种类型：徒弟未长成（养成向，借鉴种田文写法）/ 徒弟已长成（师徒碰撞为看点）
- 核心看点：主角师父和配角徒弟之间的对手戏

**神宠文**（代表：《宠魅》《从蚊子开始无限进化》）：
- 两种类型：重生为动物（全新视角）/ 驭兽养成（宠物进化为看点）
- 构思要点：①确定宠物在世界中的地位 ②设定野生/人类双地图 ③升级体系一一对应
- 宠物外貌变化是重要看点，要赋予外貌、性格等血肉感

**女帝文/仙女师傅文**（代表：《我开局震惊了女帝》）：
- 核心：女强男弱→男主变强→不再需要保护→可以保护女主（实力和地位转换）
- 女性角色：漂亮是基础，但要有各自辨识度和擅长领域
- 男主：需要表现出足够优秀的品质来匹配强大女主

### 8.4 西方奇幻写作技巧

代表作品：《诡秘之主》《放开那个女巫》《盘龙》

**四大体系**：
1. 神话体系：巨人、诸神、矮人、精灵（古希腊/北欧神话）
2. 骑士魔法体系：炼金术、召唤师、魔法师、游侠
3. 黑暗系：狼人、吸血鬼、亡灵、丧尸
4. 权力争霸体系：皇室霸权、阶级冲突

**构思要点**：
- 以中国人思路介绍西方世界观（"这个城市相当于华夏的魔都，但面积只有十分之一"）
- 强化西方元素的神秘感，利用陌生感为故事增色
- 人名、家族体系不要直接嫁接中国模式，但可适当融入东方元素增强代入感
- 设定要符合大众认知（龙贪婪、精灵姣好、狼人月圆变身），个体可违背但种族不能偏离

### 8.5 武侠写作技巧

代表作品：《雪中悍刀行》《昆仑》

**设定要点**：
- 空间不宜过大（凡人世界，空间过大人物显得空疏）
- 武力值中低武设置（太高容易写成玄幻，后期高手满地走）
- 境界提升不宜太快（避免战力值写崩）
- 正邪势不两立是核心驱动，加入庙堂谋算让故事更丰富

**写作技巧**：
1. 世界架构精巧路线（不同于玄幻的宏大路线），重笔墨在人物和剧情巧思
2. 人物必须复杂立体，正派中有大奸大恶、反派中有忠义之人（"反常"是推陈出新的途径）
3. 融合其他类型爽点（穿越/重生/系统/扮猪吃虎/权谋）
4. 将江湖之远与庙堂之高联系起来

### 8.6 通用写作技巧

**行文风格**：奇幻类符合西方特点，仙侠类符合东方特点，轻小说偏轻松搞笑

**开篇和书名**：
- 开篇切入简洁明了，先写剧情再插入设定
- 传统玄幻/仙侠：书名选用传统风格
- 脑洞玄幻/仙侠：书名与脑洞和开篇相关

**人物形象**：拒绝脸谱式主角，添加少量负面元素（好色/贪财/嘴贱）增加鲜明度

**体系设定**：
- 名称不能太复杂也不能太烂俗（如"斗灵/斗王/斗皇/斗帝"简单直白）
- 多个体系综合构成丰满世界（修炼+经济+门派+职业）
- 好的榜单设定有助于调动阅读期待（如"十大封号武帝"）

### 8.7 选题检查专项

- [ ] 是否明确了子类型（传统/脑洞/奇幻/武侠）？
- [ ] 金手指是否与子类型匹配（传统流偏辅助，脑洞流可开大但要控制节奏）？
- [ ] 力量体系是否简明清晰，有明确的境界划分？
- [ ] 世界架构是否与故事容量匹配（地图不过大也不过小）？
- [ ] 凡人流：升级是否足够缓慢写实？
- [ ] 废柴逆袭流：是否有长线期待感设计？
- [ ] 签到系统流：系统是否足够"简易"（不拖慢节奏）？
- [ ] 万倍返还文：返还机制是否有合理限制（冷却/条件/随机性）？
- [ ] 宗门建设文：宗门发展是否阶段化，弟子是否各有特色？
- [ ] 长生苟道流：是否有"被迫出手"的高潮节点打破苟的状态？
- [ ] 西方奇幻：设定是否符合大众对西方元素的认知？
- [ ] 武侠：武力值是否控制在中低武范围？

### 8.8 开篇检查专项

**第一章**：
- [ ] 是否简洁切入，避免大段世界观/设定说明？
- [ ] 废柴逆袭流：是否在一两章内让主角获得金手指？
- [ ] 强者重生流：是否展示了前世记忆带来的降维打击？
- [ ] 签到系统流：系统是否快速上线，不拖沓？
- [ ] 万倍返还文：返还机制是否在第一章展示？
- [ ] 宗门建设文：宗门的落魄现状是否快速交代？
- [ ] 长生苟道流：主角的"苟"人设是否在开篇立住？
- [ ] 开篇人物是否控制在合理数量？

**第二章**：
- [ ] 是否有明确的短期目标驱动剧情？
- [ ] 力量体系是否开始通过剧情展示？

**第三章**：
- [ ] 废柴逆袭流：是否有扮猪吃老虎的小高潮？
- [ ] 是否有升级/获宝后的爽感展示？
- [ ] 结尾是否有钩子？

**常见开篇问题**：

| 问题 | 表现 | 修复 |
|------|------|------|
| 设定先行 | 开篇大段交代力量体系/世界观 | 先写剧情再插入设定 |
| 废柴太久 | 主角被欺凌十几章无反抗 | 一两章内获得金手指 |
| 人物过多 | 开篇登场大量角色 | 控制人数，逐步引入 |
| 书名不匹配 | 传统仙侠用脑洞书名或反之 | 书名风格与内容类型一致 |

### 8.9 流水账检测专项

| 流水账模式 | 表现 | 修复方向 |
|-----------|------|----------|
| 升级机器 | 不停修炼升级，无情感变化 | 每次升级要有人物弧光和人际变化 |
| 打脸流水线 | 反派轮番出场被秒杀，模式单一 | 丰富反派关系网，打脸要有递进 |
| 设定百科 | 大段交代力量体系/境界/功法 | 通过角色遭遇逐步展示 |
| 系统对话 | 签到流主角与系统大量无意义对话 | 围绕升级爽感和优越感描写 |
| 副本流水线 | 一个秘境接一个秘境，缺乏情感连接 | 每个副本要有角色成长和情感变化 |
| 地图走马灯 | 不断换地图但缺乏深度 | 武侠走精巧路线，重人物和剧情巧思 |
| 返还流水线 | 万倍返还文"送东西→获返还"无限循环 | 馈赠对象要有故事，返还物品要推动剧情 |
| 经营日记 | 宗门建设文"招弟子→升级宗门"模式单一 | 穿插宗门间势力博弈和弟子个人故事 |
| 苟到无聊 | 长生苟道流纯苟无高潮 | 定期安排"被迫出手"的爆发事件 |

### 8.10 角色检查专项

**传统玄幻仙侠**：
- [ ] 凡人流：主角是否有低调杀伐果断的性格特点？
- [ ] 古典仙侠：是否有群像刻画（不是主角独角戏）？
- [ ] 废柴逆袭流：反派是否有关系网络（不是孤立出场被秒杀）？
- [ ] 强者重生流：金手指是否克制（不过度打脸导致后继无力）？
- [ ] 日常风仙侠：轻松幽默之余是否有主线推动？
- [ ] 主角是否有少量负面元素增加鲜明度？

**脑洞玄幻仙侠**：
- [ ] 签到流：主角言行是否符合人设（天骄流霸气/苟到流稳健）？
- [ ] 迪化流：信息差是否足够大，反差是否足够强？
- [ ] 徒弟文：师徒对手戏是否有看点？
- [ ] 神宠文：宠物是否有外貌、性格等血肉感？
- [ ] 女帝文：男主是否有足够优秀的品质匹配女主？
- [ ] 万倍返还文：馈赠对象是否有独立人格和故事（不是随机NPC）？
- [ ] 宗门建设文：弟子是否各有特色和成长曲线？
- [ ] 长生苟道流：主角的"苟"是否有一致的行为逻辑？

**西方奇幻**：
- [ ] 人名和家族体系是否符合西方特点（不是套了西方外皮的东方世界）？
- [ ] 不同种族是否有差异化行为逻辑？

**武侠**：
- [ ] 人物是否复杂立体（正派有奸恶、反派有忠义）？
- [ ] 反派智商是否在线，是否活得够久？
- [ ] 是否有人物群像刻画？

### 8.11 质量评估专项

| 评估项 | 检查要点 | 常见问题 |
|--------|---------|----------|
| 力量体系 | 境界划分是否简明清晰？ | 体系混乱或名称烂俗 |
| 升级节奏 | 升级速度是否与子类型匹配？ | 传统流太快/脑洞流太慢 |
| 爽感设计 | 爽点是否多样化且有递进？ | 单一打脸后期疲软 |
| 世界架构 | 多体系是否综合构成丰满世界？ | 只有修炼体系，世界单薄 |
| 人物立体度 | 主角是否有鲜明个性？配角是否有独立人格？ | 脸谱化、工具人 |
| 长线期待感 | 是否有贯穿全文的期待目标？ | 缺乏长线悬念 |

传统仙侠额外：文风古气、体系自洽、细节丰满度
武侠额外：江湖与庙堂的联动、人物反差、剧情巧思

## 九、都市类

> 注：新媒体都市文（女婿/战神/都市生活/虐文/婚恋文）的详细内容见第五板块「新媒体小说类」，本板块重点补充脑洞都市和传统无线都市的创作技巧。

### 9.1 题材定义与分类

都市文主要讲述现代背景下的故事，在读者熟悉的职业身份和人物关系中提供代入感，满足特定领域的现实欲望。

| 大类 | 常见题材 | 核心特点 |
|------|----------|----------|
| 新媒体向 | 战神、女婿、都市生活 | 超强节奏把控、情绪调动、章节卡点 |
| 脑洞向 | 系统、神豪、直播、娱乐圈、灵气复苏/高武 | 猎奇心理、快节奏、密集爽点、多反转 |
| 传统无线向 | 神医、奶爸、都市修真、鉴宝、重生年代 | 故事框架大、伏笔多、人物关系网丰富 |

### 9.2 脑洞都市写作技巧

**系统文**：

系统文四大原则：
1. 系统越简单越好：条件=多个奖励，远优于多个条件=奖励
2. 系统要有成长性：定时"更新"，解锁更强奖励和神秘功能，保持期待感
3. 系统为剧情服务：获得系统前的困境→如何运用制造爽点→新奖励对后续的铺垫→未获得奖励的期待感
4. 系统可以有意志但不能喧宾夺主：主角需要独立的讨喜性格，不能成为系统的陪衬

系统类型速查：
| 系统类型 | 机制 | 适合搭配 |
|----------|------|----------|
| 签到系统 | 打卡获奖励 | 无敌文、苟到无敌 |
| 选择系统 | 选项获奖励 | 脑洞文、搞笑文 |
| 抽奖系统 | 随机获奖励 | 猎奇文 |
| 任务系统 | 完成任务获奖励 | 各类型通用 |
| 神豪系统 | 给钱给车给房 | 神豪文 |
| 文娱系统 | 给才艺天赋 | 娱乐圈文 |

创作方向：①从剧情入手选系统 ②结合社会热点 ③从主角特点入手 ④从文风入手（搞笑→怨念/感谢系统）⑤从新颖性入手

**神豪文**（代表：《西虹市首富》模式）：

核心要点：
1. 设定简单、创意有趣：一句话就能描述清楚的金手指
2. 有金钱输出的必要：系统任务或生活困境催动花钱，不断一掷千金才能打脸
3. 贫穷和暴富的巨大反差：人物态度前后反差制造爽点
4. 女主不为五斗米折腰：有独特魅力和价值，不能用金钱衡量

写长技巧：
- 主角要有梦想（买球队/建商业帝国/做慈善），靠钱能解决一部分但不能全部解决
- 丰富支线剧情（感情线/事业线/家庭线），支线可跳转为主线
- 把控节奏，不要轻易引入修仙/高科技体系

常见痛点：
| 痛点 | 表现 | 解决 |
|------|------|------|
| 剧情套路化 | 开局被分手/被催房租等千篇一律 | 增加故事合理性（人物塑造）+ 结合新元素 |
| 后期流失 | 暴富后无事可做 | 主角有梦想+丰富支线+节奏把控 |

**灵气复苏/高武文**：

世界背景分类：全民修仙/修真、末世废土、赛博朋克、神祗降临

创作要点：
1. 世界背景和修炼体系开书前完善（避免后期写崩）
2. 写长：世界背景+地图构造支撑主线支线，修炼体系给主角成长空间
3. 与众不同：完善世界架构 + 融合多类型元素（神宠/神豪/战神/奶爸）

**娱乐圈文**：

创作要点：
1. 故事背景架空，避免映射现实
2. 金手指：纯穿越（前世娱乐知识）或带系统（娱乐系统，功能逐步开启）
3. 开篇切入：拉仇恨制造迫切需求，或直接拉出娱乐知识产生爽点的期待感

### 9.3 传统无线都市写作技巧

**重生年代文**：

核心三要素：
- 金手指：信息优势（不是智商/武力/技能优势）
- 目标：改变命运，弥补遗憾
- 方法：复制别人的成功

注意事项：
- 主角并非全能，只有信息优势（否则前世为何没成功？）
- 利用信息优势时充分铺垫，让读者认知到"换作我也会这样干"
- 开篇是无敌流写法，信息相对全面，不要设置障碍
- 男主人设：胸有丘壑、谋定而后动、成熟稳重

重生阶段选择：
| 阶段 | 看点 | 代表作 |
|------|------|--------|
| 儿童时代 | 做神童，"别人家的孩子" | 《重生之先声夺人》 |
| 高中时代 | 当才子，考名校，撩暗恋对象 | 《重生之财源滚滚》 |
| 大学及以后 | 赚钱创业 | 《重生完美时代》 |
| 结婚成人后 | 感情生活+事业有成 | 《重生：回到1983当富翁》 |

**都市修真文**：

两大流派：
| 流派 | 特点 | 爽感来源 |
|------|------|----------|
| 传统都市修真 | 节奏快、主线支线绵密 | 修炼升级后打脸反派 |
| 脑洞日常向 | 创意切入、文风轻松 | 信息差、修真能力在现代社会的反差 |

创作要点：
1. 先拉故事框架，修炼体系前期完善
2. 人设决定思维→思维决定行为→行为贯穿故事线
3. 传统流开篇：好创意+紧迫感（重生后要被枪毙/老婆被欺辱）
4. 脑洞流开篇：大创意有新意（高考后上修仙大学/狐仙老婆）
5. 提高粘性：日常向多玩梗+时效热点；传统向多用无线文套路+有计划换地图

**神医文**：

两种类型：
| 类型 | 特点 | 爽点 |
|------|------|------|
| 都市神医 | 医术金手指，扮猪吃虎 | 治病→获报酬/结识大佬/积累人脉→打脸 |
| 职业医生 | 文风严谨，注重专业性 | 职业发展跌宕起伏，行业内幕 |

可融合元素：奶爸、女婿、战神、修仙、乡村、种田（单一题材难写长）

**奶爸文**：

核心：萌宝养成 + 事业线双线并行
- 萌宝要涵盖所有宝宝优点（粉雕玉琢、乖巧懂事），让读者想捏脸
- 不能写着写着把萌宝写没了
- 可融合：文娱、修仙、战神、乡村直播

**鉴宝文**：

核心：猎奇捡漏心态
- 需要一定专业知识（玉石类型、专业词汇、价格）
- 经典套路：行业禁忌（打破禁忌制造悬念）、捡漏（以小博大）、竞拍（金钱碾压）、行业恩怨（世家/派系）

### 9.4 通用注意事项

**书名**：
- 脑洞文：包含吸引元素（"十连抽""SSS""被曝光""签到xx年"）或反差格式（《xx后，xxxxx》《开局xx，我xxx》）
- 传统/新媒体文：简单大气即可（《天神殿》《镇国战神》）
- 禁忌：不要强行蹭与内容无关的热点元素

**开篇切入**：
- 两个关键：开篇小创意（与市场大众区分）+ 情绪代入（愤怒/懵逼/屈辱）
- 女婿文：换个创意（如直接签离婚协议，而非给老婆洗脚）
- 重生文：换个创意（如醒来在火葬场要火化，而非醉酒醒来）

**人物取名**：
- 战神文男主：霸气（"许君临""宁轩辕""萧天策"）
- 乡村文男主：接地气（"王铁柱""王二狗"）
- 女主：符合人设（外冷内柔→"唐若雪"）
- 富豪配角：一眼看出有钱（"李万豪""张富贵"）
- 可用谐音/反差制造搞笑效果

### 9.5 选题检查专项

- [ ] 是否明确了子类型（新媒体/脑洞/传统无线）？
- [ ] 系统文：系统是否足够简单（一句话能描述清楚）？
- [ ] 系统文：系统是否有成长性（可定时更新）？
- [ ] 神豪文：是否有金钱输出的必要性（不只是有钱）？
- [ ] 灵气复苏：世界背景和修炼体系是否开书前完善？
- [ ] 娱乐圈：故事背景是否架空（避免映射现实）？
- [ ] 重生年代：金手指是否限定为信息优势？
- [ ] 都市修真：修炼体系是否前期完善？
- [ ] 神医文：是否有可融合元素支撑写长？
- [ ] 奶爸文：萌宝养成和事业线是否双线并行？
- [ ] 书名是否与子类型风格匹配？

### 9.6 开篇检查专项

**第一章**：
- [ ] 是否有开篇小创意（与市场大众区分）？
- [ ] 是否有情绪代入（愤怒/懵逼/屈辱）？
- [ ] 系统文：系统是否快速上线？
- [ ] 神豪文：贫穷与暴富的反差是否拉满？
- [ ] 重生年代：是否尽快进入重生后情节？
- [ ] 都市修真传统流：是否有紧迫感的切入场景？
- [ ] 都市修真脑洞流：大创意是否有新意？
- [ ] 人物是否控制在合理数量？

**第二章**：
- [ ] 是否围绕开篇情绪点延续（不突兀转场）？
- [ ] 是否有明确的短期目标驱动剧情？

**第三章**：
- [ ] 是否有至少一个爽点/打脸/反转？
- [ ] 结尾是否有钩子？

**常见开篇问题**：

| 问题 | 表现 | 修复 |
|------|------|------|
| 切入点雷同 | 女婿文都是洗脚开篇 | 换个创意与市场区分 |
| 背景交代过长 | 穿越/重生前大段描写 | 尽快进入核心情节 |
| 人物过多 | 开篇登场大量角色 | 控制人数，逐步引入 |
| 描写繁复 | 无关角色大段外貌描写 | 重要角色多笔墨，无关角色精简 |
| 视角混乱 | 开篇多视角频繁切换 | 新手用单一视角更易驾驭 |

### 9.7 流水账检测专项

| 流水账模式 | 表现 | 修复方向 |
|-----------|------|----------|
| 系统对话 | 主角与系统大量无意义对话 | 系统越简单越好，围绕爽感描写 |
| 花钱流水账 | 神豪文只有买买买 | 主角要有梦想，花钱要有目的 |
| 打脸流水线 | 反派轮番出场被秒杀 | 丰富反派关系网，打脸要有递进 |
| 治病流水线 | 神医文一个病人接一个 | 治病要有情感连接和人际影响 |
| 萌宝消失 | 奶爸文写着写着只剩事业线 | 萌宝养成和事业线双线并行 |
| 捡漏流水线 | 鉴宝文一个宝贝接一个 | 每次捡漏要有行业恩怨和人际冲突 |
| 情绪断裂 | 突兀转场改变冲突点和情绪点 | 围绕开篇情绪点延续，极致化 |

### 9.8 角色检查专项

**系统文**：
- [ ] 主角是否有独立的讨喜性格（不是系统的陪衬）？
- [ ] 系统是否喧宾夺主？

**神豪文**：
- [ ] 女主是否有独特魅力（不能用金钱衡量）？
- [ ] 反派态度前后反差是否合理（有人物塑造铺垫）？
- [ ] 主角暴富后是否有梦想和追求？

**重生年代文**：
- [ ] 主角是否只有信息优势（不是全能）？
- [ ] 男主人设是否成熟稳重（不是毛躁少年）？

**都市修真文**：
- [ ] 人设是否成功（决定思维→行为→故事线）？
- [ ] 日常向：是否多玩梗、接地气？

**神医文**：
- [ ] 医疗知识是否逻辑自洽？
- [ ] 是否有融合元素支撑写长？

**奶爸文**：
- [ ] 萌宝性格是否立体有趣味性？
- [ ] 萌宝是否贯穿全文（不被遗忘）？

### 9.9 质量评估专项

| 评估项 | 检查要点 | 常见问题 |
|--------|---------|----------|
| 代入感 | 读者是否能代入主角身份？ | 身份设定脱离现实 |
| 爽感设计 | 爽点是否多样化且有递进？ | 单一打脸/花钱后期疲软 |
| 情绪调动 | 是否有效调动读者情绪？ | 只有事件没有情绪 |
| 系统设计 | 系统是否简单有趣有成长性？ | 系统复杂/无成长/喧宾夺主 |
| 写长能力 | 是否有支撑写长的结构设计？ | 开局即巅峰，后期无事可写 |
| 人物取名 | 名字是否符合人设和题材？ | 战神文用接地气名字 |

神豪文额外：花钱目的性、女主独立价值、节奏把控（不引入修仙）
都市修真额外：修炼体系完善度、人设一致性、梗的时效性

## 十、职业文类

> 来源：番茄小说官方创作科普（女频职业文创作指南）

### 10.1 题材定义与分类

以职场为背景，聚焦某一行业并围绕主角在职场中的成长展开，展现某一职业或行业的独特魅力为主要内容的作品。

**核心特征**：
1. 以主角职业成长为故事主线，围绕职业技能提升+职场晋升展开，有清晰的晋升体系和职业规划
2. 通过不同场景、事件、人物，展现主角所在行业的特性特征及独特魅力

| 构思方向 | 定义 | 代表行业 | 对标作品 |
|----------|------|----------|----------|
| 现实向职业文 | 偏现实主义，接地气，职场主线明显，展现社会热点话题 | 公司职员、教师、医生、护士、厨师、投资人、猎头、销售、中介、导游 | 《杜拉拉升职记》《苏筱的战争》《心术》《外科风云》《安家》《开学》 |
| 新兴/猎奇类 | 新兴职业的趣味性，或具有神秘色彩、强烈吸引力的行业 | 殡葬类、临终关怀、盲盒设计师、汉服设计师、改娃师、收纳整理师、宠物殡葬师、考古类 | 《第七种生命》《人生大事》《三悦有了新工作》《风沙将我吹向你》《陷地之城》 |
| 言情向职业文 | 事业线与情感线相辅相成，男女主因工作交集促成感情发展 | 科学家、律师、设计师、飞行员、运动员、法医、翻译官、航天设计师 | 《冰糖炖雪梨》《炽道》《女士的法则》《精英律师》《你是我的荣耀》 |
| 古代职业文 | 古代背景下的职业故事，部分职业传承至今 | 女医、仵作、县令、御厨、绣娘、媒婆、状师、赊刀人、赶尸人 | 创作手法与现代职业文大同小异 |

**新兴职业速查**：
- 潮玩经济：潮玩设计师、盲盒设计、汉服设计师、簪娘、改娃师、洛丽塔服装设计师
- 懒人经济：收纳整理师、上门理疗师、跑腿、陪玩
- 宠物经济：宠物殡葬师、宠物美容师、宠物训导师、宠物入殓师、宠物烘焙师
- 银发经济：陪诊师、养老规划师、老年人能力评估师
- 小众稀缺：碳排放管理员、节能评估师、育婴师、心理咨询师、茶艺师、滑雪指导员

### 10.2 写作技巧

**选定行业**：
1. 通过自身生活环境，选取常见行业
2. 通过热门影视剧、出版畅销书，汇总讨论度高的职业
3. 通过社交平台（抖音/论坛/知乎/微博）发现新兴类、猎奇类行业

**收集材料**（三个方向）：
- 行业历史：通过纪录片了解行业发展历史和进程
- 行业现状+日常：通过行业剧或日常观察，了解话题与焦点
- 行业专业知识：通过社科工具书了解专业内容；通过抖音搜索"职业"关键词获取科普视频

**定故事框架主线**：
1. 探案刑侦类 → 单元剧结构+悬疑基调（如《法证先锋》《御赐小仵作》）
2. 女性职场类 → 主角晋升为主线，克服困难成长为行业精英（如《杜拉拉升职记》）
3. 行业类 → 聚焦行业，通过职业技能故事展现行业魅力，推进感情线（如《心术》《外科风云》）

**细化主角人设**：
- 职业文主角人设必须细化处理，不能泛泛使用概括统称
- 错误示例：主角是"医生"
- 正确示例：胸外科主任医师，医学博士，副教授，硕士生导师，擅长诊疗肺癌、肺部结节、气胸、食管癌等
- 剧情围绕职业技能、职场环境来写，注重主角职业上的个人成长
- 注意不要写成披着职业皮的言情文

### 10.3 注意事项

- 职业文的核心是"职业"，剧情必须围绕职业技能和职场环境展开
- 言情向职业文中，感情线要与事业线相辅相成，不能喧宾夺主
- 新兴/猎奇类职业自带新奇感，但仍需有扎实的行业知识支撑
- 古代职业文可参考现代对应职业（镖局→物流、讼师→律师、媒婆→婚介、牙行→中介/猎头、仵作→法医）

### 10.4 选题检查专项

- [ ] 是否选定了具体行业（不是泛泛的"职场"）？
- [ ] 主角职业人设是否细化（不是笼统的"医生""律师"）？
- [ ] 是否有清晰的职场晋升体系和职业规划？
- [ ] 剧情是否围绕职业技能和职场环境展开（不是披着职业皮的言情文）？
- [ ] 行业知识是否经过基本调研（纪录片/工具书/行业人士科普）？
- [ ] 言情向：事业线与感情线是否相辅相成？
- [ ] 新兴/猎奇类：行业新奇感是否有扎实知识支撑？
- [ ] 古代职业文：职业设定是否符合时代背景？

### 10.5 开篇检查专项

**第一章**：
- [ ] 是否通过具体的职业场景切入（而非大段背景交代）？
- [ ] 主角的职业身份是否在第一章明确展示？
- [ ] 是否有职场冲突或行业特色事件抓住读者？
- [ ] 探案刑侦类：开篇案件是否足够离奇？
- [ ] 女性职场类：是否展示了主角面临的职场困境？

**第二章**：
- [ ] 职场主线是否明确推进（晋升目标/职业挑战）？
- [ ] 是否通过具体事件展示行业特色？
- [ ] 言情向：男女主是否因工作产生交集？

**第三章**：
- [ ] 是否有职业技能展示的小高潮？
- [ ] 主角的专业能力是否得到初步体现？
- [ ] 结尾是否有钩子（职场危机/新挑战/感情悬念）？

**常见开篇问题**：

| 问题 | 表现 | 修复 |
|------|------|------|
| 职业背景板 | 职业只是标签，剧情与行业无关 | 围绕职业技能和职场环境展开核心剧情 |
| 行业术语堆砌 | 开篇大段专业术语 | 通过具体事件和场景自然带出行业知识 |
| 人设泛化 | 主角只是"医生""律师"等笼统称呼 | 细化到具体科室/方向/擅长领域 |
| 言情喧宾夺主 | 职业文变成纯言情 | 事业线为主，感情线为辅或相辅相成 |

### 10.6 流水账检测专项

| 流水账模式 | 表现 | 修复方向 |
|-----------|------|----------|
| 工作日志 | "上班→处理事务→下班"无限循环 | 每个工作场景要有冲突和成长 |
| 案件流水线 | 探案类一个案件接一个，缺乏情感连接 | 用人物关系主线串联各案件 |
| 晋升流水线 | "完成任务→升职→完成任务→升职" | 每次晋升要有人际冲突和内心成长 |
| 行业说明书 | 大段交代行业知识和专业术语 | 通过角色遭遇和具体事件展示行业特色 |
| 感情线插入 | 职场剧情中突兀插入大段感情戏 | 感情发展要与职场事件自然交织 |

### 10.7 角色检查专项

**现实向职业文**：
- [ ] 主角职业人设是否细化到具体方向/擅长领域？
- [ ] 同事/上司/下属是否有独立人格（不只是背景板）？

## 十一、热门标签：代嫁代娶

> 来源：番茄小说官方作家课堂（番茄女频编辑小然，2026-02-11，阅读量2.0w）
> 适用对象：偏女频，男频可参考

### 11.1 标签定义与核心看点

代嫁代娶是长篇小说中极具冲突的热门设定，核心围绕**替身成婚**展开，通过非原定配偶的错位结合，天然制造矛盾冲突、情感张力和剧情反转，适配甜宠、爽文、权谋等多种长篇创作风格。

**核心看点**：替嫁、换嫁、错认对象

- 代嫁代娶，是原定成婚者无法履行婚约，由他人替代其完成婚礼、进入婚姻关系的设定，核心关键词是**替身、错位、被迫/主动换嫁**
- 代嫁和代娶可单独出现（仅代嫁/仅代娶），也可同时出现（双方互换成婚），后者冲突感更强，更适合进行剧情延展
- 替身与原定者、替身与配偶之间的**身份差、情感差**，会形成相应对照组，是该设定的核心看点

### 11.2 常见背景设定

| 背景类型 | 说明 | 适配方向 |
|----------|------|----------|
| 家族利益 | 原定者因身体（重病/残疾/容貌受损）、意愿（不愿嫁娶/已有心上人）、意外（失踪/被绑架/突发恶疾）无法联姻，家族安排替身 | 权谋、宅斗、爽文 |
| 婚约绑定 | 双方家族早有不可违背的婚约，原定者无法履约，找他人完成仪式 | 古言、甜宠 |
| 重生换嫁 | 女配/女主带前世记忆重生，女配主动放弃原婚约选择不同对象，女主顺势而为重新活出人生 | 重生、打脸、逆袭 |
| 认错对象 | 主角因信息偏差、外界误导、主观预判失误，将两位身份/特征相似的人物混淆，认错相亲/订婚/结婚对象 | 甜宠、误会、反转 |

### 11.3 人物设定模板

**主角人设（代嫁/代娶者）**：

| 类型 | 动机 | 示例 |
|------|------|------|
| 被迫型 | 被家族胁迫（庶女被嫡母逼迫，为保护家人而代嫁） | 身不由己，初期隐忍 |
| 主动复仇型 | 家族被原定者家族陷害，替身主动代嫁/代娶伺机复仇 | 目标明确，暗中布局 |
| 主动追爱型 | 暗恋原定者的配偶，主动替代成婚 | 情感驱动，甜宠向 |
| 顺势而为型 | 重生后主动配合换嫁，规避前世悲剧 | 清醒通透，逆袭向 |

**配角人设（原定成婚者）**：

| 类型 | 行为模式 | 剧情作用 |
|------|----------|----------|
| 主动逃避型 | 不愿嫁给传闻残暴/体弱/有隐疾的配偶，寻找替代者 | 前期看笑话，后期嫉妒反扑，成为打脸对象 |
| 意外缺席型 | 因失踪/被绑架无法履行婚约，回归后性情大变 | 憎恨替身替代自己的人生，争抢制造冲突 |

**配偶设定**：

- 初期知晓婚约是联姻，对原定者毫无兴趣
- 后发现替身的与众不同，逐渐被吸引，暗中试探、默默守护
- 后期独宠替身，为替身扫清一切障碍
- 配偶**自带强势身份**（权势、财富），为替身的成长和剧情推进提供支撑

### 11.4 写作技巧（爆款三步法）

以《主母爱谁当！纨绔夫君为我挣诰命》为例：

**第一步：定背景**

以世家联姻为大背景，紧扣代嫁核心。背景设定要做到：
- 符合联姻为先、颜面至上的逻辑
- 为代嫁后的逆袭主线埋下伏笔
- 让代嫁不再是单独事件，而是串联各方利益、推动剧情的核心线索

**第二步：定人设**

围绕核心设定，立住**反差感鲜明**的人设，所有人设均服务于代嫁设定与主线推进：
- 女主（代嫁者）：聪慧通透、冷静果决，从"拒绝主母身份"到"主动携手夫君"
- 男主（纨绔夫君）：表面纨绔不羁，实则重情重义，核心目标"为女主挣诰命"，形成"纨绔变良人"的反差爽感
- 核心配角（庶妹）：心机深沉、嫉妒心强，成为女主打脸爽点的核心对象

**第三步：定主线**

以代嫁为基础，串联多条支线，爽点密集且逻辑连贯：
- 核心主线：契约夫妻 → 双向动心 → 彼此扶持 → 双向奔赴
- 内宅线：打理侯府、化解刁难、打脸庶妹与势利家族
- 朝堂线：男主逆袭、化解党争危机、为挣诰命拼搏
- 成长线：女主从被动代嫁到位居人上，男主从纨绔子弟到沉稳权臣

### 11.5 爆款案例简介

**《姐妹随军认错夫：面凶军官沦陷了》**（认错对象型）：
> 顾昭宁和养女苏静微同时议亲裴家兄弟。苏静微偷换相亲对象选了裴家长子，顾昭宁和面凶军官裴羡野大眼瞪小眼。后发现洞错房，裴羡野拒绝离婚，苏静微那边反而传来离婚消息。

**《主母爱谁当！纨绔夫君为我挣诰命》**（重生换嫁型）：
> 沈棠宁重生回庶妹换亲日，将错就错嫁给纨绔败家子。纨绔夫君为她悬梁刺股、发愤图强，封侯拜相那天夫人终于赏脸笑了。

**《主母日常》**（重生换嫁型）：
> 陆令筠和嫡妹双双重生。嫡妹抢了她的潜力股文官次子，把侯府世子让给她。陆令筠淡然一笑——没有她的谋划，那文官最多六品官的实力。这一世就做更风光的侯府主母。

### 11.6 注意事项

- 代嫁代娶的核心是**错位和冲突**，背景必须为代嫁的发生提供足够支撑，避免逻辑漏洞
- 赋予替身、配偶、原定者明确的性格、动机和成长线，避免工具人设定
- 背景和人物定好后，通过制造剧情冲突和反转定出主线，同时刻画情感线
- 代嫁和代娶可同时出现（双方互换成婚），冲突感更强，更适合剧情延展

### 11.7 选题检查专项

- [ ] 代嫁/代娶的触发原因是否合理（家族利益/婚约绑定/重生换嫁/认错对象）？
- [ ] 替身与原定者之间是否有明确的**身份差和情感差**形成对照组？
- [ ] 配偶设定是否自带强势身份（权势/财富），能支撑剧情推进？
- [ ] 是否有清晰的逆袭/成长主线（不只是换个人结婚）？
- [ ] 原定者回归/反扑的剧情是否有合理铺垫？
- [ ] 重生换嫁类：前世悲剧是否交代清楚，换嫁动机是否充分？

### 11.8 开篇检查专项

**第一章**：
- [ ] 代嫁/代娶的核心设定是否在第一章明确展示？
- [ ] 替身的处境是否引发读者同情或好奇？
- [ ] 是否有身份错位带来的即时冲突（新婚夜/初见/认错）？
- [ ] 是否避免了大段交代家族背景（用剧情带出）？

**第二章**：
- [ ] 替身与配偶的关系是否有推进（从陌生到试探）？
- [ ] 是否有来自原定者或家族的外部压力？
- [ ] 替身的目标是否明确（独善其身/复仇/追爱）？

**第三章**：
- [ ] 是否有小爽点（替身展示能力/配偶态度转变/原定者吃瘪）？
- [ ] 替身与原定者的对照是否开始显现？
- [ ] 结尾是否有钩子（新危机/身份暴露风险/感情萌芽）？

**常见开篇问题**：

| 问题 | 表现 | 修复 |
|------|------|------|
| 代嫁动机薄弱 | 读者不理解为什么要代嫁 | 用具体事件（家族危机/被迫/重生记忆）强化动机 |
| 配偶形象模糊 | 配偶只是"权贵"标签 | 第一章就展示配偶的独特性格和反差 |
| 原定者缺席 | 前三章完全没有原定者的存在感 | 至少通过侧面描写或回忆建立对照 |
| 冲突启动慢 | 代嫁后平淡度日 | 新婚夜/初见就制造身份错位的即时冲突 |

### 11.9 流水账检测专项

| 流水账模式 | 表现 | 修复方向 |
|-----------|------|----------|
| 宅斗流水线 | "处理内宅事务→化解刁难→下一个刁难"无限循环 | 每次宅斗要推进主线或感情线，不能只是打怪升级 |
| 感情线拖沓 | 男女主反复"误会→解开→新误会" | 每次情感推进要有实质性变化，不能原地踏步 |
| 打脸模板化 | 原定者/家族反复使坏，女主反复打脸 | 打脸要升级（从小事到大事），每次打脸推动格局变化 |
| 身份揭秘拖延 | 替身身份迟迟不暴露，剧情原地转圈 | 设置明确的身份暴露节点，用悬念推动而非拖延 |
| 配偶宠溺堆砌 | 大段描写配偶如何宠替身，缺乏剧情推进 | 宠溺要服务于剧情（帮替身解决困境），不能只是撒糖 |

### 11.10 角色检查专项

**替身（主角）**：
- [ ] 是否有独立人格和明确目标（不只是被动接受代嫁）？
- [ ] 性格是否前后一致（聪慧型不能突然降智）？
- [ ] 是否有成长弧光（从被动到主动/从隐忍到反击）？

**配偶**：
- [ ] 是否有反差感（表面vs真实）？
- [ ] 对替身的态度转变是否有合理过程（不能一见钟情式突变）？
- [ ] 是否有独立的事业线/成长线（不只是宠妻工具人）？

**原定者**：
- [ ] 是否有合理动机（不是无脑使坏）？
- [ ] 反扑是否有升级（不能每次都是同一套路）？
- [ ] 最终结局是否与其行为匹配？

**家族长辈**：
- [ ] 是否有独立立场（不只是推动代嫁的工具）？
- [ ] 势利/通透等性格是否一以贯之？

### 11.11 质量评估专项

| 评估项 | 检查要点 | 常见问题 |
|--------|---------|----------|
| 代嫁合理性 | 代嫁的触发原因和过程是否逻辑自洽？ | 代嫁动机牵强、过程草率 |
| 对照组张力 | 替身与原定者的对比是否持续制造张力？ | 对照组形同虚设，缺乏互动 |
| 反差爽感 | 配偶/替身的反差是否足够鲜明？ | 反差只停留在标签层面 |

| 爽点密度 | 打脸/逆转/甜宠爽点是否分布合理？ | 爽点集中在前期，后期疲软 |

## 十二、改编风向指南

> 来源：番茄小说官方作家课堂（2026-02-14，阅读量3.8w）
> 核心主题：什么样的作品更易被改编为短剧/漫剧？选择什么题材？怎么写才吸睛？
> 适用对象：所有品类

### 12.1 短剧改编风向

#### 12.1.1 女频短剧

女频短剧改编以**情感向和脑洞向**题材为主流。

**风格要求**：
- 强爽点，强情感拉扯
- 剧情节奏快，开篇有明确矛盾张力
- 剧情设定抓人眼球

**情感向推荐题材**：

| 题材设定 | 核心看点 |
|----------|----------|
| 先婚后爱 | 婚后互动张力，从契约到真情 |
| 年下 | 年龄差带来的反差萌和禁忌感 |
| 青梅竹马 | 从小到大的情感积累和错过重逢 |
| 甜宠互动 | 高糖高甜，满足情感幻想 |
| 情感修罗场 | 多角关系制造的情感张力和选择困境 |

**脑洞向创作要点**：
- 经典人设创新表达（高手下山、闺蜜同穿等经典设定做创新）
- 经典爆款元素创新延伸
- 多元素题材融合：仙侠+情感拉扯+打脸虐渣+悬疑，女性成长+复仇+情感拉扯
- **避开套路化设定模式，多加入新意和创新点**

**爆款案例**：

| 作品 | 设定 | 成功要素 |
|------|------|----------|
| 《先婚后爱，就她吧》 | 先婚后爱 | 经典设定+人设和情感拉扯上的微创新，婚后互动张力十足 |
| 《闺蜜齐穿书，你离，我隔一天！》 | 闺蜜同穿+豪门联姻 | 闺蜜同穿斗豪门的核心设定，剧情人设趣味性和感情拉扯看点十足 |

#### 12.1.2 男频短剧

男频短剧偏爱**快打脸、高爽感**的内容。

**风格要求**：
- 剧情环环相扣，开篇矛盾清晰，节奏紧凑
- 即时打脸、情绪反馈足
- 升级逆袭，收获感强

**推荐题材**：年代文、历史文、战神文

**推荐元素设定**：

| 元素 | 说明 |
|------|------|
| 接地气的狗血噱头开篇 | 快速建立矛盾，吸引眼球 |
| 婚姻关系中不对等的反抗 | 情感共鸣+爽感 |
| 家庭亲情辜负后的补救 | 亲情线拉情绪 |
| 事业线的升级 | 持续的成长爽感 |
| 金钱上积累的爽感 | 物质逆袭的直观满足 |

**经典设定需结合创新**：

| 融合方式 | 示例 |
|----------|------|
| 战神文+替嫁梗 | 军事实力+情感错位 |
| 年代重生致富+水产养殖 | 重生金手指+具体行业 |
| 穿越历史+双穿/群穿 | 多视角+信息差 |
| 种田+隐藏大佬 | 低调发展+身份反转 |

**注意事项**：避免同质化，避免过于复杂设定，信息和设定交代清晰，节奏爽快更利于改编。

**爆款案例**：

| 作品 | 设定 | 成功要素 |
|------|------|----------|
| 《重生拒绝扶弟魔，被前妻妹妹倒追》 | 年代重生+拒绝扶弟魔 | 接地气的狗血噱头+婚姻不对等反抗+年代致富爽感 |

### 12.2 漫剧改编风向

#### 12.2.1 女频漫剧

**内容方向**：逆袭打脸、聚焦家庭、夫妻关系等情感爱情矛盾。

**风格要求**：
- 强情绪、快节奏
- 主配角智商在线
- 打脸动机清晰、代入感强
- 开篇即切入核心冲突

**推荐题材**：

| 题材 | 核心看点 |
|------|----------|
| 古言种田 | 从零开始的逆袭+生活感 |
| 现言后悔流 | 渣男/渣女追悔莫及的情绪爽感 |
| 古言宅斗 | 家族内部权力博弈 |
| 重生逆袭 | 带记忆重来的降维打击 |
| 真假千金 | 身份错位+打脸 |
| 嫡女复仇 | 权谋+情感+爽感 |
| 假死重生 | 消失后归来的反转冲击 |
| 豪门千金回归 | 身份揭秘+打脸 |
| 渣男追妻火葬场 | 后悔流的极致表达 |

**创作建议**：
- 从家庭、邻里、夫妻等人际关系中挖掘矛盾，重点抓情绪流
- 亲情向和爱情向结合，情绪更浓烈，代入感更强
- 主角人设独立，智商在线

**爆款案例**：

| 作品 | 设定 | 成功要素 |

#### 12.2.2 男频漫剧

男频漫剧以**脑洞文**为主，设定简单易懂，即时反馈感强。

**风格要求**：
- 快节奏+高密度冲突
- 强爽感+反差感
- 开篇一定要爽快，立即切入核心看点
- 偏新媒体题材更容易被青睐

**推荐题材元素**：玄幻、末世、系统、神话、都市逆袭、年代重生、历史权谋、无限流、历史脑洞、鉴宝

**作品设定要点**：
- **金手指**：尽量简单直接，避免复杂的规则解释
- **多元素融合**：无限流+深化，玄幻+进化系统
- **核心公式**：快节奏爽感+强情感共鸣

**爆款案例**：

| 作品 | 设定 | 成功要素 |
|------|------|----------|
| 《斩仙台上何人？灵台方寸关门弟子》 | 神话IP+人生编辑系统 | ①传统神话IP大胆改编 ②人物形象自带烟火气，代入感极强 ③"背景通天"+"顶级大佬降维打击"的热门爽点 |

### 12.3 改编潜力自检清单

**通用检查**：
- [ ] 开篇是否在前300字内建立核心矛盾？
- [ ] 节奏是否紧凑，是否避免了大段背景交代？
- [ ] 金手指/核心设定是否一句话能说清？
- [ ] 是否有即时反馈的爽感（打脸/逆袭/甜宠）？
- [ ] 是否在经典设定基础上有创新点？

**女频专项**：
- [ ] 情感拉扯是否足够强烈？
- [ ] 主角人设是否独立、智商在线？
- [ ] 是否有清晰的打脸动机和对象？
- [ ] 多元素融合是否自然（不生硬拼凑）？

**男频专项**：
- [ ] 打脸是否即时、爽快？
- [ ] 升级逆袭的收获感是否持续？
- [ ] 设定是否简单直接（避免复杂规则解释）？
- [ ] 情绪反馈是否充足（不只是事件罗列）？

### 12.4 改编友好度评估维度

| 维度 | 权重 | 高分标准 | 低分表现 |
|------|------|----------|----------|
| 开篇冲击力 | 25% | 前300字有爆点，矛盾清晰 | 大段背景交代，冲突启动慢 |
| 节奏紧凑度 | 25% | 环环相扣，无冗余 | 拖沓、信息堆砌、原地踏步 |

| 设定辨识度 | 15% | 经典设定+创新点，一句话能概括 | 套路化、同质化、设定复杂 |
| 情感共鸣度 | 15% | 角色有血有肉，读者能代入 | 角色扁平、情感苍白 |
