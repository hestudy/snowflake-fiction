# 场景规划模板

本文档提供场景清单和场景规划的详细模板，用于步骤 8 和步骤 9。

---

## 场景类型说明

### 主动场景 (Active Scene)

主角主动追求目标，遇到冲突，遭受挫折。

```
结构：目标 → 冲突 → 挫折
```

**特点**：
- 主角采取行动
- 有明确的想要达成的目标
- 遇到阻碍（人、环境、内心）
- 结果是**不完全成功**或**付出代价的成功**

**适用场景**：
- 对抗
- 追逐
- 辩论
- 任务执行
- 调查

---

### 被动场景 (Passive Scene)

主角对挫折做出反应，内心挣扎，做出决定。

```
结构：反应 → 困难 → 决定
```

**特点**：
- 触发于上一个主动场景的挫折
- 主角处于情绪低落/困惑状态
- 内心进行价值权衡
- 最终做出新决定，开启下一个主动场景

**适用场景**：
- 反思
- 咨询/对话
- 决策
- 情感处理

## 场景节奏规划

### 短篇小说节奏模板

```
第一幕（约占20%）
├── 场景1：日常世界（主动）── 展示主角现状
├── 场景2：激励事件（主动）── 打破平衡
└── 场景3：犹豫/拒绝（被动）── 主角抗拒

第二幕上（约占25%）
├── 场景4：跨过门槛（主动）── 主角决定行动
├── 场景5：盟友/敌人（主动）── 扩展世界
├── 场景6：考验（主动）── 第一次挑战
└── 场景7：中点转折（主动）── 重大失败/成功

第二幕下（约占25%）
├── 场景8：低谷（被动）── 最黑暗时刻
├── 场景9：绝地反击准备（主动）── 重整旗鼓
└── 场景10：最后考验（主动）── 最终挑战

第三幕（约占30%）
├── 场景11：高潮（主动）── 终极对决
├── 场景12：结局（主动/被动）── 后果展现
└── 场景13：新平衡（被动）── 展示变化
```

### 节奏控制原则

1. **主动-被动交替**：避免连续多个同类型场景
2. **冲突强度递增**：后面的冲突比前面更强
3. **情绪高低起伏**：紧张后要有缓冲
4. **信息逐步释放**：不要一次性给太多

## 场景冲突类型参考

| 冲突类型 | 描述 | 示例 |
|----------|------|------|
| 人vs人 | 主角与他人的对抗 | 辩论、战斗、竞争 |
| 人vs自我 | 内心的挣扎 | 道德困境、欲望vs原则 |
| 人vs自然 | 与环境的抗争 | 暴风雨、荒野求生 |
| 人vs社会 | 与制度的对抗 | 反抗不公、打破规则 |
| 人vs命运 | 与宿命的抗争 | 预言、诅咒、命运 |
| 人vs科技 | 与机器/AI的对抗 | 黑客、AI失控 |

---

## 使用建议

1. **先列出所有场景清单**（步骤8）
2. **再逐个填写详细规划**（步骤9）
3. **每写完一章，检查场景是否偏离规划**
4. **灵活调整，但保持核心冲突不变**
