# 雪花写作法 - 每步提示词模板

本文档包含雪花写作法每个步骤的标准化提示词模板，供 Claude 在执行时参考。

---

## 步骤 0：初始化输出目录

在开始创作之前，需要确定输出目录。

### 初始化流程

```
1. 确定输出基础目录：
   - 如果用户指定了路径，使用用户指定的路径
   - 否则，使用当前工作目录下的 `novel-output/` 目录

2. 根据小说名创建子目录：
   - 使用小说标题作为目录名
   - 对标题进行安全处理（移除特殊字符，空格替换为下划线）

3. 创建目录结构：
   mkdir -p [输出目录]/[小说名]/{人物卡片,人物宝典,正文}
```

### 用户询问模板

```
在开始创作之前，我需要确认输出位置：

当前工作目录是：[当前目录]

小说将保存到：[当前目录]/novel-output/[小说名]/

如果需要更改输出位置，请告诉我新的路径。
否则，我们开始第一步！
```

### 目录结构

```
[输出目录]/
└── novel-output/
    └── [小说名]/
        ├── 00-一句话概括.md

## 步骤 1：一句话概括

### 用户询问模板

```
让我们开始创作！第一步是确定故事的核心概念。

请告诉我：
1. 【题材】你想写什么类型的故事？（如：都市、奇幻、悬疑、言情、科幻）
2. 【主角】主角是什么样的角色？（如：社恐程序员、穿越者、高中生侦探）
3. 【冲突】核心冲突是什么？（如：身份危机、生存挑战、情感纠葛）
4. 【基调】故事的整体基调？（如：轻松、沉重、搞笑、温馨）

你也可以直接描述你的创意，我来帮你提炼。
```

### AI 生成模板

```
基于你的描述，我为你生成了 5 个一句话概括选项：

【选项 A】[≤25字的一句话]
【选项 B】[≤25字的一句话]
【选项 C】[≤25字的一句话]
【选项 D】[≤25字的一句话]
【选项 E】[≤25字的一句话]

请选择你最感兴趣的一个，或者告诉我需要调整的方向。
```

### 一句话概括公式

**公式A（通用型）**：
```
[主角身份/特征] + [面临的外在困境] + [采取的核心行动] + [潜在结局暗示（不透露）]

示例：
- 社恐程序员 × 发现公司秘密 × 伪装黑客调查 × 最终做出惊人选择
- 穿越农家女 × 家徒四壁欠巨债 × 靠现代知识创业 × 逆风翻盘
```

**公式B（正负价值对立型，适合网文/商业向）**：
```
xx（负价值形容词）的xx（主角身份）想要xx（欲望，包含正价值）而去xx的故事

要点：正价值必须和负价值相对立；欲望最好是具象的

示例：
- 废柴的天才想要出人头地而去成为斗帝的故事
- 被遗弃的医女想要生存必须去陪葬的故事
- 退休的战神想要报恩而去都市中掀起血雨腥风的故事
```

## 步骤 2：五句式扩展

### 生成提示词

```
请基于以下一句话概括，生成五句式大纲：

"[用户选择的一句话概括]"

要求：
1. 第1句：介绍故事背景和主人公
2. 第2句：第一幕结尾 - 第一次灾难性事件（迫使主角投入故事）
3. 第3句：第二幕前半 - 第二次灾难性事件（道德/信念转折）
4. 第4句：第二幕后半 - 第三次灾难性事件（导向高潮）
5. 第5句：结局 + 道德前提升华

请确保：
- 包含3次明确的灾难性事件/转折点
- 明确主角的道德转变（从X到Y）
- 保持悬念，结局不要过于直白
```

### 五句式结构检查

```markdown
## 五句式大纲检查清单

- [ ] 第1句：背景+主角是否清晰？
- [ ] 第2句：第一次灾难是否迫使主角行动？
- [ ] 第3句：第二次灾难是否触发信念转变？
- [ ] 第4句：第三次灾难是否导向最终对决？
- [ ] 第5句：结局是否有情感升华？
- [ ] 整体：道德前提是否明确？
- [ ] 整体：三次灾难是否有递进关系？
```

## 步骤 4：一页纸大纲

### 生成提示词

```
请将以下五句式大纲扩展为一页纸大纲（约500字）：

"""
[用户的五句式大纲]
"""

要求：
1. 五句话每句扩展为一个段落（共5段）
2. 每段添加：
   - 具体场景描述
   - 关键对话片段（1-2句）
   - 情绪/氛围描写
3. 保持段落之间的逻辑连贯
4. 控制总字数在400-600字之间

输出格式：
## 一页纸大纲

### 第一幕：[标题]
[扩展内容]

### 第二幕（上）：[标题]
[扩展内容]

### 第二幕（下）：[标题]
[扩展内容]

### 第三幕：[标题]
[扩展内容]

### 结局：[标题]

## 步骤 5：人物背景故事

### 生成提示词

```
请为以下角色撰写背景故事（500-800字）：

**角色**：[角色名]
**当前设定**：[已有的人物卡片信息]

请深入挖掘以下方面：

1. **童年关键事件**
   - 影响他性格形成的一件事
   - 与父母/抚养者的关系
   - 最早的记忆

2. **成长经历**
   - 学生时代的重要经历
   - 第一次重大失败
   - 第一次重大成功

3. **性格形成**
   - 他最深层的恐惧是什么？从何而来？
   - 他最渴望的是什么？为什么？
   - 他对自己最大的误解是什么？

4. **当前处境**
   - 故事开始前他的生活状态
   - 他面临的最大困境
   - 他与重要人物的关系

请用叙述性的语言撰写，而非问答形式。
```

## 步骤 6：四页纸完整大纲

### 生成提示词

```
请基于以下材料，生成四页纸完整大纲（2000-3000字）：

**一句话概括**：[...]
**五句式大纲**：[...]
**一页纸大纲**：[...]
**主要角色**：[角色列表]

要求：
1. 将故事划分为 [8-15] 章（适合短篇小说）
2. 每章包含：
   - 章节标题
   - 场景设定（时间、地点）
   - 核心事件（发生什么）
   - 情感锚点（本章情绪变化）
   - 悬念/钩子（结尾留什么悬念）
3. 标注三次灾难性事件的位置
4. 确保节奏张弛有度

**时间线类型适配**：

如果 `00-一句话概括.md` 中记录了非顺叙时间线类型，章节规划需要额外适配：

- **圆环时间线**：每章标注所属时间线（现在/过去/未来），三线交织推进，标注切换锚点
- **读档制**：标注循环轮次和本轮新增信息，重复章节递减篇幅
- **纯循环**：标注循环编号和本轮微小变化，用重复元素作为循环标记
- **多线并行**：标注每章所属线，各线独立推进，标注交汇章节

非顺叙时间线的章节规划扩展格式：

```markdown
**第X章：[章节标题]**
- 场景：[时间/地点]
- POV：[视角人物]
- 核心事件：[发生什么]
- 情感锚点：[本章情绪变化]
- 钩子：[结尾悬念]
- **所属时间线**：[现在线/过去线/未来线/A线/第N轮循环]
- **时间线切换锚点**：[从上一时间线切换的触发物]
- **信息差**：[读者此时知道/不知道什么]
```

输出格式：

## 完整大纲：[小说标题]

### 故事概览
- **总字数预估**：[X万字]
- **章节数**：[X章]
- **核心主题**：[道德前提]

### 章节规划

#### 第一幕：[幕名]

**第1章：[章节标题]**
- 场景：[时间/地点]
- POV：[视角人物]
- 核心事件：[...]
- 情感锚点：[...]
- 钩子：[...]

[重复所有章节]

### 伏笔清单
| 伏笔 | 埋设章节 | 收束章节 |
|------|----------|----------|
| ... | ... | ... |
```

### 防流水账检查点

在生成正文时，必须确保每章包含：

```
## 流水账自检清单

### 1. 冲突检查
- [ ] 本章是否有明确的冲突？（人物冲突/内心冲突/环境冲突）
- [ ] 冲突是否有解决或推进？
- [ ] 是否有对抗力量阻碍主角？

### 2. 情绪起伏检查
- [ ] 读者本章会感到什么情绪？（愤怒/紧张/期待/感动）
- [ ] 是否有情绪高潮点？
- [ ] 情绪是否有变化曲线？

### 3. 期待感检查
- [ ] 读者会想知道"接下来发生什么"吗？
- [ ] 是否设置了悬念或钩子？
- [ ] 是否有未完成的目标/任务？

### 4. 节奏检查
- [ ] 是否有事件发生？（不是单纯的"过了一天"）
- [ ] 事件是否有因果关系？
- [ ] 是否有紧张与舒缓的交替？
```

**流水账警告信号**：
- 连续3段以上只是"描述发生了什么"
- 没有任何角色的情绪反应
- 没有冲突或矛盾
- 读完后不记得发生了什么

### 开篇吸引力强化

如果是小说开篇（第1-3章），需要额外检查：

```
## 开篇强化检查

### 第1章：3秒抓住读者
- [ ] 开篇第一句是否有冲击力？
- [ ] 是否避免了"背景介绍式"开场？
- [ ] 是否在首段就制造了冲突/危机/悬念？
- [ ] 信息密度是否合理？（避免信息轰炸）

### 章节结尾钩子设计

每章结尾必须设置钩子，引导读者点击"下一章"：

```
## 钩子设计提示

### 常用钩子类型

1. **悬念钩**：留下未解之谜
   示例："他不知道的是，真正的危险才刚刚开始……"

2. **转折钩**：出乎意料的揭示
   示例："而那个杀手，竟然是他的亲生弟弟。"

3. **期待钩**：预告即将发生的大事
   示例："明天，就是宗门大比的日子。"

4. **危机钩**：制造紧迫感
   示例："门外传来脚步声——有人来了。"

5. **揭秘钩**：揭示重要信息
   示例："他终于知道了，父亲当年'失踪'的真相。"

### 钩子设计原则
- 钩子必须与主线剧情相关
- 不要"为钩而钩"，制造虚假悬念
- 钩子要在后续章节收束
```

## 步骤 11：人语化处理（关键步骤）

AI 生成的正文需要经过人语化处理，才能达到发布标准。建议人工修改比例 ≥30%。

### 为什么需要人语化？

AI 是概率预测机，倾向于选择**最安全、最常见**的词汇和句式，导致：
- 句式过于整齐划一
- 词汇过于标准规范
- 缺乏个人风格和情感
- 结尾喜欢升华主题

### 人语化核心指标

| 指标 | 说明 | AI示例 | 人类示例 |
|------|------|--------|----------|
| **困惑度** | 语言的意外性 | "我非常喜欢阅读" | "书这玩意儿，那是我的精神鸦片" |
| **爆发度** | 节奏的变化 | 长度均匀的句子 | 长短句交替，突然截断 |

---

### 人语化处理流程

```
初稿 → 第一轮AI润色 → 人工标注 → 第二轮AI优化 → 人工终审 → 定稿
```

### 第一轮：AI自动润色

### 提示词：全面人语化

```
请对以下小说正文进行人语化处理，遵循以下规则：

## 必须执行

1. **句式变化**
   - 把长句拆成短句（≥20字的句子拆分）
   - 长短句交替，节奏要有顿挫感
   - 适当使用倒装句、省略句、设问句

2. **口语化处理**
   - 加入自然过渡词："说真的"、"你猜怎么着"、"话说回来"
   - 用具体事例替代抽象概念
   - 加入反问句和幽默比喻

3. **情感注入**
   - 每段至少1处感官描写（视觉、听觉、嗅觉、触觉）
   - 每章至少3处情绪锚点（人物内心独白/微表情/习惯动作）
   - 用细节代替直接描述情感

4. **角色个性**
   - 对话要符合人物设定（口头禅、语气习惯）
   - 加入角色的标志性动作
   - 体现人物的思维特点

## 严禁使用

❌ 禁止词汇：
- "综上所述"、"毋庸置疑"、"深入探讨"
- "在当今社会"、"随着...的发展"
- "值得注意的是"、"至关重要"
- "织锦"、"交响曲"、"灯塔"、"画卷"（幻觉系词汇）

❌ 禁止句式：
- "首先...其次...最后..."的列表结构
- "不仅仅是...更是..."的废话句式
- 结尾强行升华主题

❌ 禁止标点：
- 过度使用分号
- 过度使用破折号

## 输出要求

请用对比格式展示修改：

| 原文 | 修改后 | 修改理由 |
|------|--------|----------|
| [原文片段] | [修改内容] | [为什么要这样改] |

---

[待处理的正文]
```

### 提示词：对话优化

```
请优化以下对话，使其更自然、更有人味：

## 优化规则

1. **口语化**
   - 删除书面语，使用日常表达
   - 加入语气词："呃"、"嗯"、"那个..."
   - 使用不完整句子（人说话不会每句都完整）

2. **个性化**
   - 每个角色有自己的说话风格
   - 加入口头禅或习惯用语
   - 体现角色的情绪状态

3. **潜台词**
   - 话中有话，不要把意思说得太直白
   - 用动作和表情暗示言外之意
   - 适当留白，让读者自己体会

## 示例

❌ AI对话：
"我认为这个方案非常优秀，因为它能够解决我们的问题。"
"我完全同意你的观点，我们应该立即实施。"

✅ 人类对话：
"这招行。"
"那就干吧，别磨叽了。"

---

[待优化的对话内容]
```

### 提示词：情感场景深化

```
请深化以下情感场景，使其更有感染力：

## 深化规则

1. **用动作代替描述**
   - ❌ "她感到非常紧张"
   - ✅ "她的手指不停地绞着衣角，连茶杯端起来又放下，端起来又放下。"

2. **用感官代替说明**
   - ❌ "房间里很压抑"
   - ✅ "空气像凝固了一样，连呼吸都觉得费劲。墙上的钟滴答滴答，每一声都像敲在心口上。"

3. **用细节代替概括**
   - ❌ "他很伤心"
   - ✅ "他盯着那张旧照片，看了很久。然后慢慢折起来，放进贴身的口袋，又拍了拍，像是在确认它还在。"

4. **留白与暗示**
   - 不要把情感说透，让读者自己感受
   - 用环境描写烘托情绪
   - 用小物件承载情感

---

[待深化的场景]
```

### 原创性
- [ ] AI痕迹是否<30%？
