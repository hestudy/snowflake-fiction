# 辨识度优化公式和改写示范

## 核心公式

```
[常见题材] + [独特元素] = 有辨识度的作品
```

### 公式示例

| 常见题材 | + | 独特元素 | = | 有辨识度的作品 |
|----------|---|----------|---|----------------|
| 系统 | + | 情绪兑换 | = | 用愤怒值兑换能力的系统 |
| 重生 | + | 反派身份 | = | 重生到反派身上 |
| 修仙 | + | 赛博朋克 | = | 赛博修仙世界 |
| 都市 | + | 宠物进化 | = | 我的宠物能无限进化 |

## 辨识度优化示例

### 示例1：系统文优化

```
❌ 原版：
"一个普通人获得系统，从此走上人生巅峰"

✅ 优化版：
"社恐的他在系统中发现，自己的'社交恐惧值'可以兑换任何能力。
于是他开始疯狂'社恐'——
别人参加聚会，他在家研究量子物理；
别人谈恋爱，他在实验室制造机器人。
三年后，世界首富、科学巨匠、传说级黑客……
记者问：'您成功的秘诀是什么？'
他：'我只是……不想出门而已。'"
```

### 示例2：重生文优化

```
❌ 原版：
"重生回到十年前，这次他要改变命运"

✅ 优化版：
"重生回到十年前，他发现自己变成了当年害死自己的那个反派。
更要命的是——原来的主角正用当年他对自己的眼神看着他。"
```

## 书名优化公式

### 好书名的特征

| 特征 | 说明 | 示例 |
|------|------|------|
| 有画面感 | 读者能想象场景 | 《盗墓笔记》 |
| 有悬念 | 引发好奇心 | 《他知道风从哪个方向来》 |
| 有独特性 | 不容易和别人撞 | 《赛博修仙纪》 |
| 好搜索 | 关键词明确 | 《我的宠物能无限进化》 |

### 书名优化示例

```
❌ 原版：《重生之路》
问题：太泛，搜索结果淹没

✅ 优化版：《重生成了自己的仇人》
优点：有悬念，有独特性，好搜索
```

## 简介优化公式

### 简介三要素

```
1. 钩子（第一句抓人）
2. 设定（独特卖点）
3. 悬念（为什么要看下去）
```

### 简介优化示例

```
❌ 原版：
"这是一个关于修仙的故事。主角从小被欺负，后来获得了一个系统，
开始了修仙之路。一路升级打怪，最终成为最强者。"

✅ 优化版：
"修仙界有个不成文的规矩：废柴就该老老实实当废柴。
但林凡不这么想。
他的系统很特别——每次被人看不起，实力就涨一截。
于是他开始了一段'越被嘲笑越强'的修仙之路。
问题是……当他强到没人敢嘲笑他的时候，系统还能用吗？"
```

### 「背景+小剧场」简介公式（古言/言情适用）

> 来源：番茄资深编辑柳莫辞、堆堆经验

除了"钩子+设定+悬念"三要素外，还有一种更具体、更容易操作的简介写法，尤其适合古言和言情类：

**公式**：背景（一句话主线）+ 经典互动小剧场

| 部分 | 内容 | 作用 |
|------|------|------|
| 背景 | 故事梗概，一句话主线（故事冲突/人物前后命运） | 让读者快速了解故事类型和核心矛盾 |
| 小剧场 | 精彩对话片段（打脸暴爽桥段/男女主甜蜜互动） | 展示文笔和风格，直接勾住读者 |

**示例**（重生宅斗文）：

```
【背景】
前世，她倾尽所有助他登上皇位，换来的却是他与庶妹联手，
将她满门抄斩。重活一世，她要亲手夺回一切。

【小剧场】
"王爷，王妃她又把丞相府的人打了。"
"打得好。赏。"
"可是……她把人扔进了护城河。"
男人翻了一页书，淡淡道："那就再赏一条河。"
```

**为什么有效**：
- 背景让读者知道"这本书讲什么"，筛选精准受众
- 小剧场让读者直接体验"这本书好不好看"，降低点击正文的心理门槛
- 两者组合比纯文字梗概更有吸引力

## 选题自检清单

```
□ 一句话能说清独特之处吗？
□ 标签与内容匹配吗？
□ 混搭元素≤3个吗？
□ 开篇设定≤3个核心概念吗？
□ 读者为什么要看你的而不是别人的？
```

## 辨识度提升方向

| 方向 | 方法 | 效果 |
|------|------|------|
| 加独特系统 | 给常见题材加新颖的系统机制 | 让老题材焕发新意 |
| 换独特身份 | 让主角身份与众不同 | 制造新鲜感 |
| 混独特世界观 | 跨题材融合 | 创造新品类 |
| 给独特金手指 | 能力形式特别 | 增加记忆点 |
| 用独特视角 | 换个角度讲故事 | 带来新体验 |
