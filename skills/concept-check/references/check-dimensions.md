# 五维度检测标准

## 1. 题材与内容匹配检测（权重 25%）

**核心问题**：标签/分类与实际内容不符，读者会感到被欺骗。

### 不匹配表现

```
❌ 不匹配示范：
标签：玄幻
实际内容：写了80%的都市生活，20%的修仙

结果：
- 玄幻读者觉得不够玄幻
- 都市读者不想看修仙
- 两头不讨好
```

### 检测清单

- [ ] **主标签与主线是否匹配？**
  - 标签是玄幻 → 主线应该是修炼升级
  - 标签是都市 → 主线应该是都市生活

- [ ] **内容占比是否合理？**
  - 主题材应该占70%以上
  - 副题材不超过30%

- [ ] **读者预期是否被满足？**
  - 冲着标签来的读者能看到想看的内容吗？

### 题材匹配标准

| 题材 | 主线应该是 | 占比要求 |
|------|------------|----------|
| 玄幻/仙侠 | 修炼升级、境界突破 | ≥70% |
| 都市 | 都市生活、职场/情感 | ≥70% |
| 历史 | 历史背景、权谋/种田 | ≥70% |
| 悬疑 | 推理破案、真相揭露 | ≥70% |

### 时间线类型匹配检测

如果 `00-一句话概括.md` 中记录了时间线类型，额外检测时间线与题材的匹配度。

**检测清单**：

- [ ] **时间线类型与题材是否匹配？**（参照下方对照表）
- [ ] **时间线类型是否在正文中体现？**（选了圆环但实际写成了顺叙 = 不匹配）
- [ ] **时间线切换是否清晰？**（读者能分辨当前在哪条时间线上吗？）
- [ ] **时间线选择是否有必要？**（顺叙能讲好的故事不需要强行用复杂时间线）

**时间线类型适合/不适合题材对照表**：

| 时间线类型 | 适合题材 | 不适合题材 | 原因 |
|------------|----------|------------|------|
| 顺叙 | 升级流、种田、日常、言情 | — | 万能型，无不适合 |
| 圆环 | 悬疑、文艺、命运感、重生 | 纯升级流、轻松日常 | 升级流需要线性爽感，圆环会打断节奏 |
| 读档制 | 无限流、生存、解谜、策略 | 纯言情、种田 | 言情和种田的核心是情感积累，回档会清零 |
| 纯循环 | 密室、哲理、心理、短篇 | 长篇升级、百万字网文 | 循环天然适合短篇，长篇容易疲劳 |
| 多线并行 | 群像、史诗、战争、权谋 | 单主角爽文 | 多线会分散对主角的关注 |

**评分标准**（作为题材匹配维度的子项）：

| 情况 | 分数调整 |
|------|----------|
| 时间线与题材高度匹配 | +5分 |
| 时间线与题材基本匹配 | +0分（不加不减） |
| 时间线与题材不匹配 | -10分 |
| 选了非顺叙但实际写成顺叙 | -5分（名不副实） |
| 未选择时间线类型 | 不扣分（默认顺叙） |

### 修复方案

```
✅ 修复方案：

方案A：调整标签
- 如果内容偏都市，就把标签改成"都市"
- 诚实面对读者

方案B：调整内容
- 增加主题材元素的比例
- 确保主线与标签匹配
```

### 评分标准

| 分数 | 描述 |
|------|------|
| 25-30 | 标签与内容高度匹配，主题材占比≥70% |
| 15-24 | 基本匹配，但有偏离 |
| 5-14 | 明显不匹配，需要调整 |
| 0-4 | 完全脱节 |

## 2. 辨识度检测（权重 25%）

**核心问题**：同质化严重，读者没有选择你的理由。

### 无辨识度表现

```
❌ 无辨识度示范：
"一个普通人获得系统，从此走上人生巅峰"

问题：
- 这种设定太常见
- 读者为什么要看你的？
- 在千万本书中如何被发现？
```

### 检测清单

- [ ] **一句话能说清独特之处吗？**
- [ ] **与同类作品有什么不同？**
- [ ] **读者能记住什么？**
- [ ] **搜索时能用什么关键词找到？**

### 辨识度类型

| 类型 | 说明 | 示例 |
|------|------|------|
| **独特系统** | 系统机制新颖 | 情绪兑换、死亡回档 |
| **独特身份** | 主角身份特别 | 穿成反派、重生为NPC |
| **独特世界观** | 设定与众不同 | 修仙+赛博、魔法+工业 |
| **独特金手指** | 能力形式特别 | 能看到好感度、能读取记忆 |
| **独特视角** | 叙事角度特别 | 反派视角、配角视角 |

### 评分标准

| 分数 | 描述 |
|------|------|
| 25-30 | 独特卖点清晰，一句话能说清 |
| 15-24 | 有亮点但不够突出 |
| 5-14 | 同质化严重，缺乏记忆点 |
| 0-4 | 完全没有辨识度 |

## 3. 跨题材混搭检测（权重 15%）

**核心问题**：混搭元素过多，变得四不像。

### 混搭不当表现

```
❌ 混搭不当示范：
都市 + 修仙 + 系统 + 末世 + 穿越重生

问题：
- 什么都有，什么都不像
- 读者不知道这是什么类型
- 每个元素都浅尝辄止
```

### 检测清单

- [ ] **混搭元素数量是否≤3个？**
- [ ] **是否有明确的主次之分？**
- [ ] **混搭是否有化学反应？**
- [ ] **读者能理解这是什么类型吗？**

### 混搭规则

| 规则 | 说明 |
|------|------|
| 元素≤3个 | 太多会混乱 |
| 有主有次 | 主题材占70%以上 |
| 逻辑自洽 | 两个元素能融合 |
| 读者易懂 | 一句话能说清 |

### 正确的混搭方式

```
✅ 正确混搭示范：

原则：主题材（70%）+ 副题材（30%）

示例：
都市为主 + 修仙为辅 = 都市修仙
  - 主线是都市生活、职场、情感
  - 修仙是金手指，服务于都市剧情

玄幻为主 + 系统为辅 = 系统玄幻
  - 主线是修炼升级、境界突破
  - 系统是辅助，不喧宾夺主
```

### 评分标准

| 分数 | 描述 |
|------|------|
| 17-20 | 混搭合理，主次分明，化学反应好 |
| 10-16 | 基本合理，但可优化 |
| 5-9 | 元素过多或主次不分 |
| 0-4 | 四不像，需要精简 |

## 4. 世界观设定检测（权重 15%）

**核心问题**：设定过多或呈现方式不当，读者记不住。

### 设定过多表现

```
❌ 设定过多示范：
开篇三章内倾倒了：
- 修炼体系（9个大境界，每个境界9层）
- 势力分布（8大门派，每个门派的历史）
- 地理设定（东南西北四大区域）
- 物品设定（丹药等级、法宝等级）

问题：读者记不住，也不想记。
```

### 检测清单

- [ ] **开篇设定是否≤3个核心概念？**
- [ ] **设定是否通过剧情展开？**
- [ ] **读者需要记忆的内容是否过多？**
- [ ] **设定是否有必要？**

### 设定呈现原则

| 原则 | 说明 |
|------|------|
| 场景化 | 把设定融入具体场景 |
| 对话化 | 通过角色对话呈现 |
| 渐进式 | 逐步展开，不要一次倾倒 |
| 需要时才说 | 只在剧情需要时介绍 |

### 正确的设定呈现

```
✅ 场景化呈现示范：
李云跟在师兄身后，第一次走进青云宗的藏经阁。

"记住，外门弟子只能进第一层。"师兄指了指，"第二层是内门弟子才能进的。至于第三层……那是核心弟子才有资格踏入的地方。"

李云点点头，心里却在想：总有一天，他要站在最高那层。

解析：
- 通过场景和对话呈现设定
- 读者跟着主角视角理解世界
- 设定服务于剧情，不是为了设定而设定
```

### 评分标准

| 分数 | 描述 |
|------|------|
| 17-20 | 设定精炼，场景化呈现，渐进展开 |
| 10-16 | 设定基本合理，呈现方式可优化 |
| 5-9 | 设定过多或呈现方式不当 |
| 0-4 | 开篇信息倾倒，读者无法消化 |

## 5. 书名质量检测（权重 20%）

**核心问题**：书名是作品的门面，直接决定读者是否点击。好内容被烂书名埋没是最大的浪费。

> 详细的起名思路、男女频案例和优化公式，参见 `./book-title-guide.md`

### 书名质量差的表现

```
❌ 低质量书名示范：
《重生之路》《修仙传》《都市之王》

问题：
- 太泛，搜索结果被淹没
- 没有独特记忆点
- 无法判断与同类作品的区别
```

### 检测清单

- [ ] **字数是否≤15字？**（过长难记忆、难传播）
- [ ] **是否贴合实际内容？**（标题党会导致读者流失和差评）
- [ ] **能否一眼看出题材/类型？**（读者筛选效率）
- [ ] **有没有独特记忆点？**（区别于同类作品）
- [ ] **是否有悬念或好奇心钩子？**（驱动点击）
- [ ] **是否搜索友好？**（关键词明确，不易被淹没）

### 书名类型对照

| 类型 | 适用场景 | 示例 |
|------|----------|------|
| 人物型 | 主角身份有吸引力 | 《战神奶爸》《乡村神医》 |
| 符号型 | 有年代/数字元素 | 《重生：回到1983当富翁》 |
| 故事主线型 | 主线剧情有噱头 | 《我在精神病院学斩神》 |
| 分类型 | 精准吸引目标读者 | 《四合院之xxxx》 |
| 悬念型 | 有矛盾冲突 | 《开局就和老婆离婚》 |
| 爽点型 | 金手指/高光突出 | 《我的右眼是神级计算机》 |
| 搞怪型 | 风格独特有趣 | 《我一个史莱姆吊打巨龙很合理吧？》 |
| 顺口溜型 | 朗朗上口、抖机灵展示卖点 | 句式有韵律感，近期番茄热门起名方式 |

### 男女频差异

**女频加分项**：情绪词（"救命！"）、甜宠风格词、剧情噱头前置、精准概括风格词（《心肝宠》）

**男频加分项**：题材直观、金手指展示、强烈对比（低代价+高收益）、成长线明确

### 修复方案

```
✅ 书名优化方向：

1. 加热词：融入题材/身份/核心梗热词
2. 加悬念：用矛盾对比制造好奇心
3. 加数字：具体数字增强记忆点
4. 加情绪：情绪词提升代入感（女频尤其有效）
5. 精简化：删减到15字以内，去掉生僻字
6. 贴合化：确保书名与实际内容强相关
```

### 评分标准

| 分数 | 描述 |
|------|------|
| 17-20 | 书名精准、有记忆点、有悬念、贴合内容、搜索友好 |
| 10-16 | 基本合格，但缺乏亮点或记忆点 |
| 5-9 | 存在明显问题（过长/晦涩/不匹配/无辨识度） |
| 0-4 | 书名严重拖后腿，需要重新起名 |
