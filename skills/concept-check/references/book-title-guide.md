# 书名质量指南

> 基于番茄小说编辑经验总结，适用于网文和商业小说。

## 起书名四大原则

### 原则1：贴合作品设定

书名必须与实际内容强相关。标题党会导致：
- 读者被噱头吸引点击，发现内容大相径庭
- 读者流失 + 差评 → 口碑崩塌
- 读完率暴跌

### 原则2：直观展示核心卖点

利用热词让读者快速了解作品类型和爽点：

| 热词类型 | 示例 |
|----------|------|
| 题材 | 重生、种田、甜宠、系统、末世 |
| 身份 | 赘婿、学霸、总裁、王妃、战神 |
| 核心梗 | 替身、契约、逆袭、打脸 |
| 人设 | 腹黑、霸道、禁欲、废柴 |

### 原则3：对比反差制造悬念

将矛盾较大的特性组合在一起，勾起好奇心：
- 身份反差：高贵身份 + 卑微处境
- 情节反转：虐完我 → 求我回头
- 能力悖论：废柴 + 逆天金手指

### 原则4：简洁明了

- 字数控制在 **15字以内**
- 避免生僻字和拗口表达
- 朗朗上口，便于记忆和安利

## 7种起名思路

### 1. 人物型

用主角名、人物关系、职业作为书名。为人所熟知的人物更具吸引力。

```
示例：
《战神奶爸》《乡村神医》
《绝世萌宝：天才娘亲帅炸了》
```

### 2. 符号型

使用年份、时间、故事地点等元素。数字组合能提升书名吸引力。

```
示例：
《重生：回到1983当富翁》
《穿书七零我每月领八十八份工资》
```

**技巧**：带数字的书名（"八十八份工资"、"五个崽"、"六年后"）天然有记忆点。

### 3. 故事主线型

直接在书名中阐述核心主线剧情或主角关键任务。

```
示例：
《我在精神病院学斩神》
《开局地摊卖大力》
《炒股：从新手到大神》
```

**优点**：读者一眼就知道故事写什么，有效筛选精准受众。

### 4. 分类型

直接抛出题材/类型标签，吸引精准读者。

```
示例：
《四合院之xxxx》《重生xxxx》《xxxx末世》
```

### 5. 设置悬念型

通过矛盾冲突、对比反衬在标题中制造悬念。

```
示例：
《开局就和老婆离婚》
《大唐：抗旨才能变强》
《出大事了！相府嫡女她被退婚了》
```

**技巧**：情绪词（"救命！""出大事了！"）能强化悬念效果。

### 6. 抛爽点型

直接展示金手指或主角高光时刻。

```
示例：
《我是亿亿富翁，我摊牌了》
《我的右眼是神级计算机》
《修仙，我的眼睛能看见灵药生长度》
```

**注意**：爽点必须贴合内容，不能挂羊头卖狗肉。

### 7. 抽象搞怪型

通过趣味性、搞怪措辞突出独特风格。

```
示例：
《我一个史莱姆吊打巨龙很合理吧？》
```

### 8. 顺口溜型

> 来源：番茄小说女频脑洞文创作指南

通过抖机灵的方式把作品的主要卖点或人设表现出来，朗朗上口、句式别出心裁，让读者眼前一亮。强调的重点可以是任意卖点，只要顺口即可。

```
示例：
核心梗顺口溜：把脑洞设定编成顺口溜
人设顺口溜：用押韵句式展现角色特点
卖点顺口溜：将多个看点串成朗朗上口的句子
```

**技巧**：句式节奏感比内容更重要，读起来有韵律感就能吸引点击。近期番茄比较热门的起名方式。

## 男女频书名差异

### 女频书名特点

| 技巧 | 说明 | 示例 |
|------|------|------|
| 剧情噱头前置 | 把开篇亮点剧情直接做书名 | 《救命！刚重生就被死对头掐腰索吻》 |
| 情绪词强调 | "救命！""出大事了！"提升代入感 | 《出大事了！相府嫡女她被退婚了》 |
| 精准概括风格 | 简短词汇传达作品调性 | 《心肝宠》《温柔瘾》 |
| 反转悬念 | 身份+意外事件制造好奇 | 相府嫡女被退婚→为什么？怎么反击？ |
| 数字组合 | 具体数字增加记忆点 | 《穿书七零我每月领八十八份工资》 |

### 男频书名特点

| 技巧 | 说明 | 示例 |
|------|------|------|
| 题材直观 | 开头直接圈定故事领域 | 《炒股：从新手到大神》 |
| 金手指展示 | 用主角特殊能力做书名 | 《修仙，我的眼睛能看见灵药生长度》 |
| 强烈对比 | 低代价+高收益制造爽感 | 《修仙五分钟，无敌两千年》 |
| 突出卖点 | 从新手到大神的成长线 | 让读者清楚get到故事内容 |

## 书名质量检测清单

### 必检项

- [ ] **字数是否≤15字？**（过长难记忆）
- [ ] **是否贴合实际内容？**（标题党会反噬）
- [ ] **能否一眼看出题材/类型？**（读者筛选效率）
- [ ] **有没有独特记忆点？**（区别于同类作品）
- [ ] **是否有悬念或好奇心钩子？**（驱动点击）

### 加分项

- [ ] 包含数字组合（增强记忆）
- [ ] 包含情绪词（增强代入）
- [ ] 包含对比反差（制造张力）
- [ ] 搜索友好（关键词明确，不易被淹没）
- [ ] 朗朗上口（便于口碑传播）

### 减分项

- [ ] 使用生僻字或拗口表达
- [ ] 过于文艺晦涩（网文读者不买账）
- [ ] 与大量已有作品撞名
- [ ] 书名与内容严重不符
- [ ] 无法从书名判断任何信息

## 评分标准

| 分数 | 描述 |
|------|------|
| 17-20 | 书名精准、有记忆点、有悬念、贴合内容 |
| 10-16 | 基本合格，但缺乏亮点或记忆点 |
| 5-9 | 存在明显问题（过长/晦涩/不匹配） |
| 0-4 | 书名严重拖后腿，需要重新起名 |

## 书名优化公式

### 公式1：题材热词 + 独特元素

```
[题材关键词] + [独特设定/金手指] = 有辨识度的书名

示例：
修仙 + 眼睛能力 = 《修仙，我的眼睛能看见灵药生长度》
重生 + 七零年代 + 数字 = 《穿书七零我每月领八十八份工资》
```

### 公式2：矛盾对比

```
[低代价/负面] + [高收益/正面] = 制造爽感的书名

示例：
修仙五分钟 + 无敌两千年
废柴少年 + 逆天系统
```

### 公式3：情绪词 + 剧情噱头

```
[情绪词] + [核心剧情亮点] = 女频常用的高点击书名

示例：
救命！ + 刚重生就被死对头掐腰索吻
出大事了！ + 相府嫡女她被退婚了
```

### 公式4：身份 + 意外

```
[高/低身份] + [反转事件] = 制造悬念的书名

示例：
相府嫡女 + 被退婚
亿亿富翁 + 摊牌了
```

## 书名拆解案例

> 来源：番茄资深编辑柳莫辞、堆堆经验

### 案例：《被赶出府，她身穿嫁衣扭头嫁残王》

逐词拆解，展示如何在一个书名中同时塞入故事性+反转+悬念：

| 部分 | 文字 | 功能 |
|------|------|------|
| 引子 | "被赶出府" | 以负面事件切入，制造同情+好奇（为什么被赶？） |
| 场景 | "她身穿嫁衣" | 交代关键场景——大婚当天，画面感极强 |
| 反转 | "扭头嫁残王" | 核心噱头——被赶出去不哭不闹，当场悔婚嫁别人 |

**留下的悬念链**：
1. 她为什么被赶出府？
2. 大婚当日嫁给别人，她会有什么后果？
3. 这个"残王"是谁？（暗示男主，读者期待残王真实身份）

**拆解公式**：`负面事件 + 关键场景 + 反转行动 = 故事性书名`

### 案例：《重生后嫁你弟夺你江山》

| 部分 | 文字 | 功能 |
|------|------|------|
| 题材标签 | "重生后" | 一个词圈定题材，精准筛选读者 |
| 复仇手段1 | "嫁你弟" | 直接用"你"指代仇人，复仇情绪拉满 |
| 复仇手段2 | "夺你江山" | 升级打击，不只是感情报复，还要夺权 |

**为什么有效**：书名本身就是一句狠话，读者读完就能感受到女主的复仇决心，两个看点（嫁弟+夺江山）叠加形成强噱头。
