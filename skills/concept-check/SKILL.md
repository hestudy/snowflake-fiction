# 创意与选题检查：定位题材脱节、辨识度不足、混搭不当、书名质量差的选题问题

检测小说选题和创意相关的常见问题，包括：
- **题材与内容脱节**：标签与实际内容不匹配
- **缺乏辨识度**：同质化严重，没有独特卖点
- **混搭不当**：元素过多，变得四不像
- **世界观设定问题**：设定过多或呈现方式不当
- **书名质量差**：书名不吸引人、不贴合内容、缺乏记忆点

> 核心洞察：选题决定了作品的**天花板**。内容再好，选题不对，读者也找不到你。书名是第一道门槛。

---

## 检测流程

```
原文分析 → 五维度扫描 → 定位问题 → 评分分级 → 输出报告
```

### 你的任务

1. **五维度扫描** - 逐项检测五个维度（详见知识库）
2. **定位问题** - 标记具体问题和严重程度
3. **评分分级** - 按权重计算综合分数
4. **给出修复建议** - 针对每个问题给出具体优化方向
5. **提供优化示范** - 选取最严重的问题进行优化示范

## 注意事项

1. **诚实面对读者**：标签要与内容匹配
2. **要有独特卖点**：读者为什么要选你？
3. **混搭要克制**：贪多必失
4. **设定要渐进**：不要开篇就倾倒
5. **一句话能说清**：如果说不清，说明不够清晰
6. **书名是门面**：好内容不能被烂书名埋没，15字以内、有记忆点、贴合内容
