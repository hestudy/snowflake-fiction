# 创意与选题检查 Skill

检测小说选题和创意相关的常见问题，包括：
- **题材与内容脱节**：标签与实际内容不匹配
- **缺乏辨识度**：同质化严重，没有独特卖点
- **混搭不当**：元素过多，变得四不像
- **世界观设定问题**：设定过多或呈现方式不当

## 为什么选题问题致命？

| 问题 | 读者反应 | 平台影响 |
|------|----------|----------|
| 题材不匹配 | 找不到想看的 | 标签投诉、退订 |
| 无辨识度 | 为什么要看你的？ | 推荐转化低 |
| 混搭不当 | 四不像 | 读者定位混乱 |
| 设定过多 | 记不住 | 弃书率高 |

## 检查维度

### 1. 题材与内容匹配检测

**核心问题**：标签/分类与实际内容不符，读者会感到被欺骗。

#### 不匹配表现

```
❌ 不匹配示范：
标签：玄幻
实际内容：写了80%的都市生活，20%的修仙

结果：
- 玄幻读者觉得不够玄幻
- 都市读者不想看修仙
- 两头不讨好
```

#### 题材匹配检测清单

- [ ] **主标签与主线是否匹配？**
  - 标签是玄幻 → 主线应该是修炼升级
  - 标签是都市 → 主线应该是都市生活

- [ ] **内容占比是否合理？**
  - 主题材应该占70%以上
  - 副题材不超过30%

- [ ] **读者预期是否被满足？**
  - 冲着标签来的读者能看到想看的内容吗？

#### 题材匹配标准

| 题材 | 主线应该是 | 占比要求 |
|------|------------|----------|
| 玄幻/仙侠 | 修炼升级、境界突破 | ≥70% |
| 都市 | 都市生活、职场/情感 | ≥70% |
| 历史 | 历史背景、权谋/种田 | ≥70% |
| 悬疑 | 推理破案、真相揭露 | ≥70% |

#### 修复方案

```
✅ 修复方案：

方案A：调整标签
- 如果内容偏都市，就把标签改成"都市"
- 诚实面对读者

方案B：调整内容
- 增加玄幻元素的比例
- 确保主线与标签匹配
```

### 2. 辨识度检测

**核心问题**：同质化严重，读者没有选择你的理由。

#### 无辨识度表现

```
❌ 无辨识度示范：
"一个普通人获得系统，从此走上人生巅峰"

问题：
- 这种设定太常见
- 读者为什么要看你的？
- 在千万本书中如何被发现？
```

#### 辨识度检测清单

- [ ] **一句话能说清独特之处吗？**
- [ ] **与同类作品有什么不同？**
- [ ] **读者能记住什么？**
- [ ] **搜索时能用什么关键词找到？**

#### 建立辨识度的方法

**公式**：`[常见题材] + [独特元素] = 有辨识度的作品`

| 常见题材 | + | 独特元素 | = | 有辨识度的作品 |
|----------|---|----------|---|----------------|
| 系统 | + | 情绪兑换 | = | 用愤怒值兑换能力的系统 |
| 重生 | + | 反派身份 | = | 重生到反派身上 |
| 修仙 | + | 赛博朋克 | = | 赛博修仙世界 |
| 都市 | + | 宠物进化 | = | 我的宠物能无限进化 |

#### 辨识度类型

| 类型 | 说明 | 示例 |
|------|------|------|
| **独特系统** | 系统机制新颖 | 情绪兑换、死亡回档 |
| **独特身份** | 主角身份特别 | 穿成反派、重生为NPC |
| **独特世界观** | 设定与众不同 | 修仙+赛博、魔法+工业 |
| **独特金手指** | 能力形式特别 | 能看到好感度、能读取记忆 |
| **独特视角** | 叙事角度特别 | 反派视角、配角视角 |

#### 辨识度优化示例

```
❌ 原版：
"一个普通人获得系统，从此走上人生巅峰"

✅ 优化版：
"社恐的他在系统中发现，自己的'社交恐惧值'可以兑换任何能力。
于是他开始疯狂'社恐'——
别人参加聚会，他在家研究量子物理；
别人谈恋爱，他在实验室制造机器人。
三年后，世界首富、科学巨匠、传说级黑客……
记者问：'您成功的秘诀是什么？'
他：'我只是……不想出门而已。'"
```

### 3. 跨题材混搭检测

**核心问题**：混搭元素过多，变得四不像。

#### 混搭不当表现

```
❌ 混搭不当示范：
都市 + 修仙 + 系统 + 末世 + 穿越重生

问题：
- 什么都有，什么都不像
- 读者不知道这是什么类型
- 每个元素都浅尝辄止
```

#### 混搭合理性检测清单

- [ ] **混搭元素数量是否≤3个？**
- [ ] **是否有明确的主次之分？**
- [ ] **混搭是否有化学反应？**
- [ ] **读者能理解这是什么类型吗？**

#### 正确的混搭方式

```
✅ 正确混搭示范：

原则：主题材（70%）+ 副题材（30%）

示例：
都市为主 + 修仙为辅 = 都市修仙
  - 主线是都市生活、职场、情感
  - 修仙是金手指，服务于都市剧情

玄幻为主 + 系统为辅 = 系统玄幻
  - 主线是修炼升级、境界突破
  - 系统是辅助，不喧宾夺主
```

#### 混搭规则

| 规则 | 说明 |
|------|------|
| 元素≤3个 | 太多会混乱 |
| 有主有次 | 主题材占70%以上 |
| 逻辑自洽 | 两个元素能融合 |
| 读者易懂 | 一句话能说清 |

---

### 4. 世界观设定检测

**核心问题**：设定过多或呈现方式不当，读者记不住。

#### 设定过多表现

```
❌ 设定过多示范：
开篇三章内倾倒了：
- 修炼体系（9个大境界，每个境界9层）
- 势力分布（8大门派，每个门派的历史）
- 地理设定（东南西北四大区域）
- 物品设定（丹药等级、法宝等级）

问题：读者记不住，也不想记。
```

#### 世界观检测清单

- [ ] **开篇设定是否≤3个核心概念？**
- [ ] **设定是否通过剧情展开？**
- [ ] **读者需要记忆的内容是否过多？**
- [ ] **设定是否有必要？**

#### 正确的设定呈现

```
✅ 场景化呈现示范：
李云跟在师兄身后，第一次走进青云宗的藏经阁。

"记住，外门弟子只能进第一层。"师兄指了指，"第二层是内门弟子才能进的。至于第三层……那是核心弟子才有资格踏入的地方。"

李云点点头，心里却在想：总有一天，他要站在最高那层。

解析：
- 通过场景和对话呈现设定
- 读者跟着主角视角理解世界
- 设定服务于剧情，不是为了设定而设定
```

#### 设定呈现原则

| 原则 | 说明 |
|------|------|
| 场景化 | 把设定融入具体场景 |
| 对话化 | 通过角色对话呈现 |
| 渐进式 | 逐步展开，不要一次倾倒 |
| 需要时才说 | 只在剧情需要时介绍 |

## 注意事项

1. **诚实面对读者**：标签要与内容匹配
2. **要有独特卖点**：读者为什么要选你？
3. **混搭要克制**：贪多必失
4. **设定要渐进**：不要开篇就倾倒
5. **一句话能说清**：如果说不清，说明不够清晰
