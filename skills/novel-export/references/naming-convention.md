# 输出目录命名规范

导出时必须遵守的目录命名和结构规则。

---

## 平台-文件夹名称映射

导出时**必须**使用以下固定的文件夹名称，**不得使用其他变体**：

| 用户输入 | 输出文件夹名称 |
|----------|----------------|
| 番茄、番茄小说、番茄格式、番茄网 | `番茄小说` |
| 起点、起点中文网、起点格式、阅文 | `起点中文网` |
| 晋江、晋江文学城、晋江格式 | `晋江文学城` |
| 知乎、知乎盐选、盐选、知乎专栏 | `知乎盐选` |
| 七猫、七猫小说、七猫格式 | `七猫小说` |
| 纯文本、通用、txt | `通用纯文本` |
| Word、docx、word文档 | `Word文档` |

---

## 目录结构

```
小说目录/
├── 正文/                    # 源文件
│   ├── 第1章.md
│   ├── 第2章.md
│   └── ...
└── 导出/                    # 固定使用"导出"文件夹
    ├── 番茄小说/            # 统一命名，不创建变体
    │   ├── 第1章.txt        # 按章节独立导出
    │   ├── 第2章.txt
    │   └── ...
    └── 起点中文网/
        ├── 第1章.txt
        ├── 第2章.txt
        └── ...
```

---

## 命名约束

1. **文件夹名称必须统一**：无论用户输入"番茄"、"番茄小说"还是"番茄格式"，输出文件夹名称**必须**是 `番茄小说`
2. **复用已有文件夹**：如果 `导出/番茄小说/` 目录已存在，直接在此目录下导出，**不得**创建新的变体文件夹
3. **禁止合并文件**：批量导出时，每个章节**必须**导出为独立文件，**禁止**合并为单个文件
4. **文件扩展名**：导出文件统一使用 `.txt` 扩展名（Word 格式除外）
5. **文件命名**：保持与源文件相同的章节名，仅更改扩展名（如 `第1章.md` → `第1章.txt`）
