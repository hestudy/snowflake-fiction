# 角色检查维度详解

## 1. 人设扁平化检测

**核心问题**：所有角色说话风格一样，分不清是谁在说话。

### 扁平化表现

```
❌ 扁平化示范：
主角说："我觉得这个方案很不错，我们应该尝试一下。"
配角A说："我也认为这个方案很好，我们可以试试。"
配角B说："这个方案确实不错，我同意。"

问题：三人说话风格完全一致，毫无个性。
```

### 扁平化检测清单

- [ ] **遮住名字能分辨是谁吗？**
  - 如果不看清名字，能通过说话风格判断角色吗？
  - 每个角色有独特的语气、用词、句式吗？

- [ ] **每个角色有独特说话习惯吗？**
  - 口头禅
  - 常用句式
  - 情绪表达方式

- [ ] **不同角色性格差异明显吗？**
  - 急性子 vs 慢性子
  - 直接 vs 委婉
  - 乐观 vs 悲观

### 修复示例

```
✅ 个性化示范：
主角沉吟道："试试吧，反正也没别的办法。"

配角A立刻拍桌子："干！出了事我担着！"

配角B却皱眉："你们疯了？这事要是让老爷子知道……"
```

### 扁平化修复公式

```
1. 确定角色性格标签
   - 角色A：冲动、直率
   - 角色B：谨慎、犹豫
   - 角色C：冷漠、简洁

2. 为每个角色设计说话风格
   - 角色A：短句、感叹号、口语化
   - 角色B：长句、问号、犹豫词
   - 角色C：极简、省略语气词

3. 对话时套用风格
```

## 2. 人设一致性检测

**核心问题**：角色行为与之前设定的性格矛盾。

### 不一致表现

```
❌ 不一致示范：
前文设定：主角"城府极深，从不表露情绪"
后文描写：主角"激动得满脸通红，拍着桌子大骂"

问题：人设自相矛盾，读者会感到困惑。
```

### 一致性检测清单

- [ ] **行为是否符合已建立的人设？**
- [ ] **情绪反应是否与性格一致？**
- [ ] **决策逻辑是否与价值观匹配？**
- [ ] **如果有反常，是否有铺垫？**

### 如何处理"合理的反常"

```
✅ 合理反常示范：
往日里泰山崩于前而不改色的苏云，此刻却握着那份信笺，手在发抖。
信上只有一句话——他唯一的弟弟，死了。

修复公式：
一贯表现 + 特殊触发点 = 合理的反常
```

### 一致性检测要点

| 检查项 | 一致 | 不一致 |
|--------|------|--------|
| 性格表现 | 符合设定 | 突然转变 |
| 情绪反应 | 与性格匹配 | 过激或过淡 |
| 决策逻辑 | 有迹可循 | 随心所欲 |
| 语言风格 | 统一 | 忽变 |

## 3. 配角工具人化检测

**核心问题**：配角只为主角服务，没有自己的生活和动机。

### 工具人表现

```
❌ 工具人示范：
配角的存在只是为了：
- 给主角送情报
- 被主角救
- 夸主角厉害
- 当背景板

问题：配角像NPC，世界不真实。
```

### 工具人检测清单

- [ ] **配角有自己的目标吗？**
  - 不只是帮主角
  - 有自己的追求

- [ ] **配角有自己的生活吗？**
  - 不出场时在做什么？
  - 有家人朋友吗？

- [ ] **配角有自己的高光时刻吗？**
  - 有独立的贡献
  - 不总是被主角压一头

- [ ] **配角有自己的性格吗？**
  - 说话风格独特
  - 有优缺点

### 修复示例

```
✅ 独立配角示范：
王胖子虽然是主角的发小，但他有自己的梦想——开一家全城最大的酒楼。
每次主角遇到困难，他不是只会说"我去帮你打架"，而是：
"这事儿，得从生意上入手。你等着，我去查查他们的货源。"

修复公式：
配角 = 自己的目标 + 独特的贡献 + 独立的人格
```

### 配角设计要点

| 类型 | 工具人 | 独立配角 |
|------|--------|----------|
| 目标 | 帮主角 | 有自己的追求 |
| 出场 | 只在需要时 | 有自己的故事线 |
| 贡献 | 被动配合 | 主动作为 |
| 性格 | 单薄 | 丰满 |

## 4. 视角问题检测

**核心问题**：同一段落内频繁切换视角，读者会混乱。

### 视角杂乱表现

```
❌ 视角杂乱示范：
林风感到很紧张。（林风视角）

经理看着林风，心里想：这小子肯定做了亏心事。（突然切换到经理视角）

旁边的同事A也在想：林风平时挺老实的，今天怎么怪怪的？（又切换到同事视角）

问题：同一段落内频繁切换视角，读者会混乱。
```

### 视角检测清单

- [ ] **是否保持单一视角？**
- [ ] **切换视角时是否有明确分隔？**
- [ ] **是否有"上帝视角"偷看他人心理？**

### 正确的视角处理

```
✅ 正确示范（保持单一视角）：
林风感到很紧张，手心都在出汗。

对面的经理盯着他看，表情有些奇怪——那是怀疑吗？

旁边的同事A欲言又止，似乎想说什么，但最终没有开口。
（始终从林风视角观察，不偷看他人心理）
```

### 视角切换规则

| 规则 | 说明 |
|------|------|
| 一章一人 | 建议一章只从一个角色视角出发 |
| 切换有分隔 | 换视角时用新章节或空行 |
| 不偷窥心理 | 除非明确切换视角，否则不写他人想法 |
