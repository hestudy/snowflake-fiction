# 角色质量检查

检测小说中角色相关的常见问题，包括：
- **人设扁平化**：所有角色说话风格雷同
- **人设前后不统一**：行为与设定矛盾
- **配角工具人化**：配角只为主角服务
- **视角杂乱**：频繁切换视角导致混乱

## 使用方式

```
/character-check [文本内容]
/character-check --扁平化 [文本]        # 只检查人设扁平化
/character-check --一致性 [文本]        # 只检查人设一致性
/character-check --配角 [文本]          # 只检查配角工具人化
/character-check --视角 [文本]          # 只检查视角问题
/character-check --设定 [人物宝典路径] [文本]  # 配合人物设定
```

## 检查维度

### 1. 人设扁平化检测

**核心问题**：所有角色说话风格一样，分不清是谁在说话。

```
❌ 扁平化示范：
主角说："我觉得这个方案很不错，我们应该尝试一下。"
配角A说："我也认为这个方案很好，我们可以试试。"
配角B说："这个方案确实不错，我同意。"
→ 三人说话风格完全一致，毫无个性

✅ 个性化示范：
主角沉吟道："试试吧，反正也没别的办法。"
配角A立刻拍桌子："干！出了事我担着！"
配角B却皱眉："你们疯了？这事要是让老爷子知道……"
```

### 2. 人设一致性检测

**核心问题**：角色行为与之前设定的性格矛盾。

检测清单：
- [ ] 行为是否符合已建立的人设？
- [ ] 情绪反应是否与性格一致？
- [ ] 决策逻辑是否与价值观匹配？
- [ ] 如果有反常，是否有铺垫？

### 3. 配角工具人化检测

**核心问题**：配角只为主角服务，没有自己的生活和动机。

检测清单：
- [ ] 配角有自己的目标吗？（不只是帮主角）
- [ ] 配角有自己的生活吗？（不出场时在做什么？）
- [ ] 配角有自己的高光时刻吗？
- [ ] 配角有自己的性格吗？

### 4. 视角问题检测

**核心问题**：同一段落内频繁切换视角，读者会混乱。

规则：
- 建议一章只从一个角色视角出发
- 换视角时用新章节或空行
- 除非明确切换视角，否则不写他人想法

## 评分标准

| 等级 | 分数 | 描述 | 建议 |
|------|------|------|------|
| 优秀 | 80-100 | 角色鲜明，个性突出 | 保持即可 |
| 良好 | 60-79 | 有个性但可加强 | 针对性优化 |
| 及格 | 40-59 | 存在明显问题 | 需要改写 |
| 不及格 | 0-39 | 严重角色问题 | 需要重构 |

## 一句话自检

```
遮住角色名字，你能分辨出是谁在说话吗？
如果不看设定，角色的行为符合你的印象吗？
配角不出场的时候，他在做什么？
```
