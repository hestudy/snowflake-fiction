# 创意与选题检查

检测小说选题和创意相关的常见问题，包括：
- **题材与内容脱节**：标签与实际内容不匹配
- **缺乏辨识度**：同质化严重，没有独特卖点
- **混搭不当**：元素过多，变得四不像
- **世界观设定问题**：设定过多或呈现方式不当

## 使用方式

```
/concept-check [书名 + 简介 + 前三章]
/concept-check --题材 [书名 + 简介 + 内容]     # 只检查题材匹配
/concept-check --辨识度 [书名 + 简介]          # 只检查辨识度
/concept-check --混搭 [书名 + 简介 + 内容]     # 只检查混搭合理性
/concept-check --世界观 [内容]                 # 只检查世界观设定
/concept-check --优化 [书名 + 简介]            # 提供辨识度优化建议
```

## 检查维度

### 1. 题材与内容匹配检测

检测清单：
- [ ] 主标签与主线是否匹配？
- [ ] 内容占比是否合理？（主题材应该占70%以上）
- [ ] 读者预期是否被满足？

### 2. 辨识度检测

**辨识度公式**：`[常见题材] + [独特元素] = 有辨识度的作品`

| 常见题材 | + | 独特元素 | = | 有辨识度的作品 |
|----------|---|----------|---|----------------|
| 系统 | + | 情绪兑换 | = | 用愤怒值兑换能力的系统 |
| 重生 | + | 反派身份 | = | 重生到反派身上 |
| 修仙 | + | 赛博朋克 | = | 赛博修仙世界 |

检测清单：
- [ ] 一句话能说清独特之处吗？
- [ ] 与同类作品有什么不同？
- [ ] 读者能记住什么？
- [ ] 搜索时能用什么关键词找到？

### 3. 跨题材混搭检测

**混搭规则**：
- 元素≤3个（太多会混乱）
- 有主有次（主题材占70%以上）
- 逻辑自洽（两个元素能融合）
- 读者易懂（一句话能说清）

### 4. 世界观设定检测

**设定呈现原则**：
- 场景化：把设定融入具体场景
- 对话化：通过角色对话呈现
- 渐进式：逐步展开，不要一次倾倒
- 需要时才说：只在剧情需要时介绍

检测清单：
- [ ] 开篇设定是否≤3个核心概念？
- [ ] 设定是否通过剧情展开？
- [ ] 读者需要记忆的内容是否过多？
- [ ] 设定是否有必要？

## 评分标准

| 等级 | 分数 | 描述 | 市场表现预测 |
|------|------|------|--------------|
| 优秀 | 80-100 | 创意独特，卖点清晰 | 潜力大 |
| 良好 | 60-79 | 有亮点但可加强 | 表现中等 |
| 及格 | 40-59 | 存在同质化问题 | 需要优化 |
| 不及格 | 0-39 | 缺乏辨识度 | 建议重构 |

## 选题自检清单

```
□ 一句话能说清独特之处吗？
□ 标签与内容匹配吗？
□ 混搭元素≤3个吗？
□ 开篇设定≤3个核心概念吗？
□ 读者为什么要看你的而不是别人的？
```
