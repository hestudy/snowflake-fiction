# 角色设计

覆盖雪花写作法步骤3（人物卡片）→步骤5（背景故事）→步骤7（人物宝典）的完整人物深化链。

## 使用方式

```
# 完整深化（默认，步骤3→5→7）
/character-design 主角 林小满
/character-design 反派 --from 03-完整大纲.md
/character-design 新增配角 神秘商人

# 快速卡片（仅步骤3）
/character-design 快速 主角 都市职场女性

# 更新已有角色
/character-design 更新 林小满 --深化背景
/character-design 更新 林小满 --重写宝典
/character-design 更新 林小满 --添加关系 王经理
```

## 输出物

| 步骤 | 输出文件 |
|------|----------|
| 步骤3：人物卡片 | `03-人物卡片/[类型]-[名].md` |
| 步骤5：背景故事 | `04-人物背景/[名]-背景.md` |
| 步骤7：人物宝典 | `06-人物宝典/[名]-宝典.md` |

## 角色类型与深度

| 类型 | 推荐深度 |
|------|----------|
| 主角 | 完整三步 |
| 反派 | 完整三步，重点心理 |
| 导师 | 步骤3+5 |
| 配角 | 步骤3（简版） |

详细步骤说明请参考 `skills/character-design/SKILL.md`。
