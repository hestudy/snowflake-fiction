# 场景规划

覆盖雪花写作法步骤8（场景清单）→步骤9（场景详细规划）的完整场景设计链。

## 使用方式

```
# 全量规划（默认，步骤8→9）
/scene-plan
/scene-plan --from 03-完整大纲.md

# 局部规划（指定章节，修改大纲后重跑）
/scene-plan 第3章
/scene-plan 第3章-第5章

# 仅生成清单（步骤8）
/scene-plan --list-only
```

## 输出物

| 步骤 | 输出文件 |
|------|----------|
| 步骤8：场景清单 | `07-场景清单.md` |
| 步骤9：场景规划 | `08-场景规划/场景[##]-[名].md` |

## 场景类型

| 类型 | 结构 | 用途 |
|------|------|------|
| 主动场景 | 目标 → 冲突 → 挫折 | 推进情节 |
| 被动场景 | 反应 → 困难 → 决定 | 深化人物 |

生成清单后自动执行节奏检查（主被动交替、冲突强度递增、场景密度）。

详细步骤说明请参考 `skills/scene-plan/SKILL.md`。
