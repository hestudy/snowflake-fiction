# 综合质量检查

一键完成文本的全面质量评估，包括：
- **人语化程度检测**（AI痕迹）
- **流水账检测**（内容张力）
- **内容质量评估**（吸引力）

## 使用方式

```
/quality-check [文本内容]
```

## 检查维度

| 维度 | 权重 | 检查点 |
|------|------|--------|
| 人语化程度 | 30% | 高频AI词、句式整齐、缺乏情感、翻译腔、强行升华 |
| 流水账程度 | 35% | 无冲突、无情绪、无期待感、信息堆砌 |
| 内容质量 | 35% | 冲突强度、情绪密度、期待感、节奏控制、钩子设计 |

## 评分标准

| 等级 | 分数 | 描述 | 建议 |
|------|------|------|------|
| 优秀 | 80-100 | 质量过硬 | 可直接发布 |
| 良好 | 60-79 | 有小问题 | 建议优化 |
| 及格 | 40-59 | 有明显问题 | 重点改写 |
| 不及格 | 0-39 | 存在严重问题 | 需要重构 |

## 推荐工作流

```
1. /quality-check [正文]
2. 根据报告决定：
   ├── 评分≥70 → 进入人语化
   ├── 评分50-69 → 先修复问题
   └── 评分<50 → 重新审视内容
3. /humanize-text [正文]
4. /quality-check [处理后的正文]
```

## 相关命令

- `/humanize-text` - 人语化处理
- `/boring-detect` - 流水账专项检测
- `/opening-check` - 开篇质量检查
